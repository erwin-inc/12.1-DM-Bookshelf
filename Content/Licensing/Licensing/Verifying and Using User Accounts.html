Verifying and Using User Accounts

Once users are added to the user groups, they can use the given credentials to log on to erwin Data Modeler (erwin DM) and use it.

To verify user account and use erwin DM, follow these steps:

  1. Open the End User Portal (EUP) email.


  2. Click the password reset link.
  3. Enter a new password and click Submit.

  4. Password is reset and the user is logged on to the EUP. If you are an administrator user, you can add more users to the user group.

  5. Start erwin DM.
    The erwin Data Modeler Licensing screen appears.

  6. Click Account Based.

  7. Enter your account-based licensing username and password.

  8. Click OK.
