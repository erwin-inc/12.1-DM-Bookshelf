Master Group, Master User, and User Groups

The primary components of Account Based Licensing are:

  • Groups: Each organization has a Master Group and multiple User Groups (Standard Groups). Groups are created by the erwin Licensing team during your account-based licensing configuration.

  • Users: There are three user types, Master User, Administrator, and Standard User. Each organization has one master user under the master group, one administrator and one or more standard users under user groups.

  • License Code: Each organization has one or more license codes that are assigned to each user group. This is managed by the erwin Licensing team during your account-based licensing configuration.

Once your license request is verified, the erwin Licensing team will contact you for the following information:

  • Master User and Master Group:
    Provide a master user's valid email address and name. This user will be your in-house license administrator and must belong to a department that does not use erwin Data Modeler. (erwin DM). For example, someone from the administration or purchase department.
    Also, provide a name for your master group. For example, your organization's name. A master group is a parent group for all user groups in your organization.

    Each organization can have only one master group and master user.

  • User Groups:

    Provide the number of user groups that would use erwin DM and their names. For example, Data Modelers, Developers, and so on.

Based on this information, erwin's Licensing team will set up account-based licensing for you and share the following with the designated master user:

  • End User Portal (EUP) URL

  • Master User's EUP credentials

Once the above information is available to the master user, they can add users to user groups. For more information, refer to the Adding Users to User Groups topic.
