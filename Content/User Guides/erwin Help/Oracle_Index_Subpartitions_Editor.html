Oracle Index Subpartitions Editor

Use the Oracle Index Subpartitions Editor to modify the properties for a subpartition of an index for a table or materialized view. The following describes the options available for this dialog:

Store In

When you select this option you must select a Tablespace for the subpartition.

Tablespace

Displays the selected tablespace for the subpartition. Click the button to open the Oracle Select Tablespaces dialog.

Subpartition by name

When you select this option, enter a subpartition name in the Subpartition Name column. When you click on a specific subpartition name, you can use the button and the button to move it up or down in the list, or click the Delete button to delete it. Click the button within the Edit Properties column to open the Oracle Partition Storage Dialog.
