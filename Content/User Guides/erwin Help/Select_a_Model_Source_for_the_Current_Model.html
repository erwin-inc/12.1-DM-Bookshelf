Select a Model Source for the Current Model

You use the options in the Source Model pane to select the model source for the current model.

To select a model source for the current model

  1. Click Model Source on the Model menu.

    The Model Sources Properties dialog opens.

  2. Click New.

    The Add Model Source Wizard opens on the Source Model pane.

  3. Work with the following options:

    Load From

    Select a *.erwin file or a model located in a mart as your source model. Click Load to select the model. You can load any file type listed on the "Files of types" drop-down list.

    Open Models in Memory

    Displays the name and location of models you have open in your current modeling session. You can choose one of these models as your source model.

  4. Click Next.

    The Type Selection Pane opens.

You can click the Add button at any time to add the model source.
