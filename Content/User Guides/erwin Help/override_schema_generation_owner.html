Override Schema Generation Owner

On the Forward Engineer Schema Generation Wizard use the Owner Override tab to select an object group and override name for the schema generation. The object group and name you indicate overrides the owner name assignment for that object group in the model. You can customize the options by selecting the check boxes and entering owner names.

  • Select the Global Override check box and enter an owner name to override all owners for the listed object groups.
  • Clear all the check boxes to use the owner names defined in the model during schema generation.

The list of available objects is based on the current target server and may include the following object groups: Table, Index, View, Materialized View, Trigger, Sequence, Synonym, Cluster, Cluster Index, and Library.
