Db2 z/OS Property Editors

The following list summarizes support for Db2 z/OS property page editors.

  • Auxiliary Tables
  • Database Roles (Db2 z/OS 9.1 and above)
  • Databases
  • Files
  • Functions
  • Index
  • Materialized Query Tables
  • Permissions (Define Db2 z/OS Permissions at Database Level, Define Db2 z/OS Permissions at Object Level)
  • Schemas
  • Sequences
  • Storage Groups
  • Stored Procedures
  • Synonyms
  • Tables
  • Tablespace
  • Triggers (Define a Db2 z/OS Table Trigger, Define a Db2 z/OS View Trigger)
  • Trusted Contexts (Db2 z/OS 9.1 and above)
  • Users
  • View

Refer to your Db2 database documentation for detailed information on specific Db2 objects and properties. The erwin® Data Modeler documentation for the property editors provides brief descriptions of the controls in each dialog and tab, which you can use as a point of reference while working with database design features.
