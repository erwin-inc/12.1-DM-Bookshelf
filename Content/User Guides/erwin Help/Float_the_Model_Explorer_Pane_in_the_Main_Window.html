Float the Model Explorer Pane in the Main Window

When the Model Explorer is not positioned (docked) next to the workspace boundary, it is floating. You can float the Model Explorer Pane in the main workspace by dragging the pane anywhere within the workspace. By default, the Model Explorer pane is docked.

You can also double-click the Model Explorer title bar to return to the previous docked or floating Model Explorer position. For example, if you reposition a floating Model Explorer and then double-click the Model Explorer gripper bar, the Model Explorer returns to the previous docked position.
