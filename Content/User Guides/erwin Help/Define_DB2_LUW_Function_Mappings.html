Define Db2 LUW Function Mappings

Use the options in the Db2 for LUW Function Mapping Editor to define function to function mappings.

To define Db2 LUW Function Mappings

  1. Right-click a Function Mapping in the Model Explorer and select Properties.

    The Db2 for LUW Function Mapping Editor opens.

  2. Select a function mapping name and work with the following options:

    Click New on the toolbar to create a function mapping. Use the Enter filter text text box to filter a large list of names to locate the one you want to work with.

    Name

    Specifies the function mapping name.

    Generate

    Specifies whether or not to generate DDL for this object during Forward Engineering.

  3. On the General tab, work with the following properties:
    Function Mapping Properties:
    Type

    Specifies the function mapping type. Select Function Reference or Specific Function from the drop-down list.

    Function

    Specifies the function to associate with the function mapping. Create or select a function from the drop-down list.

    Use Infix (check box)

    Specifies whether or not to generate the data source function in infix format.

    Function Mapping Server Information:
    Server

    Specifies the server name to associate with the function mapping. Create or select a server from the drop-down list.

    Server Type

    Indicates the server type.

    Server Version

    Indicates the server version.

    Wrapper

    Specifies the wrapper to associate with the function mapping. Create or select a wrapper from the drop-down list.

  4. Click the Function Mapping Options tab to specify function mapping options and their values.
  5. (Optional) Click the Comment tab and enter any comments that you want to associate with the object.
  6. (Optional) Click the UDP tab to work with user-defined properties for the object.
  7. (Optional) Click the Notes tab to view and edit user notes.
  8. Click Close.

    The Db2 for LUW Function Mapping Editor closes.

Define Db2 LUW Function Mapping Options

Use the options in the Db2 for LUW Function Mapping Options tab to define the function mapping options that you want to enable.

Follow these steps:

  1. Right-click a Function Mapping in the Model Explorer and select Properties.

    The Db2 for LUW Function Mapping Editor opens.

  2. Select a function mapping name and click the Function Mapping Options tab.
  3. Work with the following options:
    Name

    Specifies the function mapping option that applies to the function selected in the General tab.

    Value

    Specifies the value for the function mapping option, as a character-string constant.

  4. Click Close.

    The function mapping option is saved and the editor is closed.
