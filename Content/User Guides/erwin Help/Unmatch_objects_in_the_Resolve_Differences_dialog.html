Unmatch Objects in the Resolve Differences Dialog

If you used the Match feature to override automatic matching in Complete Compare, you can also undo a match action. You select the row that contains the matched objects and then unmatch the objects.

To unmatch objects in the resolve difference dialog

  1. In the Resolve Differences dialog, click the line that contains the items you want to unmatch.

    The Unmatch icon becomes active in the standard toolbar.

  2. Click the Unmatch icon.

    The items are moved to the different lines in the Differences list.

The toolbar buttons are available or unavailable (dimmed) depending on whether or not the action is available for the item selected.
