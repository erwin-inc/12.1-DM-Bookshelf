Display a Key Group Only in the Logical Model

You can indicate that a key group only appears in the logical model. If you select this option, your logical key groups will not have indexes in the corresponding tables on the physical side. This procedure applies to a logical/physical model, but also can be applied to a logical-only model. If you derive a new physical model from a logical-only model in which the key groups are marked "Logical Only," those key groups are not created in the new physical model.

To display a key group only in the logical model

  1. Right-click the entity that contains the key group to specify as logical only and click Key Group Properties.

    The Key Group Editor opens.

  2. Select the key group in the Navigation Grid that you want to only appear in a logical model, and select the Logical Only check box.
  3. Click Close.

    The logical only specification is saved for the key group and the Key Group Editor closes.
