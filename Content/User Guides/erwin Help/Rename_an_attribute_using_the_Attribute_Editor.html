Rename an Attribute Using the Attribute Editor

You can rename an attribute using the Attribute Editor.

To rename an attribute using the Attribute Editor

  1. Right-click the entity on the diagram that contains the attribute that you want to rename and click Attribute Properties.

    The Attribute Editor opens.

  2. Click the name cell of the attribute in the Navigation Grid that you want to rename.
  3. Enter the new name and click Close.

    The attribute is renamed and the Attribute Editor closes.
