Enter a Name and Synchronization Direction for the Link Object

You use the Link page on the Link Wizard to enter a name and synchronization direction for the link object. You can also optionally create a definition for the link object.

To enter a link object name and synchronization direction

  1. Open the Column or Attribute Editor and select the column or attribute for which you want to create the link object.
  2. Click the Link tab then click New on the toolbar.

    The Link Wizard opens.

  3. Click the Link page and work with following options:
    Name

    Enter a name for the link object. A default name is created for you, which you can use or modify.

    Synchronization Direction

    Select a synchronization direction from the drop-down list:

    From This

    Synchronize from this column/attribute to the target column/attribute.

    None

    Suspend synchronization. Use this option to turn off synchronization as you make changes to either the source or target object.

    To This

    Synchronize from the target column/attribute (specified in the Target page of the wizard) to the source column/attribute (selected when you opened the Link Wizard).

    Two Way

    Synchronize in both directions. A change made to either the source or target column/attribute is synchronized to the linked object.

    You can change the synchronization direction for an existing link object on the Link tab of the Column or Attribute Editor.

    Definition

    (Optional) Type a definition for the link object. You can use the cut, copy, paste, edit, and spell check controls in the pane when you create your definition.

    The selected column/attribute becomes the source object for the link. You then define a target object for the link in the Target page.
