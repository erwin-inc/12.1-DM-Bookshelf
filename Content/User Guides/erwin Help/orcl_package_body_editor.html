Oracle Package Body Editor

Open the Oracle Package Body Editor by selecting the Body tab from the Oracle Package Editor. Open the Oracle Package Editor by selecting Package from the Database menu. The Oracle Package Body Editor corresponds to the CREATE PACKAGE BODY statement. There is one multi-line entry field to enter or edit the package body. There is no syntax checking for this entry field.

You can copy and paste either Stored Procedures or Functions into the package body by clicking on either the Procedures button or the Functions button.

For specific information about the options for the Oracle Package Body Editor, see Oracle Package Editor Dialog - Body Tab Options.
