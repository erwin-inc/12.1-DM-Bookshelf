Load or Select a Model for Synchronization

Use the Right Model Selection dialog to load or select a model source and location to which you want to synchronize the current model. You can set the selected model as read-only by selecting the check box in this pane.

To load or select a model for synchronization

  1. Click DB Sync in one of the property editors, such as the Column Editor.

    The DB Sync wizard opens on the Right Model pane.

  2. Work with the following options:
    File

    Select this option, then click the Load button to browse your file system for a local *.erwin file.

    Database/Script

    Select this option, then click the Load button to browse your file system for a database or script. The database or script is taken through the reverse engineer process and displayed as an open model in the workspace.

    Mart

    Select this option, then click the Load button to open a model from a mart.

    Open Models in Memory

    Displays the name and location of models that are open in the workspace.

    Set selected model as read-only

    Select this check box to indicate that the model is read-only. When you select the read-only option you cannot import objects or properties into the model in the Resolve Differences dialog.

  3. Click Next.

    The Type Selection pane of the DB Sync wizard opens.
