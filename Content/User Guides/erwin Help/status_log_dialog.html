Status Log Dialog

When you click the Finish button to perform either an Import from External Format or an Export to External Format, a Status Log dialog opens to display the progress of the import or export. The Import from External Format Status Log opens during the import from external format process, and the Export to External Format Status Log opens during the export to external format process.

When the import or export has completed, you can either click Close to close the Status Log dialog, or you can click Save Log to save the status log file to the default location. You set the default location for saved status log files in the Diagnostics tab of the Options dialog, which is opened by selecting Options from the Tools menu.
