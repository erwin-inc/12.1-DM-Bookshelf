Save an Option Set

You can customize the default option set in the Forward Engineer/Schema Generation and Reverse Engineer wizards to suit your requirements. Save your custom option set and use it with any model and with any session of forward or reverse engineering.

Follow these steps:

  1. Click Save As near the top of the wizard.

    The Save current Option Set to a destination dialog opens.

  2. Select a destination:
    • XML file - save the option set to an XML file.
    • Selected Model - save the option set with the model. The model name displayed corresponds to the model name (not file name) created in the Model Properties dialog.
  3. Enter a name for the option set:
    • For an XML file, click Browse to name the file and select a location for the saved file.
    • For an option set saved with the model, enter the Option Set name in the text box.
  4. Click OK.

    Your option set is saved and you return to the wizard.

If you open the r9.8 version of PostgreSQL FE option set, on the Forward Engineer Schema Generation Wizard, revisit the Other Options category under Option Selection and ensure the desired options are selected.
