Use the Details Pane

You can view every change made to a model in the Details pane. It is the default pane when the program is launched. Transactions appear linear within the tree with the most recent transaction at the top of the tree. Transaction details can either be expanded or collapsed. Changes made to the model are automatically reflected in the the Details pane. When the model is updated, the changes reflect in the Action Log.

To switch to the Details pane, click the Details tab at the bottom of the Action Log pane.
