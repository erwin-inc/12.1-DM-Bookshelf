Change the Object Creation Order in a Property Editor

The Object Creation Order tab appears on selected physical object property editors to specify the order of DDL generation for the available objects. The default order is the order in which each object was created. The Object Creation Order tab lets you control the order in which model-level stored procedures and functions are forward engineered.

You cannot add, modify, or delete any objects using this tab.

To change the object creation order in a property editor

  1. Open the property editor for the physical object for which you want to change the object creation order and click the Object Creation Order tab.

    The Object Creation Order tab opens displaying the available objects and the name, schema, and type of each object.

  2. Verify and change the creation order of the objects using the up and down arrow buttons and click Close.

    The object creation order is changed and the property editor closes.
