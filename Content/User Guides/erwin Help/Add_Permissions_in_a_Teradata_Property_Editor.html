Add Permissions in a Property Editor

You can use the Permission tab to create, edit, or delete permissions that apply to the current editor. For example, if you click the Permission tab in the Teradata Function editor, any permissions that you create, edit, or delete affects the Function that you selected in the Function editor.

To add permissions in a Teradata property editor

  1. Open the property editor for an object and click the Permission tab.

    The Permission tab opens.

  2. Enter the values and select the options for a permission:

    You can click the Permission Editor icon to open the Teradata Permission Editor.

    Name

    Displays the permissions for the selected physical object.

    Type

    Specifies the permission type. Select a type from the drop-down list.

    Generate

    Generates DDL for this object during Forward Engineering.

  3. Click Close.

    The property editor closes.
