Oracle Table Editor - Validation Tab

Use the Validation tab to add constraints and validation rules. The following describes the options available for this tab:

Check Constraints

Displays the constraint names. The following describes the information in this grid:

Name

Enter a constraint name.

Validation Rule

Click on the drop-down list to select an existing validation rule.

Edit

Click the within this column to open the Validation Rules dialog to create or modify a validation rule for a constraint.

Constraint State

Click the within the Constraint State column to open the Oracle Constraint State dialog where you can modify the properties of an Oracle constraint state.
