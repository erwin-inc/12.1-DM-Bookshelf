Print Current Template Source

You can print the text displayed in the Template Source pane. This text changes dynamically, based on the template that is selected in the Templates pane.

Although you can resize the Template Source pane, you must scroll through the text that cannot be displayed in one pane. Print the template source for offline reference.

To print current template source

  1. Click the Print Current Template Source icon on the Template Editor toolbar.

    The Print dialog opens.

  2. Select a printer, and click Print.

    The template source text is printed.
