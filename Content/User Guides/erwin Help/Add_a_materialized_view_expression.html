Add a Materialized View Expression

To add a materialized view expression

  1. In a physical model based on Oracle, right-click a materialized view in the diagram, and select Database Materialized View Properties.

    A context menu with additional options opens.

  2. Click the Select option on the context menu.

    The Materialized Views Editor opens on the Select tab.

  3. Click Newto open the New View Columndialog.
  4. Enter the column name and expression, and click OK.
