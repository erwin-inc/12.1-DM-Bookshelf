Oracle External Table Dialog

Open the Oracle External Table Dialog when you click on the External Properties button in the General tab of the Oracle Table Editor. The following describes the options for the Oracle External Table Dialog:

Access Driver Type

When you select this check box, select one of the following options:

Oracle Loader
Oracle Datapump
Reject Limit

When you select this check box, select one of the following options:

UNLIMITED
LIMIT

If you select the LIMIT option, enter a value in the text box.

Default Directory

Use the drop-down to select an Oracle directory. If you need to edit the properties for this directory, click the button to open the Oracle Server Directories Editor.

Access Parameter Type

Select one of the following options:

Opaque Properties
CLOB

Selecting one of these options clears any options you selected for Access Drive Type and Reject Limit.

Access Parameters

Edit your access parameters in this text box.

Locations

A grid that contains Oracle directory information.
