Working on erwin DM NoSQL

erwin DM NoSQL enables you to model and manage your unstructured cloud data as effectively as traditional data sources. It is a native, enterprise-class data modeling solution. erwin DM NoSQL is an extension of erwin Data Modeler that enables “any data” from “anywhere” to be visualized for better understanding and decision-making.

You can connect to erwin DM NoSQL via erwin Data Modeler Workgroup Edition. Your .erwin file is first exported to the XML format and then compressed to the ZIP format. This ZIP file is then transfered to erwin DM NoSQL, where it is unzipped and added to your collection of data models.
