Oracle Editors - Physical Tab

The Physical tab is used in the Oracle Index Editor for Tables and the Oracle Index Editor for Materialized Views. Use the Physical tab to assign physical and storage properties. The following describes the options available for this tab:

Unique

Select this check box for a unique index.

Bitmap

Select this check box for a bitmap index. This option is not available for a clustered materialized view.

Storage properties

Click this button to open the Oracle Storage Properties Dialog where you can assign storage properties to the index. This option is disabled when using a global temporary table.

Physical Property

Use the following options to assign various physical properties:

Parallel Type
  • NOPARALLEL
  • PARALLEL This option is disabled when using a global temporary table. If you select this option, enter a value in the text box.
REVERSE

This option is not valid for a Bitmap index or for an index organized table.

ONLINE
COMPUTE STATISTICS
Compress Type
  • NOCOMPRESS
  • COMPRESS If you select this option, enter a value in the text box.This option is not available for a Bitmap index.
Sort Type
  • NOSORT This option is not available for a Bitmap index. You cannot select both this option and the ONLINE option.
