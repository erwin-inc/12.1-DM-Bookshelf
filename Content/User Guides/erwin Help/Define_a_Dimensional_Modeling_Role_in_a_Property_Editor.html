Define a Dimensional Modeling Role in a Property Editor

The Dimension tab appears on selected physical storage object property editors to let you define dimensional modeling roles for the physical storage object when you have selected to use dimensional modeling for the current model.

To define a dimensional modeling role

  1. Open the property editor for a physical storage object and click the Dimension tab.

    The Dimension tab opens.

  2. Complete the following fields on the Dimension tab and click Close.
    Dimensional Modeling Role

    Specifies the role of the physical storage object in the dimensional model.

    Dimension Type

    Specifies how to update information in the data warehouse if the physical storage object has a dimension role.

    Your edits are saved and the property editor closes.
