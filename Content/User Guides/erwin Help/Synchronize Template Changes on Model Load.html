Synchronize Template Changes on Model Load

Every time you open a model with a bound template, you can synchronize template changes automatically instead of launching the Model Template Synchronization Wizard.

To synchronize template changes automatically,

  1. Open the model that uses the model template with which you want to synchronize.
  2. On the Home tab, click Model Templates.
  3. Click Synchronize on Model Load.
  4. Save the model.

    Synchronization occurs automatically whenever you open the model; the wizard does not open.

In case you want to change the auto-synchronization settings at a later point in time,

  1. Launch erwin Data Modeler.
  2. Without opening any model, go to Home > Model Templates.
  3. Clear Synchronize on Model Load. You can then open the model without synchronizing it with the template.

Synchronization on loading the model does not work as expected for inherited templates. As a workaround, do either of the following:

  • Refresh the template using the Home > Model Templates > Refresh Template option.
  • Select Preserve the template binding while creating a new model.
