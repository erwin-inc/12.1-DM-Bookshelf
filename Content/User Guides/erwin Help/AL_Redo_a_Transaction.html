Redo a Transaction

You can redo only the last transaction undone. Redo Range sequentially recreates multiple transactions. The transaction becomes bold in the Action Log to indicate it was recreated. Use the Action Log to verify only the transactions you want to Redo are recreated.

To redo a transaction, click Redo on the erwin toolbar. The last undone transaction in the tree is no longer grayed out in the Action Log pane.
