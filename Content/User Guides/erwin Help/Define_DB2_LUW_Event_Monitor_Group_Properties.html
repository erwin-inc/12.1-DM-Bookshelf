Define Db2 LUW Event Monitor Group Properties

You can create an Event Monitor Group and associate it with an event monitor.

To define Db2 LUW Event Monitor Group properties

  1. Right-click an Event Monitor in the Model Explorer and select Properties.

    The Db2 for LUW Event Monitor Editor opens.

  2. Click the Event Monitor Group tab and then click the Event Monitor Table Editor icon.

    The Db2 for LUW Event Monitor Group Editor dialog opens.

  3. Select an existing Event Monitor Group and work with the following options:

    Click New on the toolbar to create a event monitor group. Use the Enter filter text text box to filter a large list of names to locate the one you want to work with.

    Name

    Specifies the name of the Event Monitor Group.

    Schema

    Specfies the schema for the table.

  4. Click the General tab and work with the following options:
    Event Group Type

    Specifies the logical data group for which the target table is being defined. Select from the drop-down list.

    Table Name

    Specifies the target table for the event group.

    Tablespace

    Specifies the tablespace in which the table name mentioned earlier is to be created.

    Deactivate Percentage

    Specifies the percentage the table should be full for the event to automatically deactivate.

    Truncate

    Specifies that the STMT_TEXT and STMT_VALUE_DATA columns are defined as VARCHAR(n), where n is the largest size that can fit into the table row.

  5. Click the Include Elements tab and specify the elements that you want to be included in the table.
  6. Click the Exclude Elements tab and specify the elements that you want to be excluded from the table.
  7. Click the UDPtab to work with user-defined properties for the Event Group Monitor.
  8. Click the Notestab to view or edit user notes.
  9. Click Close.

    The Db2 for LUW Event Monitor Group Editor closes.

Include or Exclude Elements in an Event Monitor Group

You can create an Event Monitor Group and associate it with an event monitor. If the events are written to a table, you can specify the elements of the table that are to be included and excluded.

Follow these steps:

  1. Right-click an Event Monitor in the Model Explorer and select Properties.

    The Db2 for LUW Event Monitor Editor opens.

  2. Click the Event Monitor Group tab and then click the Event Monitor Table Editor icon.

    The Db2 for LUW Event Monitor Group Editor dialog opens.

  3. Click the Include Elements tab or the Exclude Elements tab.
  4. Click New and add an element that you want to include to or exclude from the output table.
  5. Click Close.

    The elements are added and the dialog is closed.
