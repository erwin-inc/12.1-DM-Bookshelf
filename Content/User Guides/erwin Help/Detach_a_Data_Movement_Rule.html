Detach a Data Movement Rule

You use the Attachment tab in the Data Movement Rule Editor to detach a data movement rule from one or more tables in a model. You can detach a data movement rule if you no longer need it attached to a table in a physical model.

The Data Movement feature is available in a physical model when you select the Data Movement check box on the General tab in the Model Editor.

To detach a data movement rule

  1. Click Data Movement Rules on the Model menu.

    The Data Movement Rule Editor dialog opens.

  2. Select a data movement rule.
  3. In the Attachment tab, perform one of the following tasks:
    • Double-click a table in the Attached Tables list, the table appears in the Unattached Tables list.
    • Select a table in the Attached Tables list, and then click , the table appears in the Unattached Tables list.
    • Click to move all tables from the Attached Tables list to the Unattached Tables list.
  4. Click Close.

    The Data Movement Rule Editor closes.
