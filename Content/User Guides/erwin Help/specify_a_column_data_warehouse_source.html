Specify a Column Data Warehouse Source

Use the Data Source tab in the Table Column Editor to specify the data warehouse source for the selected column.

Ensure that you have selected the Data Movement option in the Model Editor to enable data movement modeling features.

To specify a column data warehouse source

  1. Right-click the column in the Model Explorer for which you want to specify a data warehouse source and click Properties.

    The Table Column Editor opens.

  2. Click the Data Source tab and work with the following options:
    Data Sources

    Lets you specify a data warehouse source to attach to the selected column. Use the buttons on the toolbar to sort, arrange the list, restore inheritance, or open either the Data Movement Source Editor or the Data Sources Selector to work with data warehouse sources.

    DS Comment

    Lets you enter a comment to describe how the source is integrated into the data warehouse. Use the buttons on the toolbar to cut, copy, and paste comment text, restore inheritance, work with the Text Editor, or check spelling.

  3. Click Close.

    The column data warehouse source is specified and the Table Column Editor closes.
