View Property Change Details

You can view the property change details for the displayed property selected in the tree. The old and new values are displayed.

To view the details of a property transaction

  1. Expand the transactions in the Details pane or Summary pane, select a property node and click the Property Details button.

    The Property Change Details dialog opens. Property displays the property type of the selected text in the Summary tree, Old Value displays the original value of the property selected before a change was made, and New Value displays the current value of the property selected after a change was made.

  2. Click Close when you are finished viewing the property details.

    The Property Change Details dialog closes.
