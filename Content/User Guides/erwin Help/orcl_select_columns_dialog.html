Oracle Select Columns Dialog

The Oracle Select Columns Dialog opens when you click on a cell under Columns in the grid in the Oracle LOB Storage dialog. The following describes the options available for the Oracle Select Columns Dialog:

Available Columns

Displays the available columns in the current model that you can select and move to the Selected Columns list. Click on a column that you want and use the button to move it to the Selected Columns list.

Selected Columns

Displays the columns that you selected and moved from the Available Columns list. If you decide that a column does not belong in the Selected Columns list, click on it and use the button to move it back to the Available Columns list.
