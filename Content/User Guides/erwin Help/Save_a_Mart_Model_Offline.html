Save a Mart Model Offline

You save a mart model offline to work on that model without connecting to Mart.

When the model is saved offline, the status of the model is changed from Online Mart Model to Offline Mart Model in the status bar.

Follow these steps:

  1. Open a model from Mart.
  2. Click File, Save or Save As.

    The Save As dialog opens.

  3. Navigate to the file location and click Save.

    The Save As dialog closes and the Save Model Offline or Detach Model From Mart dialog opens.

  4. Click Offline.

    The Save Model Offline or Detach Model From Mart dialog closes and the model is saved as an offline mart model.

To avoid any rebinding issues in case of an offline mart model, at a later stage, when you open this model, do the following:

  • Save that model back to mart
  • Change to online model
  • Make a duplicate copy of the model by using Save As.

For more information, refer Rebinding an Offline Mart Model to Mart.
