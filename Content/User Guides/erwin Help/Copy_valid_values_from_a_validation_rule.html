Copy Valid Values from a Validation Rule

You can save time and ensure consistency by copying valid values from an existing validation rule to a new validation rule you create.

To copy valid values from a validation rule

  1. ClickValidation Rules on the Model menu.

    The Validation Rule Editor opens.

  2. Click the New button.

    A new validation rule with a default name is added to the Navigation Grid.

  3. Select the validation rule you just added, change the physical name, select Valid Values from the Type drop-down, and select the Physical Only check box if you want the validation rule to only appear in the physical model.
  4. Click the General tab and click the Import button in the Valid Values section.

    The Open dialog opens.

  5. Select the location of the .csv or .txt file that contains the valid values that you want to copy and click Open.

    The valid values are imported and display in the Valid Values section.

  6. Make any changes that may be needed and click Close.

    Valid values are copied to the new validation rule and the Validation Rule Editor closes.
