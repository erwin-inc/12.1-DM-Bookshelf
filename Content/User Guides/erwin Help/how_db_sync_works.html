How DB Sync Works

The DB Sync process uses a wizard with a series of dialogs to specify options for performing a database-level compare of specific objects in the current model with matching objects in another model or database. You start the wizard when you click the DB Sync button in one of the property editors, for example, in the Columns dialog. Your current model is selected by default. You then select the model or database to synchronize with. You can apply filters and other synchronization options, or click Compare to begin the synchronization using the preset defaults. Use the Resolve Differences dialog to compare the current model with the selected model.

Click a pane in the navigation window of the wizard to open that pane. Click Next in each pane to navigate sequentially through the panes in the wizard. Review the text in the Overview pane for basic concepts on the DB Sync process. The Sync with Model Source wizard includes panes that let you do the following:

  • In the Left Model pane, The name and path of the current model is displayed. Your open model is selected by default. You can set the selected model as read-only by selecting the check box in this pane. When you indicate that the current model is read-only, you cannot import objects or properties into the model in the Resolve Differences dialog.
  • In the Right Model pane, select a model source and location you want to synchronize the current model to. You can set the selected model as read-only by selecting the check box in this pane.
  • In the Type Selection pane, specify the object types you want to synchronize between the source and the target model. You can also choose a filter to select only those objects in the source model not linked to any model source, or create an option set to specify specific objects. By default, all objects in the source model are included in the link process.
  • In the Left Object Selection pane, specify which model objects from the current model participate in the synchronization process.
  • In the DB Sync Options pane, set options for the synchronization objects.
