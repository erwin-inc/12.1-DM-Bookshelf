Enter a Foreign Key Comment to a Relationship

You can use the Comment tab in the Relationships editor to enter a comment for a relationship.

To enter a foreign key comment to a relationship

  1. Click Relationships on the Model menu.

    The Relationship editor opens.

  2. Select a relationship from the Relationship drop-down list to which you want to add a foreign key comment, then click the Comment tab.

    The Comment tab opens.

  3. Enter the foreign key comment in the Foreign Key Comment field and click OK.

    Click an icon to cut, copy, or paste text, or to work with the Text Editor.

    Comments are saved and the Relationships editor closes.
