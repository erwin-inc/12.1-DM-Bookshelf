Show Detailed Information for a Macro

Detailed documentation for each macro is available from a shortcut menu. Review the documentation to learn more about correct usage and expected results.

To show detailed information for a macro

  1. Right-click a macro name in the Macros pane in the Template Editor.

    A shortcut menu opens.

  2. Click Show Macro Information on the menu.

    A new window opens that shows information for the selected macro.

For more detailed information about all macros, see the Template Language and Macro Reference guide, which is installed by default to the \Doc directory during product installation.
