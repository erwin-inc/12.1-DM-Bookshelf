Advisory Details

The Advisory Details dialog provides additional information related to a failed transaction. In cases where a great deal of text displays in the main Advisories dialog, you can use the Details dialog to display complete information for a selected advisory message. For successful transactions, this dialog does not display additional information.

For example, for a failed transaction related to reverse engineering a physical name, the problem related to the object property for the specific transaction is described in an informational message.

To display additional information about the object properties for that transaction, click Details in the Advisories pane.
