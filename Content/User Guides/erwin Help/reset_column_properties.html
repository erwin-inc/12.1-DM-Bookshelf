Reset Column Properties

You use features in the Reset Column Properties dialog in a physical model to reset column properties for one column, all columns in a table, or all columns in your model.

To reset column properties

  1. Right-click the table that contains the columns that you want to reset and click Column Properties.

    The Table Column Editor opens.

  2. Select the column name in the Navigation Grid that you want to reset and click the Reset Property Inheritance button.

    The Reset Column Properties dialog opens.

  3. Work with the following options:
    Scope of Reset operation

    Specifies the scope of the column properties to reset. Select one of the following actions:

    Only Column <column name>

    Specifies to only reset the properties of the column selected in the Table Column Editor.

    All Columns of Table <table name>

    Specifies to reset the properties of all columns in the selected table.

    All Table Columns in model

    Specifies to reset the properties of all columns in the current model.

    Restrict to Columns that are attached to user-defined Domains

    Specifies to restrict the column properties to reset to only those attached to user-defined domains. This option is only available when you select to work with all columns of a given table, or all columns in the model. Select the check box to enable this option.

    Select properties to reset

    Lets you select the individual column properties to reset.

    Select user-defined properties to reset

    Lets you select the individual user-defined properties to reset, if any have been defined.

  4. Click OK.

    Column properties are reset, the Reset Column Properties dialog closes, and you return to the Table Column Editor.

  5. Click Close.

    The Table Column Editor closes.
