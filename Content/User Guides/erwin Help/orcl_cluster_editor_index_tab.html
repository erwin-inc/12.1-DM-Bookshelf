Oracle Cluster Editor - Index Tab

Use the Index tab to create an Oracle cluster index. This tab is disabled unless you select Index Cluster as the Cluster Type in the General tab. The following describes the options available for this tab:

Create Index

Select this check box to create an index, and select the options for this index:

Name

Enter a name for the index.

Owner

Enter a database Owner name for the index.

Physical Property

Select the physical property options that you require from the following:

  • Parallel Type: Select NOPARALLEL or PARALLEL. If you select PARALLEL, enter a value in the text box.
  • SORT
  • NOCOMPRESS
  • REVERSE
  • COMPUTE STATISTICS
Storage Properties

Select the Storage Properties button to open the Oracle Storage Properties Dialog.

The following describes general information regarding Index on Clusters:

  • You can only have one index per cluster.
  • You cannot create an index on a hash cluster.
  • The properties UNIQUE, BITMAP, NOSORT, ONLINE, COMPRESS, and COMPRESS VALUE are not applicable for a clustered index.
