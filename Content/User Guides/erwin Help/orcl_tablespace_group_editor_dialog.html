Oracle Tablespace Group Editor Dialog

The Oracle Tablespace Group Editor is used to create new tablespace groups or edit the properties of existing ones. The Oracle Tablespace Group Editor is available only for Oracle 11g. Open the Oracle Tablespace Group Editor from the General tab of the Oracle Tablespace Editor by clicking on the button next to the Tablespace Group drop-down list box.

The Oracle Tablespace Group Editor dialog includes the following tabs:

General

Lets you add tablespaces to selected tablespace groups.

Dependencies

Displays any dependencies for the tablespace group.

Comment

Lets you enter any comments that you want to associate with the object.

UDP

Lets you enter user-defined property values for the object.

Notes

Lets you view or edit user notes.

The following describes all of the other options available in this dialog:

Tablespaces Groups

Displays the available tablespace groups.

New

Opens the New Tablespace Group dialog where you can specify the name for the new tablespace group.

Rename

Opens the RenameTablespace Group dialog where you can assign a different name to the tablespace group.

Delete

Select a tablespace group from the Tablespaces Groups list that you want to delete, and click the Delete button.
