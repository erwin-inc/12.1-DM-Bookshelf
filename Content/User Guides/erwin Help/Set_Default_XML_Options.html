Set Default XML Options

Use the XML tab in the Options dialog to set global defaults for XML features for all modeling sessions.

To set XML options

  1. Click Options from the Tools menu.

    The Options dialog opens.

  2. Specify the following options on the XML tab:
    XML Import
    Always prompt to validate the XML file against the schema for erwin DM

    Enables display of the XML file validation dialog every time you attempt to open an XML file, which prompts you to choose whether or not you want to validate the XML file against the erwin DM XML schema.

    Always validate the XML file against the schema for erwin DM

    Specifies to not have the XML file validation dialog display when you attempt to open an XML file so that file validation always occurs before the XML file is opened.

    Never validate the XML file

    Specifies to not have the XML file validation dialog display when you attempt to open an XML file, and that file validation will not occur before the XML file is opened.

    Default: Always prompt to validate the XML file against the schema for erwin DM.

    XML Export
    Set the default format to the standard XML format for erwin DM

    Specifies to use the standard XML format which does not expand names or macros, but is suitable for importing back into erwin DM. When you select this option, the Set saving the minimum information needed as the default option is also selected so that the minimum amount of information needed by erwin DM to reopen the model is saved.

    Properties that are either calculated or that are read-only are not written out. You can choose to not use this option and still use the standard XML format.

    Set the default format to the erwin Repository XML format

    Specifies to use the erwin Repository format which expands names, macros, and inherited physical names.

    Once you save an XML file in this format it cannot be imported (opened) into erwin DM.

    Default: Set the default format to the standard XML format for erwin DM.

    Click OK.

    The default XML Options are set and the Options dialog closes.
