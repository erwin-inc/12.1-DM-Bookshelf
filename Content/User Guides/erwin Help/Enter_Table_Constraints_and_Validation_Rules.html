Enter Table Constraints and Validation Rules

Use the Validation tab in the Table Editor to add or modify table constraints and validation rules. Depending on your target DBMS, you may have other options available in this tab to work with constraints and validation rules.

To enter constraints and validation rules for a table

  1. Click Tables on the Model menu.

    The Table Editor opens.

  2. Select the table in the Navigation Grid for which you want to specify constraints and validation rules.

    Use the Enter filter text box to filter a very large list of tables to quickly locate the one that you need.

  3. Click the Validation tab and work with the following options:
    Validation

    Displays the constraint associated with a validation rule. You can use the toolbar to sort the validation rule list or to create or delete validation rules.

    Validation Rule

    Displays the validation rule associated with the constraint. The drop-down list contains all of the validation rules currently assigned to the table. You can select a different validation rule to associate with the constraint by clicking on the drop-down.

  4. Click Close.

    The Table Editor closes.
