Delete a Materialized View Column Expression

To delete a materialized view column expression

  1. In a physical model based on Oracle, right-click a materialized view in the diagram, and select Database Materialized View Properties.
  2. Click the Select option from the context menu.

    The Views Editor opens.

  3. On the Select tab, select the expression you want to delete from the Materialized View Columns list and click the Delete button.

    The expression is deleted from the Materialized View Columns list.

  4. Click Close.

    The Materialized Views Editor closes.
