Set Layout Preferences for a Diagram

You use the options in the Layout tab to set preferences for relationship line layout, entity and table sizing, object alignment, and diagram layout.

To set layout preferences for a diagram

  1. Click Diagrams on the Diagram menu.

    The Diagram Editor opens.

  2. From the Edit ER Diagrams in drop-down list, select the model or subject area that has the diagram for which you want to set layout preferences.

    The diagrams available in the selected model or subject area appear.

  3. Select the diagram for which you want to set layout preferences, and go to the Layout tab.
  4. Work with the following options:
    Manual Layout
    Allow Manual Relationship Layout

    Allows manual reshaping of relationship lines. This check box is selected by default. Clear this check box to disallow manual layout of relationship lines.

    Allow Manual Entity/Table Resizing

    Allows manual resizing of selected entities, tables, or views. This check box is selected by default. Clear this check box to disallow manual resizing of entities, tables, and views throughout the diagram.

    Grid Alignment
    Align Objects to Grid

    Aligns entities, tables, or views to the nearest vertical and horizontal gridlines. New objects "snap" to the grid automatically. Existing objects are not automatically aligned with the grid unless they are copied, pasted, or moved. Clear this check box if you do not want entities to align with the layout grid (default).

    Alignment Grid Spacing

    Defines the distance in pixels between the gridlines. Enter a digit or click the arrows to increase or decrease the default value.

    Align Grid Color

    Defines the grid color. To change the default color, click the color pick icon () and pick a color.

    Visual Cues
    Show Page Grid

    Displays a grid for the diagram.

    Show Background

    Displays a background for the diagram. Clear the check box to have a white background.

    Show Callout

    Displays callouts for the relationship texts in the diagram.

    Selection Grips Color

    Defines the color of the grip that surrounds a selected object. To change the default color, click the color pick () icon and pick a color.
