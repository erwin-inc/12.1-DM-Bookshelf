Add Text to a Drawing Object

Drawing objects enhance the appearance of your model and add meaningful graphic or text annotations. You can add text to a drawing object. You use the Drawing Objects editor to type drawing object text. You use the Text tab in the Drawing Objects editor to type or copy and paste the text that you want to include on the diagram window for the selected drawing object.

To add text to a drawing object

  1. Right-click a drawing object, and then select Properties.

    The Drawing Objects dialog opens.

  2. On the Text tab, enter the text in the space provided and click OK.

    The text is added to the drawing object.

You can define the default settings for font and size by applying a theme using the Shape Editor.
