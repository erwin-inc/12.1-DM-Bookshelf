Reorder Index Members

Use the Table Index Editor to reorder index members for a table in the physical model.

You cannot use the Table Index Editor to reorder columns in a foreign key index.

To reorder index members

  1. Right-click the index in the Model Explorer for which you want to reorder index members and click Properties.

    The Table Index Editor opens.

  2. Click the Members tab.
  3. Select each index member in the Index Members list and click the up or down arrow buttons to move the index member up or down in the list. You can click the Reset Order button to open the Reset Order dialog to specify one of the following options:
    Reset Index Member Order from Key Group Member Order for Index <index name>
    Reset Index Member Order from Key Group Member Order for All Indexes of the Table <table name>
    Reset Index Member Order from Key Group Member Order for all Indexes of the model
  4. Click Close.

    The index members are reordered and the Table Index Editor closes.
