Sort the Valid Value List

The validation rule values can be sorted in any order you need.

To sort the valid value list

  1. ClickValidation Rules on the Model menu.

    The Validation Rule Editor opens.

  2. Select a validation rule in the Navigation Grid for which you want to sort valid values list and click the General tab.
  3. Click the Sort button on the Valid Values toolbar and select one of the following sort options:

    Specifies that the valid value list is sorted in creation order

    Specifies that the valid value list is sorted in alphabetic order.

    Specifies that the valid value list is sorted in reverse alphabetic order.

  4. Click Close.

    Valid values are sorted and the Validation Rule Editor closes.
