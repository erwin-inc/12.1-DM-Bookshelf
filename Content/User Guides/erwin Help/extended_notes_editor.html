The Extended Notes Editor

Use the Extended Notes Editor to add or update an extended note. You can add extended notes for both physical and logical model objects.

Follow these steps:

  1. Open the property editor for an object, go to the Extended Notes tab, and click Edit.

    The Extended Notes Editor dialog appears and displays the following fields:

    Enter filter text

    Searches for the extended notes that includes the keyword.

    Transfer

    Marks the extended note for transfer to the Notes tab. To transfer a note, click the Transfer between notes and extended notes icon in the toolbar. When you transfer a note from the Extended Notes tab to the Notes tab, the following changes occur:

    • The Name, Importance, and Status information is deleted.
    • The value in the Author field is deleted and the current user is displayed in the Created By field.
    • The value in the Event field is deleted and User Note is displayed.
    Name

    Specifies the name of the extended note.

    Event
    Status

    Displays the property inheritence icon for the extended notes that are inherited from the parent table. This column is available only in the Column Editor dialog.

    Comment

    Displays the comment that is entered in the Extended Notes tab. This column also displays the note that is transferred from the Notes tab.

    Migrate

    Migrates the extended note from a parent table to the child table. For example, in a table named User, you have added a note to the column User_ID and selected the Migrate check box. When you use User_ID as a foreign key in another table, the note is displayed. This column is available only in the Column Editor and Domain Editor dialogs.

    Author

    Specifies the name of the user that entered the extended note. The user name that has currently logged in to the computer is displayed.

    Importance

    Specifies the importance that you would like to provide to this extended note.

    Note Status

    Specifies the status for the extended note.

    Modified Time

    Specifies the date and time at which the extended note was created or last modified.

  2. Click New to add a new extended note.
  3. Use the remaining toolbar icons to sort, delete, and move the extended notes up or down.
