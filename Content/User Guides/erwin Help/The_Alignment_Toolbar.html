The Alignment Toolbar

Use options on the alignment toolbar to align objects on the Diagram window. When you align objects, they are not automatically spaced. For instance, if you select several entities on the same horizontal line and click Align Left, the entities will automatically layer on the diagram window.

Align Left

Aligns the objects based on the left boundary of the leftmost object that you select first.

Align Right

Aligns the objects based on the right boundary of the rightmost object that you select first.

Align Center

Aligns the objects horizontally based on the center of the first object that you select.

Align Top

Aligns the objects based on the top boundary of the topmost object that you select first.

Align Bottom

Aligns the objects based on the bottom boundary of the bottommost object that you select first.

Align Middle

Aligns the objects vertically based on the center of the first object that you select.
