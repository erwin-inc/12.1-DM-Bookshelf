Specify That a Domain Appears in the Logical Model Only

You can indicate that a selected domain displays only in the logical model.

You cannot apply this feature to the default domains (<default>, Blob, Datetime, Number, or String).

To specify that a domain that appears in the logical model only

  1. Click Domains on the Model menu.

    The Domain Editor opens.

  2. Select the domain in the Navigation Grid that you want to only appear in the logical model, select the check box in the Logical Only column, and click Close.

    The logical-only specification is set and the Domain Editor closes.
