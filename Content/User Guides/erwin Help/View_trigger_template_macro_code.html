View Trigger Template Macro Code

You can use the Trigger Template dialog to view trigger template macro code.

To view trigger template macro code

  1. In the physical model, click RI Trigger Templates on the Database menu.

    The Trigger Templates dialog opens.

  2. Select the trigger template that you want to view in the Built-In Trigger Template or User Override list.

    The trigger template macro code displays in the Template Code box. Maximize the box to view the code more easily.

  3. Click Close.

    The Trigger Templates dialog closes.
