Change Physical Storage Object Parameters

Use the physical property editors to change storage parameters as needed during model development.

To change physical storage object parameters

  1. Click Target - <database> on the Model menu and then select an option for the physical object that you want to change (such as Tablespace, Rollback Segment, Database).

    The property editor for the physical object opens.

  2. Select the object in the Navigation Grid for which you want to change storage parameters.
  3. Click the General tab and change the value as needed for each parameter.

    Any parameters you leave blank are assigned a default value by the target server. For more information about setting physical storage parameters, see your SQL vendor's documentation.

  4. Work with the other tabs to further change or define storage parameters.
  5. Click Close.

    Storage parameters are changed for the physical storage object and the property editor closes.

Limited synchronization (name only) is supported for physical storage objects. If you import a physical storage object and rename it, you cannot export the name change. However, you can export the information to create a new object. erwin® Data Modeler does not compare or synchronize index physical properties, table physical properties, or synonyms.
