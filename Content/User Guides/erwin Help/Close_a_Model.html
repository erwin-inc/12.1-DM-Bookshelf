Close a Model

When you finish working on a erwin® Data Modeler model either locally or from the mart, you can close the current model and continue working on another model. When you close the model you are working on, the Close Model dialog opens and provides various options for saving and closing the model. This dialog shows the status of the model file, such as if it was opened from the server or its lock status. The Close Model dialog opens when you close a model in any of the following ways:

  • File/Close from the application menu
  • Close model when closing the diagram window
  • Close button in the Complete Compare wizard

To save a model to the mart, you must be connected to the mart.

To close a model

  1. Click Close on the File menu.

    The Close Model dialog opens.

    Important: When you close a model that does not have any changes, it is immediately closed. However, if the model was opened during Complete Compare, then it is displayed regardless of whether there are changes.

  2. Work with the following options:
    Mark/Unmark selected model for activation

    Specifies whether to activate the model.

    This option is not available unless you are exiting the application and are closing multiple open models after performing a Complete Compare.

    Mark/Unmark selected models for closing

    Specifies whether to close the selected model.

    This option is not available unless you are exiting the application and are closing multiple open models after performing a Complete Compare.

    Mark/Unmark selected models for saving

    Specifies whether to save the selected model when there are unsaved changes. Clear the check box next to the model name to close the model without saving the changes.

    Invert (selected rows become unselected, unselected rows become selected)

    Inverts the selections on the list.

    Browse for a directory path

    Opens the Save As dialog so you can select a directory in which to save the model. If the Save check box was not selected, it is set.

    Browse for a Model Manager path

    Opens the Save erwin Model Manager Model As dialog so you can select a library in which to save the model on the mart. If the Save check box was not selected, it is set.

    This option is not available unless you are connected to the mart.

    Save

    Specifies to save the model. By default, this check box is selected when there are unsaved changes. Clear the check box to close the model without saving the changes.

    Specifies whether to unlock a model that resides in the mart after changes are made to it.

    Default: Selected if the model was opened from the mart in the Locked state and changes were made to the model. Not selected if the model was opened from the mart in the Unlocked state and changes were made to the model.

    When you save and close a model opened from the mart in the Locked state, the lock on the model is released.

    Model Name

    Lists the name of the open model.

    File Name

    Lists the file name of the open model. This field is the same as the Model Name when the Use Default Name check box is selected.

    Model Path

    Shows the path of each open model.

    Use default names

    Specifies to use the model name as the file name, if the file name has not been set.

  3. Click OK.

    The Close Model dialog closes and the model is closed.
