How the Cursor Behaves in the Model Explorer

When you drag model objects in the Model Explorer, the cursor changes to indicate the default action that you are performing. You can change the default operation manually by pressing another key while dragging as long as the manual change is a legal operation for the selected object.

The following lists how you can change the default behavior:

Action

To change the default to this action

Move

Press Shift while dragging

Move as a member of subject area from a model

Press Ctrl and Shift while dragging

Copy as a member of subject area from another subject area

Press Ctrl while dragging
