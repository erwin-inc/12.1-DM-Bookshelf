Views in the Model Explorer

The Model tab in the Model Explorer lists each view in a model. Each view is listed in alphabetical order. When you right-click the Views folder or a view in the Model Explorer, a shortcut menu appears.

The shortcut menu for the Views object includes the following option:

New

Creates a new view in the Model Explorer and in the diagram.

The shortcut menu for a specific view includes the following options:

Go To Diagram

Switches to the diagram focus to the selected view.

Delete

Deletes the selected view from the model.

Properties

Opens the View Editor.
