The Link Wizard

The Link Wizard is provided to let you create and manage Link objects, and define Link object properties. The Link Wizard is launched from within the Link tab of either the Attribute Editor or the Column Editor.

The Link Wizard includes the following pages:

Overview

Describes the functionality of the Link Wizard.

Link

Lets you define the name and definition of a Link object, and specify the synchronization direction of either the attribute or column associated with the Link object.

Target

Lets you select or create a target attribute or column to which to link.

Properties

Lets you specify the properties to synchronize.
