Defining View Column Properties for Target Databases

In a physical model, the View Column Editor contains options for defining common view column properties such as physical data type and expression and are not database-specific.

However, more options are provided for some of the target databases that require you to define additional properties. The following target databases have additional, database-specific options for defining view column properties:

Db2 for z/OS

Hive

Oracle

PostgreSQL

SQL Server
