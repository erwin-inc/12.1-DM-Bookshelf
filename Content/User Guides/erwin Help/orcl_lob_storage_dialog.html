Oracle LOB Storage Dialog

Open the Oracle LOB Storage dialog by clicking on the LOB Storage button in the Oracle Partition Description Dialog. The following describes the options for the Oracle LOB Storage dialog:

Segment Name

Enter a value for segment name in the grid.

Columns

When you click on a cell under Columns, it opens the Oracle Select Columns dialog from which you can select a column from those in the current model.

Parameters

Add a LOB parameter by clicking on the in this column. Opens the Oracle LOB Parameters Dialog where you can set parameters for LOB storage.

LOB storage is supported in the following features:

  • Forward Engineering
  • Reverse Engineering
