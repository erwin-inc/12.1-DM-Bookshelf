Oracle Table and Materialized View Editor - General Tab

Use the General tab to edit the physical properties of a table or materialized view.

The options available to you vary slightly depending on whether you are in the Oracle Table Editor or the Oracle Materialized View Editor.

Oracle Table Editor

The following describes the options for the General tab in the Oracle Table Editor:

Physical Properties

Select this check box to assign physical properties, and select one of the following options:

Regular

In addition to selecting the Regular option you can also select Table Compression. If you also select Table Compression, select one of the following table compression options:

  • COMPRESS
  • NO COMPRESS

Click the Storage Properties button to open the Oracle Storage Properties Dialog to assign specific storage properties.

Index Organization

If you select this option, click the Storage Properties button to open the Storage Properties Dialog to assign specific storage properties. Click the IOT Properties button to open the Oracle Index Organized Table dialog to assign index organized table properties.

External Organization

If you select this option, click the External Properties button to open the Oracle External Table Dialog to assign properties to an external table.

Cluster

If you select this option, you can click the Cluster Properties button to open the Select Cluster Dialog.

Global Temporary

If you select the Global Temporary check box, you can also select the following:

On Commit

Select one of the following On Commit options:

  • PRESERVE ROWS
  • DELETE ROWS

Oracle Materialized View Editor

The following describes the options available for the General tab in the Oracle Materialized View Editor:

Physical Properties

Select this check box to assign physical properties, and select one of the following options:

Regular

In addition to selecting the Regular option you can also select Table Compression. If you also select Table Compression, select one of the following table compression options:

  • COMPRESS
  • NO COMPRESS

Click the Storage Properties button to open the Oracle Storage Properties Dialog to assign specific storage properties.

Index Organization

If you select this option, click the Storage Properties button to open the Storage Properties Dialog to assign specific storage properties. Click the IOT Properties button to open the Oracle Index Organized Table dialog to assign index organized table properties.

Cluster

If you select this option, you can click the Cluster Properties button to open the Select Cluster Dialog.
