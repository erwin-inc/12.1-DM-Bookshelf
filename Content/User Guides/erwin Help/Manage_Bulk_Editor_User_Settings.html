Manage Bulk Editor User Settings

Use the User Settings page in the Bulk Editor Wizard to manage settings and preferences for the Bulk Editor.

Changing item order in the Bulk Editor grid is done within the Bulk Editor, not the Bulk Editor Wizard.

To manage Bulk Editor user settings

  1. Click the button on the Bulk Editor toolbar.

    The Bulk Editor Wizard opens.

  2. Click User Settings and work with the following options:
    Automatically update the current selection set

    Specifies whether changes you make to the selection set are shown the next time you enter the wizard.

    Include Owner Path column for better model context

    Specifies that in addition to the standard Name and Type columns, an Owner Path column be added to the Bulk Editor that shows the names of each object in the hierarchy of objects that own the current object. For example, having an Owner Path column present in the Bulk Editor can be useful when there is one column or index used in multiple tables, and you want to know the table to which that column or index belongs.

    Always auto-check Name property

    Specifies to always automatically select the check box that is located next to the Name property on the Property Types page if you are working with a Logical or Logical/Physical model.

    Always auto-check Physical Name property in Physical model

    Specifies to always automatically select the check box that is located next to the Physical Name property on the Property Types page if you are working with a Physical or Logical/Physical model.

    Limit width of wide data columns on initial display

    Specifies to restrict the width of columns that contain lengthy text to 250 pixels upon the initial display in the Bulk Editor. All other columns are sized to match the best width so that a column's entire title and widest data value is displayed.

    Once the data is displayed in the Bulk Editor, you can resize the columns as needed by dragging the divider between column headings, or double-clicking on that divider to have the column automatically resized to match its contents.

  3. Click Edit.

    User settings for the Bulk Editor are set and the Bulk Editor Wizard closes.
