Defining Avro Fields

The following properties apply to an Avro record field.

Tab

Property

Description
General Domain Parent Specifies the domain parent of the selected field
Namespace Specifies the name of the field's namespace
Doc Specifies the JSON string describing the field
Physical Name Specifies the physical name of the field
Physical Data Type Specifies the physical data type of the selected field
Logical Type

Specifies the logical data type of the selected field
Default Specifies the default value for the selected field
Union Data Type Specifies one or more primitive data types that apply to the field
