Drawing Objects

Drawing objects enhance the appearance of your model and add meaningful graphic or text annotations. For example, if you want to prepare a model for analysis and presentation, you can circle the tables on which you want to focus. Utilize drawing objects in your model in the following ways:

  • Move drawing objects by dragging them onto the diagram window.
  • Layer multiple drawing objects and then arrange the order as needed. For example, you can use the Bring to Front command to easily rearrange multiple drawing object layers.
  • Select multiple drawing objects and then use the align commands in order to position the objects on the diagram window.
  • Add text to a drawing object, change the text properties, or change the fill color or line color for a drawing object, by using the Drawing Objects editor.
  • Establish the default font and color settings for drawing objects, just as you can for other model objects, by using the Default Fonts & Colors editor.
