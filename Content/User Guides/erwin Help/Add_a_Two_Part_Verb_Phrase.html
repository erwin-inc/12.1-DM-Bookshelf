Add a Two-Part Verb Phrase

You can add a two-part verb phrase in a logical model to describe the relationship that you create between a parent and child entity.

To add a two-part verb phrase

  1. Click Relationships on the Model menu.

    The Relationships editor opens.

  2. Select the relationship from the Relationship drop-down list to which you want to add a two-part verb phrase, then click the General tab.

    The General tab opens.

  3. Enter a verb phrase in the following fields, then click OK.
    Parent-to-Child

    Enter the verb phrase to describe the parent-to-child relationship.

    Child-to-Parent

    Enter the verb phrase to describe the child-to-parent relationship.

    The two-part verb phrase is added to the model and the Relationships editor closes.
