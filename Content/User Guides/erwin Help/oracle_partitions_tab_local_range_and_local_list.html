Oracle Partitions Tab - Local-Range and Local-List

The following describes the options available when you select the Local-Range partitioning type in the Partitions tab of either the Oracle Table Index Editor or the Oracle Materialized View Index Editor:

Partitions

Displays the partitions. The following describes the information in this grid:

Partition Name

Enter a partition name in this column.

Edit Properties

When you click the button in this column, it opens the Oracle Partition Description dialog where you can modify the properties for the selected partition.

The Partitions grid is populated with the number of rows equal to the number of partitions for the table or materialized view to which the index belongs. If you delete or change any partitions for the table or materialized view (from the Oracle Table Editor or the Oracle Materialized Views Editor), the grid will refresh to reflect those changes.
