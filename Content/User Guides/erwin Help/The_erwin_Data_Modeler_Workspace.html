The erwin Data Modeler Workspace

The erwin Data Modeler workspace includes the following components:

  • Welcome Page
  • Quick Access Toolbar
  • Theme Options
  • Ribbon
  • Toolbars
  • Model Explorer
  • Diagram Window
  • Other Panes
    • Properties
    • Objects Count
    • Action Log
    • Advisories
    • Bulk Editor
  • Status Bar

You use the components of the workspace to perform logical or physical modeling using IDEF1X, IE or DM (Dimensional Modeling) notation.
