Transfer Model to erwin DM NoSQL

Once you have connected to erwin DM NoSQL, to transfer your model in the XML format, do the following:

  1. Go toTools > erwin DM NoSQL group.
  2. Click Send Data ().

A Data Transfer dialog box is displayed, which displays the progress of the data transfer operation.

Ensure that you save the model before transferring it to erwin DM NoSQL.

Once the data transfer operation is completed, you can view your model in one of the following ways:

  • By visiting erwin DM NoSQL. For more information, refer to erwin DM NoSQL documentation.
  • By launching erwin DM NoSQL in erwin Data Modeler
