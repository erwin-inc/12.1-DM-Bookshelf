Oracle Materialized View Column Editor

Use the following options and tabs in the Oracle Materialized View Column Editor to modify properties for a column in a materialized view. You open this editor by right-clicking the materialized view on the diagram that contains the column you want to manage, and select Column Properties.

Materialized View

Displays the materialized view to which the column belongs. You can select a different materialized view using the drop-down control.

Previous

Positions the editor on the previous column in the Navigation Grid.

Next

Positions the editor on the next column in the Navigation Grid.

Sort

Sorts the columns by alphabetic, reverse alphabetic, or physical order. You select the method you want using the drop-down menu that opens after you click the Sort button. Visual cues are provided for how the column list is currently sorted:

Specifies that the list is sorted in alphabetic order.

Specifies that the list is sorted in reverse alphabetic order.

Specifies that the list is sorted in physical order.

New

Creates a new column that becomes the current object in the editor.

Delete

Deletes the selected column.

DB Sync

Opens the DB Sync wizard so you can perform a database level compare of objects in the current model with matching objects in another model.

Reset order

Lets you specify one of the following options to reset column order:

Reset order to match table column physical order

Resets column order so it matches the table column physical order in the model.

Reset all

Opens the Reset Order dialog so you can specify the scope and type of the column order reset actions. Click OK when you are done with your selections.

Specifies to move the selected column to the top of the list.

Specifies to move the selected column up one level in the list.

Specifies to move the selected column down one level in the list.

Specifies to move the selected column to the bottom of the list.

Opens the Reset Column Properties dialog so you can reset properties to restore inheritance for the selected column.

Help

Opens online help for the editor.

Lets you filter a very large list of columns in the Navigation Grid to quickly locate the column for which you need to view or define properties.

Name

Specifies the name of the column. You can change the name of the column in this field.

Domain Parent

Specifies the domain parent of the column.

Physical Data Type

Specifies the physical data type for the column.

Expression

Specifies whether the column is a user-defined expression.

General

Lets you assign a domain to a materialized view column.

Expression

Lets you edit the selected materialized view column expression.

Indexes

Lets you view and assign index membership.

Style

Lets you manage themes and display options for the column.

Where Used

Lets you view where the column is used within the model.

Comment

Lets you enter any comments that you want to associate with the object.

UDP

Lets you enter user-defined property values for the object.

History

Lets you view the history information of the object.

Notes

Lets you view or edit user notes.
