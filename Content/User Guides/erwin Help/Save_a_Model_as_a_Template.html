Save a Model as a Template

You can save any model as a template to save its settings and contents for quick reuse.

To save a model as a template

  1. Do one of the following:
    • On the File tab, click Save As.
    • On the Home tab, click Save, and then click Save As.
    • On the Quick Access Toolbar, click the Save arrow, and then click Save As.

    The Save As dialog box opens.

  2. In the Save as Type list, select the template type (*.erwin_tmpl).
  3. In the File Name box, enter the template name.
  4. Click Save.

    The model is saved as a template and the dialog closes.

Saving a model using the *.erwin_tmpl extension creates a model template. You can use this template to create models, but any changes you make to the template are not propagated to the models that use it. If you want to be able to synchronize changes in a model template with the models that use it, use the bind model template functionality.
