Define Oracle Script Templates

Use the Oracle Script Template Editor to create or modify script templates.

To define an Oracle script template

  1. Click Pre & Post Scripts on the Database menu.

    The Oracle Script Template Editor opens.

  2. Click New on the toolbar.

    A new script template is added to the list.

    Use the navigational buttons (Previous, Next, Sort, New, Delete, Browser) to select another action.

  3. Enter the values and select the options for the new script template for the following items and click Close:
    Name

    Specifies the name of the script template. Change the name of the script template in this field.

    Type

    Specifies to generate the script template for a model or a table.

    Generate As

    Specifies to generate the script template pre-creation or post-creation.

    Generate

    Lets you select whether or not to generate DDL for this object during Forward Engineering. If you do not select the check box, no DDL will be generated for this object during Forward Engineering.

    Complete the following on the Code tab:

    Code

    Lets you review or assign code to a script template.

    Your changes are saved and the Oracle Script Template Editor closes.

You can optionally review the code assigned to the script template in the Expanded tab and then click the Comment or UDP tab to add a comment or UDPs. Click Cancel to close the dialog without saving your changes.
