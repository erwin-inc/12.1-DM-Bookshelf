Preview the FE Script

Use the SQL tab in the Table Editor to preview the forward engineering (FE) script of a table and the corresponding comments, partitions, and physical storage.

To preview the FE script for a table

  1. Click Tables on the Model menu.

    The Table Editor opens.

  2. Select the table in the Navigation Grid for which you want to preview the FE script.

    Use the Enter filter text text box to filter a very large list of tables to quickly locate the one that you need.

  3. Click the SQL tab to preview the FE script.
  4. Click Close.

    The Table Editor closes.
