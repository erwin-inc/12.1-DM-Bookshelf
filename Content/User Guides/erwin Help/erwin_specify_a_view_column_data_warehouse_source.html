Specify a View Column Data Warehouse Source

You can use the Data Source tab in the View Column Editor to view and specify data warehouse source assignments.

Ensure that you have selected the Data Movement option in the Model Editor to enable data movement modeling features.

To specify a view column data warehouse source

  1. Right-click the view that contains the view column for which you want to specify a data warehouse source and select Column Properties.

    The View Column Editor opens.

  2. Select the view column in the Navigation Grid for which you want to specify a data warehouse source.
  3. Click the Data Source tab and work with the following options:
    Data Sources

    Lets you specify a data warehouse source to attach to the selected view column. Use the buttons on the toolbar to sort, arrange the list, restore inheritance, or open either the Data Movement Source Editor or the Data Sources Selector to work with data warehouse sources.

    DS Comment

    Lets you enter a comment to describe how the source is integrated into the data warehouse. Use the buttons on the toolbar to cut, copy, and paste comment text, restore inheritance, work with the Text Editor, or check spelling.

  4. Click Close.

    The view column data warehouse source is specified and the View Column Editor closes.
