Oracle Table Editor - Supplemental Logging Tab

Use the Supplemental Logging tab to modify properties for supplemental logging. The following describes the options available for this tab:

Log Groups

Displays the log groups. The following describes the information in this grid:

Name

Enter a value for a log group name.

Log Always

Select this check box for the log group if you want the Log Always option set.

Log Column

Click the button within the Log Column column to open the Oracle Attach Columns for Logging Dialog where you can select the log column.

Key ID

Select one of the following for the Key ID:

ALL
Primary Key
Unique
Foreign Key
