Disconnect from the Mart

When you finish working on the online mart models, disconnect from Mart to end the session.

When you disconnect from Mart, you can select whether you want to detach the model, save the model offline, or close the model.

Follow these steps:

  1. Open a model from the Mart.
  2. Click Mart > Disconnect.

    The Save Mart Models dialog opens. If you open multiple models, the dialog lists all the modified mart models.

  3. (Optional) Click Browse for a Mart Path.

    The Save As dialog opens.

  4. (Optional) From the Catalog, select a new location and click Save.

    The model is saved in the new location and the Save As dialog closes. The Save Mart Models dialog shows the new Model Path.

  5. In the Save Mart Models dialog, select the Save checkbox for the models that you want to save and click OK.

    The Save Mart Models dialog closes and the Mart Offline dialog opens. The Mart Offline dialog lists all the mart models that are currently open.

  6. Select the model, and from the Save to drop-down list, select your option.
    Offline

    Saves the model offline to the local directory. You can work on the offline model without connecting to the Mart. Every update to the offline model is recorded until you rebind the model to the Mart.

    Detach

    Copies the model and saves the copy to the local directory. The detached model is not connected to the mart and it is considered as a non-mart model.

  7. (Optional) If you have selected Offline or Detach, click Open Folder, and navigate to the required folder location to save this model.

    The Model Path field shows the new location of the model.

  8. Click OK.

    The Mart Offline dialog closes and the model is saved as an offline mart model.
