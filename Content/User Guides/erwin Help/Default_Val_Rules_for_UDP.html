Default Value Rules for a User-Defined Property

A default value is not required for a user-defined property, except for the Date type. The following rules apply to default values:

Command

Select a file or application that you want to launch when you assign a UDP.

Int, Real, and Text

Enter the default value you want to appear when you assign a UDP.

List

Enter default values since the List data type requires that you enter a series of list choices. Separate each choice by a comma (,) and denote a default list value by preceding the default list entry with a tilde (~).

Date

Enter a default date or click to scroll though dates by month, day, and year. You can also click to select a date from the calendar. If you do not specify a date, the default value is populated with the actual date.
