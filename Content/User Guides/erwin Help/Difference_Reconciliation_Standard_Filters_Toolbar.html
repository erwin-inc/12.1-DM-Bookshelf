Difference Reconciliation Standard Filters Toolbar

The standard filter buttons allow you to change the display of differences in order to refine the compare process. You can select any combination of filters. To show only the differences between the models, deselect the "Include Equal" and "Include Resolved Items" buttons and select the three remaining buttons.

Show Equal Items

Select this button to display objects and properties that are the same in both models.

Show Resolved Items

Select this button to display property and object rows that were resolved into equal rows during your Difference Reconciliation session. When you choose this action, the rows are marked with the change indicator color and the change indicator symbol in the status column.

Show Not Equal Items

Select this button to display objects and properties that are not the same in both models.

Show Unaligned Objects on the Right

Select this button to display objects that do not exist in the right model, but are present in the left model. This filter does not affect property rows.

Show Unaligned Objects on the Left

Select this button to display objects that do not exist in the left model, but are present in the right model. This filter does not affect property rows.

Show Conflict Items

Select this button to display objects that are in conflict.
