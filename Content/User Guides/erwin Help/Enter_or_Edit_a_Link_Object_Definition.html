Enter or Edit a Link Object Definition

Enter or edit a definition for a Link object on the Link tab of either the Attribute Editor or Column Editor. A Link object definition can consist of any text you find useful to document the Link object.

To enter or edit a Link object definition

  1. Open either the Attribute Editor or Column Editor as needed.
  2. Select the entity or table from the drop-down that contains the attribute/column that uses the Link object.
  3. Click the Link tab.
  4. Select the Link object in the grid for which you want to enter or edit a definition and click the button in the Definition column.

    The Link Definition editor opens.

  5. Enter the definition or edit an existing one.

    Use the toolbar to save the definition as a separate text file, print, search and replace text, cut, copy, or paste text, or check spelling.

  6. Click OK.

    You are returned to the property editor.

  7. Click Close.

    The Link object definition is saved and the property editor closes.
