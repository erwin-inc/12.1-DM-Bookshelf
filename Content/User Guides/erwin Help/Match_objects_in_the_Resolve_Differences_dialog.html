Match Objects in the Resolve Differences Dialog

The Complete Compare process automatically matches objects as it compares the two models. If you want to override the automatic matching, you can force a match using the Match feature. When you match objects, they appear together on the same row in the Object View row so you can see the differences in the property sets.

To match objects in the Resolve Differences Dialog

  1. In the Resolve Differences dialog, select an object in the Object View that you want to match with another object.

    The Match icon becomes active in the standard toolbar.

  2. Click the Match icon.

    The cursor changes to a double-ended arrow.

  3. Click the object you want to match with the object you selected first.

    A confirmation message is displayed.

  4. Click Yes.

    The item is moved to the same line in the Differences list.

The toolbar buttons are available or unavailable (dimmed) depending on whether or not the action is available for the selected item. For example, if you select a table in the left model list and it already matches a table in the right model list, the Match operation is not completed.
