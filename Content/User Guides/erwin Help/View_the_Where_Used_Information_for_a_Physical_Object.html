View the Where Used Information for a Physical Object

Use the Where Used tab in a physical property editor to view where an object is used in a model.

To view where used information

  1. Open the property editor for the object for which you want to view its where used information, and select the object in the Navigation Grid.
  2. Click the Where Used tab and work with the following options:
    Sort

    Sorts the list by alphabetic, reverse alphabetic, alphabetic by object type, name, or reverse alphabetic by object type, name. You select the method you want using the drop-down menu that opens after you click the Sort button. The Sort button image changes to indicate how the list is currently sorted.

    Edit

    Lets you edit the properties of a selected object in the Where Used list. Select the object and click the button to open the selected object's property editor.

    Name

    Displays the name of the object used by the selected object in the editor.

    Type

    Displays the object type.

    Parent

    Displays the parent object of the selected object.

    Parent Type

    Displays the parent object type.

  3. Click Close.

    The property editor closes.
