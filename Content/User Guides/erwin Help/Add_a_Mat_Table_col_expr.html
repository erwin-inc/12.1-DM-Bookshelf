Add a Materialized Query Column Expression in the Materialized Query Tables Editor

You can use the Materialized Query Tables editor to add a materialize query column expression.

To add a materialized query column expression in the materialized query tables editor

  1. In a physical model based on Db2, right-click a materialized query table, and select Materialized Query Table Properties.

    The Materialized Query Tables dialog opens.

  2. Click the Select tab, then in the lower section of the dialog, click New.

    The New Materialized Query Table Columndialog opens.

  3. Complete the following fields:
    Name/Alias

    Defines the name of the materialized query column expression.

    Expression

    Defines the expression text.

  4. Click OK to add the new materialized query column.

    You are returned to the Materialized Query Table editor.
