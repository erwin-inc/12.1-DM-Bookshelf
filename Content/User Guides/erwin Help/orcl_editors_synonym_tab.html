Oracle Editors - Alias Tab

Use the Alias tab to create, edit, or delete synonyms that apply to the current editor. For example, if you click on the Synonym tab in the Oracle Stored Procedure editor, any synonym that you create, edit, or delete affects the Stored Procedure that you selected in the Oracle Stored Procedure Editor.

The Alias tab is identical for the Oracle property editors that use it, as listed below:

  • Function, Materialized View, Package, Sequence, Stored Procedure, Table, and View editors

The following describes the options available for this tab:

Name

Shows the synonyms for the selected object in tree format. Select the Synonym that you want to edit.

New

Opens the New Synonym dialog where you can specify the name for the new synonym.

Rename

Opens the Rename Synonym dialog where you can assign a different name for the synonym.

Delete

Select a synonym from the Synonym list box that you want to delete, and click the Delete button.

Owner

You can enter a name in this text box to assign a database Owner to the selected synonym. You cannot specify an owner for a public synonym.

Public

Creates a public synonym. A public synonym can be accessed by every user in a database.

Generate

Generates DDL for this object during Forward Engineering. If you do not select the check box, no DDL will be generated for this object during Forward Engineering.

For model-level synonyms, use the Oracle Synonym Editor.
