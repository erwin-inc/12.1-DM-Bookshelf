Change Font and Color for a Drawing Object

You can select a different font and color for a specific drawing object. In other words, you can override the any settings that the object has inherited from its parent object (for example, diagram).

To change the font and color for a drawing object

  1. Right-click a drawing object and select Override Fonts and Colors.

    The Style tab of the Shape Editor dialog opens.

  2. Work with the following options:
    Theme

    Specifies the theme that you want to apply to the selected drawing object. Click to edit the theme, or click to create a new theme.

    If you select values for the options that are listed under Local overrides to Theme, it means that these values will be used instead of the values mentioned in the theme.

    Font Options

    Specifies the default font styles for the selected drawing object. Click to change the font style details. You can also choose to inherit, override, or harden this setting.

    Fill Options

    Specifies the default fill color for the selected drawing object. Click to change the fill style details. You can also choose to inherit, override, or harden this setting.

    Line Options

    Specifies the default line styles for the selected drawing object. Click to change the outline style details. You can also choose to inherit, override, or harden this setting.

    Outline Options

    Specifies the default outline styles for the selected drawing object. Click to change the outline style details. You can also choose to inherit, override, or harden this setting.

  3. Click Close.

    The changes are applied to the selected table and the Shape Editor dialog closes.
