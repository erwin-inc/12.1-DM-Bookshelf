Pre and Post Scripts Editor - Code Tab

Use the Code tab to review or assign code to a script. The following describes the options for the Code tab:

Unattached Scripts

Displays all scripts in the current model. Use the arrow buttons to move an unattached script to the Attached Scripts box.

Attached Scripts

Displays the first line of the script code for an attached script. You can use the buttons to move an attached script back to the Unattached Scripts box.

Code

Displays the script template code, which can include SQL statements and erwin Data Modeler macros.

Macro Toolbox

Opens the Macro Toolbox where you can select an macro and double-click it to add the macro to the script.
