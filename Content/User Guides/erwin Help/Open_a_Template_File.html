Open a Template File

You can open a forward engineering template file (*.fet) and use the features in the Template Editor to add or modify its templates, review template source and expanded text, insert macros, print template source, and so on.

To open a template file

  1. Select Forward Engineer, Forward Engineering Templates from the Actions menu (in either a physical or logical/physical model).

    The Template Editor opens.

    When the editor is invoked, it attempts to select the Create Entity template by default.

  2. Click the Open Template File icon.

    The Open dialog opens.

  3. Search for the template file you want to open and click Open.

    The Open dialog closes and the template file opens in the Template Editor. Template source displays in the Template Source pane, and the Expanded Text pane displays the processing status. You can now use the features in the Template Editor to work with the templates and apply template changes to the template file.
