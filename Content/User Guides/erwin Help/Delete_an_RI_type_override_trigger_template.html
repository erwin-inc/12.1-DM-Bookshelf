Delete an RI Type Override Trigger Template

Use Model Explorer to delete an RI type override trigger template you previously created. To delete an RI type override trigger template, right-click the RI type override trigger template in the Model Explorer that you want to delete and click Delete. The RI type override trigger template is removed from the Model Explorer.

You can delete custom templates, but you cannot delete the built-in erwin® Data Modeler templates.
