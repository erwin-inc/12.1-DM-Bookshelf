Select the Source for a Materialized View

To select the source for a materialized view

  1. In a physical model based on Oracle, right-click a materialized view in the diagram, and select Database View Properties.
  2. Select the From option from the context menu.

    The Views Editor at the From tab opens.

  3. Select one of the following options:
    • To include the selected table, view, or materialized view in the From list, double-click the object in the Available Tables/Views/MaterializedViews list.
    • To remove the selected table, view, or materialized view from the From list, select the object in the From list and click the button.
  4. Select a table, view, or materialized view from the From list, and click or to move the selected table, view, or materialized view up or down one position in the list. Repeat this step until the From list shows the order you want.
  5. Click OK.
