Select the Model Objects From the Source Model to Link

You use the options in the Source Model Object Selection pane of the Link Model Source wizard to specify which model objects from the source model participate in the link process.

To select the source model for the link process

  1. Click Design Layers, Link Model Source on the Actions menu.

    The Link Model Source wizard opens on the Source Model pane.

  2. Work with the following options:
    Load From File or erwin Model Manager

    Select a model in a mart or a local file. You can load a file of any type listed on the "Files of type" drop-down list. Click Load to browse your file system and indicate a renamed or changed model as your source model.

    Set selected model as read-only

    Changes will not be saved to the current model.

  3. Click Next.

    The Type Selection pane of the Link Model Source wizard opens.
