Table Addition Methods

You can add an independent table and dependent table using any of the following methods:

  • Select in the Toolbox and click the diagram window.
  • Add an entity in the logical model, and open the physical model.
  • Reverse engineer an existing database.
  • Copy a table from one model and paste it into another model.
  • Synchronize a model with a database or script file.
  • Right-click the Tables icon in the Model Explorer and select New.

You can add consecutive multiple tables by pressing the Ctrl key before you click the table tool. When you click again in the diagram window, another table is created.
