Summary of On-Diagram Editing

The on-diagram editing features can be summarized as follows:

To...

Then…

Edit an entity, attribute, table, column, or view name

Select and press F2.

Open the object editor

Double-click or select and press Alt+Enter.

Save changes in an object editor

Enter

Cancel changes in an object editor

Esc (Escape)

Insert a new attribute or column

Select and press Enter.

Edit an annotation

Select and press F2.

Move to the next object in the same entity

Tab or Down Arrow

Move to the previous object in the same entity

Shift+Tab or Up Arrow

Create a copy of an entity or table

Ctrl+Left-click+drag-and-drop

Move an object to another entity or table

Left-click+drag-and-drop

Save your changes to the diagram object

Ctrl + Enter

Cancel your changes

Esc

Diagram/No Selection

Clear all selections

Click the empty space on the diagram

Open the context menu

Right-click or Shift + F10

Move one increment in the desired direction

Up, Down, Left, or Right arrow

Move one page in the desired direction

Shift + Up, Down, Left, or Right arrow

Move to the edge of the binding rectangle in the desired direction

Ctrl + Up, Down, Left, or Right arrow

Move one page down

PgDn or Shift + Down arrow

Move one page up

PgUp or Shift + Up arrow

Move on page up and left

Shift + Home

Move to top left of bounding rectangle

Ctrl + Home

Move on page down and left

Shift + End

Move to bottom left of binding rectangle

Ctrl + End

Move on page up and right

Shift + PgUp

Move to top right of binding rectangle

Ctrl + PgUp

Move on page down and right

Shift + PgDn

Move to bottom right of binding rectangle

Ctrl + PgDn

Move object

Drag

Enter

Esc

Tab

Shift + Tab

Move to the previous or next character

Left or Right arrow

Shift + Left or Right arrow

Move to the beginning of the text

Home

Select the text from the current cursor position to the beginning of the text

Shift + Home

End

Shift + End
