Delete a Custom Macro Category

You can delete a custom macro category that you created if you no longer need it.

You cannot delete the categories provided with this product.

To delete a custom macro category

  1. Select Forward Engineer, Forward Engineering Templates from the Actions menu (in either a physical or logical/physical model).

    The Template Editor opens.

  2. Click the Edit Macro Categories icon on the Template Editor toolbar.

    The Macro Categories dialog opens.

  3. Ensure Category View is selected, select the custom category that you want to delete and click the Delete button.

    A dialog opens prompting you to confirm the deletion.

  4. Click Yes in the dialog to confirm deletion

    The macro category will not be deleted until you click OK in the Macro Categories dialog.

    The dialog closes and the macro category is removed from the Categories list.

  5. Click OK.

    The Macro Categories dialog closes.

  6. Click Close.

    The Template Editor closes.
