Entity Editor

Use the following options and tabs in the Entity Editor in a logical model to add a new entity or modify the properties of an existing entity.

Previous

Positions the editor on the previous entity in the Navigation Grid.

Next

Positions the editor on the next entity in the Navigation Grid.

Sort

Sorts the entities by alphabetic or reverse alphabetic order. You select the method you want using the drop-down menu that opens after you click the Sort button. Visual cues are provided for how the entity list is currently sorted:

Specifies that the entity list is sorted in alphabetic order.

Specifies that the entity list is sorted in reverse alphabetic order.

New

Creates a new entity that becomes the current object in the editor.

Delete

Deletes the selected entity.

Help

Opens online help for the editor.

Lets you filter a very large list of entities in the Navigation Grid to quickly locate the entity for which you need to view or define properties.

Name

Specifies the name of the entity. You can change the name of the entity in this field.

Logical Only

Specifies whether the entity is suppressed from a physical model as a table and appears in a logical model only.

Volumetrics Tab

Lets you enter sizing information including growth estimates.

Use the Volumetrics dialog to view and report table and database size calculations.

Definition Tab

Lets you enter any comments that you want to associate with the entity.

Style Tab

Lets you specify the styles for the entity. The styles are used to format the font and other graphical themes used in the display of the entity.

Icon Tab

Lets you specify the icon to use for the entity.

UDP Tab

Lets you enter user-defined property values for an entity.

History tab

Lets you view the history information for the entity.

Notes Tab

Let you view or edit user notes.
