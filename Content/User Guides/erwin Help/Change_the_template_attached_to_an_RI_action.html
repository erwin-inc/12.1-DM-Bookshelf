Change the Template Attached to an RI Action

You can use the Trigger Template Editor in a physical model to change the template attached to an RI action.

You must select the RI Type Override check box for triggers during schema generation to generate an RI type override template in the schema.

To change the template attached to an RI action

  1. Click RI Trigger Templates on the Model menu.

    The Trigger Template Editor opens.

  2. Click the Referential Integrity Type tab.
  3. Select the RI action to which you want to attach a different trigger template and select a new trigger template using the drop-down control.
  4. Click Close.

    The Trigger Template Editor closes.

To reattach the system default trigger template to the selected referential integrity type, click the Rebind button.
