Specify a Progress Server Validation Error Message

Use the General tab in the Validation Rule Editor to specify a validation error message when your target server is Progress. The text you enter displays as an error message when an invalid value is entered in the associated column.

To specify a server validation error message

  1. ClickValidation Rules on the Model menu.

    The Validation Rule Editor opens.

  2. Select the validation rule in the Navigation Grid for which you want to specify a validation error message.
  3. Click the General tab and enter a text message in the Error Message box.
  4. Click Close.

    The Progress validation error message is specified and the Validation Rule Editor closes.
