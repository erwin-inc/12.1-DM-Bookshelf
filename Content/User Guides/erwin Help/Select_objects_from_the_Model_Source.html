Select Objects From the Model Source

You use the features in the Source Model Object Selection pane to select the model objects to copy from the model source to the current model.

To select objects from the source model

  1. Click Design Layers, Add Model Source on the Actions menu.

    The Add Model Source Wizard opens on the Source Model pane.

  2. Click the Object Selection link in the Explorer pane.

    The Source Model Object Selection pane opens.

  3. Work with the following options:

    Choose Objects Using Sets

    Indicates that you plan to use the preset options for selecting objects sets, such as Subject Areas, Indexes, Triggers, or objects by owner. The available sets appear in a tree in the wizard. Clear this check box if you plan to use a customized set of selected options from the object list.

    Selected Objects

    Displays the tree of model objects for the source model. Select or clear the check box next to the object name. Selected objects are included in the derived model.

    If you create a customized list of objects from the object list, then select the Choose Objects Using Sets option, the tree of model objects for the source model is reset to the default.

  4. Click Next.

    The Naming Standards pane opens.

You can click the Add button at any time to add the model source.
