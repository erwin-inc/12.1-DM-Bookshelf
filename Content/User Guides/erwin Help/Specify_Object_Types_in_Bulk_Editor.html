Specify Object Types to Use in the Bulk Editor

Use the Object Types page in the Bulk Editor Wizard to select the object types that populate the Bulk Editor during the current editing session. The object types in the model are listed in a tree with icons next to them, and their names match with the names in the Model Explorer.

The object types that are selected here affect the contents of the Property Types and the Object Instances pages. Only valid properties for the selected object type populate the Property Types page. Similarly, objects of the selected object types populate the Object Instances page.

From release r9.6 onward, you can edit the comments that are entered in the Extended Notes tab. To include comments in the Bulk Editor, select the Extended Notes object type on this page.

Follow these steps:

  1. Click on the Bulk Editor toolbar.

    The Bulk Editor Wizard opens.

  2. Click Object Types and work with the following options:
    Option Set

    Specifies the option set to use to populate the selections in the Bulk Editor Wizard. An option set contains the definition of the objects and properties that are selected in a previous session. Use the toolbar icons to load, save, create, or delete an option set.

    Use Selected Objects

    Specifies to limit the list of object types to only those objects that are selected in the current diagram. Otherwise, all object types are shown.

    Names

    Opens a menu so you can select to show object names as either logical names or physical names. When you select one of the options, the toolbar button changes to provide you a visual indication of your choice:

    Logical Names

    Physical Names

    Physical Names, show owner

    Physical Names, show owner using User

    The Logical Names option does not display when you are working in either a logical-only or a physical-only model.

    Select All

    Selects the check boxes for all the items in the list.

    Select None

    Removes all check box selections so that no items are selected in the list.

    Toggle Selection

    Specifies to select check boxes for any unselected items and remove check box selections for selected items.

  3. Click Edit.

    Object type selections are specified for the Bulk Editor and the Bulk Editor Wizard closes.
