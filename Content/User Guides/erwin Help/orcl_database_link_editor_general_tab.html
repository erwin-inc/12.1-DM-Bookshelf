Oracle Database Link Editor - General Tab

Use the General tab to assign the connection type to a database link. The following describes the options for the General tab in the Oracle Database Link Editor dialog:

Shared

Select this option if you want a shared connection to the database link. When you select this option, an authenticated user is required, and can be any one of the three Connection Types.

Connection Type is automatically selected and grayed out because it is not possible to deselect it for this option.

Public

Select this option if you want a public connection to the database link. Select either CURRENT USER or FIXED USER. No user authentication is available.

If you do not select either Shared or Public, then the database link is considered Private, and no user authentication is available.

Connection Type

Specify the connection type using one of the following options:

CURRENT USER

This is the default connection type when the Shared option is selected. This is one of two available connection types if you selected the Public option.

  • Authenticated User Enter a name in this text box. This option is not available if you selected the Public option, or if you made no selection.
FIXED USER

This option is available if you selected either Shared or Public. Select this option to assign a fixed user as the connection type and enter a value in the following, as appropriate:

  • User This field is required when you select FIXED USER.
  • Authenticated User If you selected the Shared option, enter a value in the Authenticated User field. This option is not available if you selected the Public option, or if you made no selection (neither Shared or Public).
AUTHENTICATED USER

This connection type is only available if you selected Shared.

  • Authenticated User Enter a name in this text box.
Service Name

Enter Service Name information in this field.
