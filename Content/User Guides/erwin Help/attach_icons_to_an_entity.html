Attach Icons to an Entity

Use the Icon tab in the Entity Editor to attach small or large icons to an entity in the logical model.

To attach icons to an entity

  1. Click Entities on the Model menu.

    The Entity Editor opens.

  2. Select the entity in the Navigation Grid to which you want to attach an icon.
  3. Click the Icon tab and work with the following options:
    Small Icon

    Specifies the small icon bitmap (*.bmp) that displays when you select to display the Entity Icon on the Display Style Sheet. Select an icon from the drop-down list or click New to open the Image Property Editor where you can import, rename, or delete an image.

    Large Icon

    Specifies the large icon bitmap (*.bmp) that displays when you select to display the Entity Icon on the Display Style Sheet. Select an icon from the drop-down list or click New to open the Image Property Editor where you can import, rename, or delete an image.

  4. Click Close to save your changes.
