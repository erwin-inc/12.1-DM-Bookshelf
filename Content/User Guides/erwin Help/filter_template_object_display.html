Filter the Display of Model Template Objects

A erwin® Data Modeler model contains many objects. When you bind a model template to a model or create a template, all model objects display in the Model Template dialog. If you want to filter the object display so that it is a shorter list, make your selections in the Define Type Filter dialog. A shorter list makes it easier for you to define the model objects that you want in your template.

Follow these steps:

  1. On the Home tab, click Model Templates.
  2. Click Define Filter.

    The Define Type Filter dialog box opens.

  3. Select or clear the check boxes for the objects that you want to appear on the Model Template dialog box. You can also use the toolbar to select or clear all check boxes.
  4. Click OK.

    Template object display is defined and the Define Type Filter closes.

You can also open the Define Type Filter dialog from within the Model Template Synchronization Wizard as you are working with template objects.
