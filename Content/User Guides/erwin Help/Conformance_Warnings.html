Conformance Warnings

Conformance warnings are used to let you know if you are violating dimensional modeling standards. By default, a conformance warning displays if the role you manually assign conflicts with dimensional modeling standards which dictate that:

  • A fact table cannot have any parent relationships.
  • A fact table cannot be the child of an outrigger table.
  • A fact table must have a child relationship.
  • An outrigger table cannot have a parent relationship to a fact table.
  • An outrigger table must have a parent relationship.
  • The child of an outrigger table must be a dimension table.
