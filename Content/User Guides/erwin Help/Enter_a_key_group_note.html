Enter a Key Group Note

You use the Notes tab in the Key Group Editor to enter user notes for a selected key group in a logical model. User notes are for documentation purposes only at the object level, and do not migrate as comments when you derive a physical model from a logical model, unlike a definition.

To enter a key group note

  1. Right-click the entity that contains the key group for which you want to enter a note and click Key Group Properties.

    The Key Group Editor opens.

  2. Select the key group in the Navigation Grid for which you want to enter a note and click the Notes tab.
  3. Click the New button on the toolbar.

    The Key Group Notes editor opens.

  4. Enter note text. You can use the options on the toolbar to cut, copy, or paste text, open the text editor, or check spelling.
  5. Click Close.

    Note text is saved and the Key Group Notes editor closes.

  6. Click Close.

    The Key Group Editor closes.
