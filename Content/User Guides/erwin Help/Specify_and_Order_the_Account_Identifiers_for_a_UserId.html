Specify and Order the Account Identifiers for a User Id

The Accounts tab on the Teradata User Id Editor lets you enter account identifier for a user ID.

To specify and order the account identifiers for a user ID

  1. Click Target-Teradata, Users on the Model menu.

    The Teradata User Id Editor opens.

  2. Click the Accounts tab.

    A grid of the account ids that are available for this user id is displayed. You can add and order the account names and ids.

  3. Use the toolbar buttons to arrange and add account names and ids into the desired sequence.

    The account names are a list of user strings that have a maximum length limit of 30 characters. When they are forward engineered, single quotes must surround Teradata strings.

  4. Click Close.

    The Teradata User Id Editor closes.
