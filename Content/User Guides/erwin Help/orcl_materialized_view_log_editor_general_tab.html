Oracle Materialized View Log Editor - General Tab

Use the General tab to assign general options for the materialized view log. The following describes the options available for this tab:

Filter Columns

Select the column on which you want to filter by selecting columns from the Available Columns list and adding them to the Selected Columns list.

Available Columns

To add a column to the Selected Columns list, select it and click the button.

Selected Columns

To remove a column from this list and place it back into the Available Columns list, select the column and click the button.

For more information on these parameters, see your vendor's SQL documentation.
