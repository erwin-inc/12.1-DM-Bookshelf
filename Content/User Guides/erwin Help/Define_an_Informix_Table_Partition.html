Define an Informix Table Partition

Use the Partitions tab in the Informix Table Editor to define a table partition.

To define an Informix table partition

  1. Click Tables on the Model menu.

    The Informix Table Editor opens.

  2. Select the table in the Navigation Grid for which you want to define a partition.

    Click New on the toolbar to create a new table. Use the Enter filter text box to filter a very large list of tables to quickly locate the one that you want to define.

  3. Click the Partitions tab and work with the following options:
    Sort

    Sorts the partitions by alphabetic or reverse alphabetic order. Click the Sort button to specify how to sort the list.

    New

    Creates a new table partition.

    Delete

    Deletes the selected table partition.

    Name

    Displays the partition names for the current table.

    Expression

    Lets you specify the expression to use to select different rows. Click the button to open a text editor to work with expression text.

    IN_DBSPACE

    Lets you assign the dbspace for the partition. Click New to add a new dbspace or click the button to open the Informix DBSpace Editor to work with an existing dbspace.

  4. Click Close.

    The Informix table partition is defined and the Informix Table Editor closes.
