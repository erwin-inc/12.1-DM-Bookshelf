Rename an Attribute on the Diagram Window

You can change an attribute name directly on the diagram window. You can also change the entity name and other attributes in the same entity by using the Tab key or the Down Arrow key.

To rename an attribute on the diagram window

  1. Click and the click the attribute name that you want to change.

    The Edit box opens.

  2. Use the Backspace or Delete key to erase a part of or the entire name, and then enter the new name. Click Shift+Enter or click anywhere outside the entity box.

    Your changes are saved.

After you edit an attribute name, click the Tab key to move to the next attribute name in the entity.
