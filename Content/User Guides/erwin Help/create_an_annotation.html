Create an Annotation

You can create annotations for your model. Use the Annotation editor to add the text and indicate formatting options.

To create an annotation

  1. Click the annotation tool in the erwin toolbox and then click a location on the workplace where you want the annotation to appear.

    The annotation is created, and an active cursor appears in the annotation box.

  2. Enter your annotation text in the annotation box. Click anywhere outside the annotation box when you are finished. You can select and drag the edge of the annotation box to change its size in any direction.
  3. Right-click the annotation and select Properties to open the Annotation Editor. Work with the following options:
    Name

    Specifies the name of the annotation.

    Type

    Specifies the shape object that contains the annotation. Select a shape object (such as rectangle) from the drop-down list.

    Physical Only

    Specifies that you want to annotation to appear only in the physical model (for Logical/Physical models only).

  4. Work with your annotation text on the Text tab. A toolbar with cut, copy, paste, expanded text editing, and spell check options is provided.
  5. (Optional) Click the Style tab to select an Optional Theme or to create a Theme for the annotation.
  6. (Optional) Click the Where Used tab to review where the annotation has been used.
  7. (Optional) Click the UDP tab to work with user-defined properties for the audit policy.
  8. Click Close to save your changes and close the editor.

You must use the Annotation Editor to edit an annotation. In-place editing is not supported for annotations.
