Define a Db2 LUW Server Option

Server options indicate the options that are enabled when the server definition is created. You access the Db2 for LUW Server Option Editor when you click the UDB Server Option Editor icon in the Server Options tab of the Db2 for LUW Server Editor.

Follow these steps:

  1. Enter a name for the server option in the Name column.

    For example, enter the CPU_Ratio, Node.

  2. Enter a value for the server option in the Value column.

    For example, enter 2.0, Comp001.

  3. (Optional) Click the Commenttab and enter any comments that you want to associate with the object.
  4. (Optional) Click the UDPtab to work with user-defined properties for the object.
  5. (Optional) Click the Notestab to view and edit user notes.
  6. (Optional) Click the Extended Notes tab to view or edit user notes.
