The Materialized Query Table Editor

The Materialized Query editor includes all the features you need to create, modify, and maintain Db2 materialized query tables.

The following functionality is available on the editor tabs. Click the hypertext link on the tab name to jump to a procedure related to the features in that tab.

  • Click the Select tab to specify the materialized query columns, and create a column reference to any table in your model.
  • Click the From tab to specify the source tables for a materialized query table. You move an available table or view to the pool of tables/views you want to use in a materialized query table.
  • Click the Clauses tab to specify search conditions for generating the SQL code for a materialized query table.
  • Click the General tab to define general properties for a selected materialized query table.
  • Click the Query Options tab to work with query options for the selected materialized query table.
  • Click the Distribution tab to work with distribution options.
  • Click the Tablespace tab (Define a Db2 z/OS Tablespace, Define a Db2 LUW Tablespace) to select a tablespace to store the selected table.
  • Click the Organize By tab to work with organization options.
  • Click the Partitions tab to specify the partition type and elements.
  • Click the User Defined SQL tab to view and change a user-defined DDL statement.
  • Click the SQL tab to view the SQL code for the selected materialized query table.
  • Click the Alias tab to work with alias names for the selected materialized query table for Db2 LUW. Click the Alias and Synonym tab to work with synonym and alias names for a Db2 z/OS table.
  • Click the Permission tab to add and define permissions at the server level.
  • (Optional) Click the Style tab and format the font and other graphical themes used in the display of the columns.
  • (Optional) Click the Icon tab and specify the icon that you want to use for the table.
  • (Optional) Click the Comment tab to enter any comments you want associated with the materialized query table.
  • (Optional) Click the UDP tab to work with user-defined properties for the materialized query table.
  • (Optional) Click the History tab to view the history information of the materialized query table.
  • (Optional) Click the Notes tab to view and edit user notes.
  • (Optional) Click the Extended Notes tab to view or edit user notes.
