Add a Table

Adding a table is one of the first physical objects you add when you create a new physical model.

To add a table

  1. Click in the Toolbox and then click anywhere on the diagram workspace.

    A table displays on the diagram workspace. The table name area is active.

  2. Enter a name for the table and press Shift+Enter.

    The table name displays above the table on the diagram workspace.
