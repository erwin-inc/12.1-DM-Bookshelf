Oracle Partitions Tab - Composite-List

The following describes the options available when you select the Composite-List partitioning option in the Partitions tab:

Template

When you click the Template button, it opens the Oracle List Subpartition Dialog.

Partition Columns

Displays the partition columns. Click the to open the Select Columns Editor where you can select the columns to add to the Partition Columns list.

Subpartition Column

A drop-down list box where you can select the subpartition column.

Partitions

Displays the partitions in the table. The following describes the information in this grid:

Partition Name

Enter a value for a new Partition Name. Additional columns display in this grid when there are columns displayed in the Partition Columns list. The column names of the Partition Columns list are the additional column headings within this grid. Click within a cell under a column name and a drop-down appears where you can assign a value.

Edit Properties

To edit tablespace properties for a partition, click the button within the Edit Properties column to open the Oracle Partition Description dialog.

Subpartitions

Click on the within the Subpartitions column to open the Oracle List Subpartition Dialog.

Composite-list partitions are supported in the following features:

  • Forward Engineering
  • Reverse Engineering
