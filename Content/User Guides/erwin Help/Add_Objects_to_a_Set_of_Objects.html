Add an Object to a Set of Objects

You can add an entity, table, view, drawing object, or subtype relationship to a group of objects.

To add an entity, table, view, drawing object, or subtype relationship to a group of objects, click the Select icon on the erwin toolbar and press the Ctrl key, and then click each additional object that you want to select.

Subtype relationship symbols are treated as unique model objects. To include subtype relationships in a selection group, you must use a lasso to select them or select them individually. When you use the Lasso tool to select model objects, any objects that are not completely enclosed in the lasso will not be factored into the selection.

For all relationships, the relationship is automatically selected only if you select both the parent and child entity, table, or view in the relationship. You cannot copy and paste a relationship independent of its corresponding parent and child entities, tables, or views.

You also cannot select individual attributes or columns for a Copy function from the Edit menu; the entire entity or table to which they belong can only be selected in this way. Individual attributes or columns can only be copied to another entity, table, or view within the same model using the drag-and-drop method.
