Change Page Margins for Printing

By default, the entire page is used for printing. When you print a model, all the supported page sizes for all the printers installed on your computer are listed. You can select the page size required for the current diagram and set separate page margins.

To change the page margins

  1. Click Page Setup on the File menu.

    The Page Setup Editor opens.

  2. Click the Margins tab and enter a new value (in inches) in any of the margin text boxes:
    Top Margin

    Specifies the margin between the page header and the diagram area.

    Header Margin

    Specifies the margin for the header.

    Left Margin

    Specifies the margin between the left border and the diagram area.

    Right Margin

    Specifies the margin between the right border and the diagram area.

    Footer Margin

    Specifies the margin for the footer.

    Bottom Margin

    Specifies the margin between the page footer and the diagram area.

    To enter a fraction, place the cursor after the decimal point and enter the decimal equivalent.

  3. Click Close to save your settings, or Cancel to cancel your selections.
