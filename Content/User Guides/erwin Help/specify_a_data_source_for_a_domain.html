Specify a Data Source For a Domain

When you work with Dimensional Modeling notation in the physical model, you can use the Data Source tab in the Domain Editor to specify a data warehouse source for a selected domain.

Use this feature to document your data warehouse assignments and transform information for each domain in your dimensional model. All source documentation is inherited by the columns attached to the domain. You can override any source properties on a column-by-column basis in the Table Column Editor.

Select the Data Movement check box on the General tab of the Model Editor to activate this feature. Click Model Properties on the Model menu to open the Model Editor.

To specify a data source for a domain

  1. Click Domains on the Model menu.

    The Domain Editor opens.

  2. Click the Data Source tab and work with the following options:
    Data Sources

    Lets you specify a data warehouse source to attach to a column. Use the buttons on the toolbar to sort, arrange the list, restore inheritance, or open either the Data Movement Source Editor or the Data Sources Selector to work with data warehouse sources.

    DS Comment

    Lets you enter a comment to describe how the source is integrated into the data warehouse. Use the buttons on the toolbar to cut, copy, and paste comment text, restore inheritance, work with the Text Editor, or check spelling.

  3. Click Close.

    The data source is specified for the domain and the Domain Editor closes.
