CSV File Column Description Format

You must format the column descriptions in the CSV file that you want to import as specified below. Otherwise, you will not be able to import the file.

Unique ID

A unique number for a column

Column Type

Must always be set to "DWXC"

Diagram ID

Must always be set to 0 (zero)

Table ID

A unique identifier for the table that contains this column

Table Type

Must always be set to "DWXT"

Column Sequence Number

A number that identifies the order in which columns appear in the Detail tab of the Data Warehouse Sources dialog; the range is 0 to n

Column Name

The name of the column

Data Type

The data type of the column

Comment

The comment associated with a column
