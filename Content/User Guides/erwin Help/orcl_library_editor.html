Oracle Library Editor

Use the following options and tabs in the Oracle Library Editor to define Oracle library properties. You can open the Oracle Library Editor by selecting Target - Oracle, Library on the Model menu.

Previous

Positions the editor on the previous library in the Navigation Grid.

Next

Positions the editor on the next library in the Navigation Grid.

Sort

Sorts the libraries by alphabetic or reverse alphabetic order. You select the method you want using the drop-down menu that opens after you click the Sort button. Visual cues are provided for how the library list is currently sorted:

Specifies that the list is sorted in alphabetic order.

Specifies that the list is sorted in reverse alphabetic order.

New

Creates a new library that becomes the current object in the editor.

Delete

Deletes the selected library.

DB Sync

Opens the DB Sync wizard so you can perform a database level compare of objects in the current model with matching objects in another model.

Help

Opens online help for the editor.

Lets you filter a very large list of libraries in the Navigation Grid to quickly locate the library for which you need to view or define properties.

Name

Specifies the name of the library. You can change the name of the library in this field.

Owner

Lets you assign a database Owner to the library.

Use Replace Syntax

Lets you specify whether the library DDL that is generated during Forward Engineering to replace any occurrence of an identically-named library. If this check box is not selected, the library DDL generated will only attempt to create the object and will not attempt to replace it.

Generate

Generates SQL during forward engineering. Clear the check box if you do not want to generate SQL.

General

Lets you specify a database link for a library.

Comment

Lets you enter any comments that you want to associate with the object.

UDP

Lets you enter user-defined property values for the object.

Notes

Lets you view or edit user notes.

Libraries are supported in the following features:

  • Forward Engineering
  • Reverse Engineering
