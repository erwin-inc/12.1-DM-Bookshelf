Add a New Script

Use the Script Template Editor to add a new script in a physical model.

To add a new script

  1. Click Script Templates on the Model menu.

    The Script Template Editor opens.

  2. Click New on the toolbar.

    A new template is added in the Navigation Grid using a default name. You can change the name in this field.

  3. Work with the options on the Code and Object Creation Order tabs to further define the script.
  4. Click Close.

    The script is added and the Script Template Editor closes.
