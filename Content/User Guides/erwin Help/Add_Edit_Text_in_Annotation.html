Add or Edit Text in an Annotation

You use the Annotation Editor to add or edit text in an annotation.

To add or edit text in an annotation

  1. Right-click an annotation and click Properties.

    The Annotation Editor dialog opens.

  2. In the Text tab, enter the text you want to display in the annotation.
  3. (Optional)To include line breaks, press the Enter key.
  4. Use the following toolbar options to refine the text:
    Cut ()

    Cuts the selected text and saves in the clipboard.

    Copy ()

    Copies the selected text and saves in the clipboard.

    Paste ()

    Pastes the text that is most recently saved in the clipboard.

    Edit ()

    Opens the Shape Text editor and displays the text available in the Text tab.

    Spell Check ()

    Checks the spelling of the text available in the Text tab.

  5. Click Close.

    The Annotation Editor closes.
