Add a Physical Storage Object Using the Model Explorer

You can add a physical storage object to your model using the Model Explorer.

To add a physical storage object

  1. Right-click a physical storage object in the Model Explorer and click New. The physical storage objects that display depend on the target server you have selected. For example, if you select Oracle as your target server, you can right-click Clusters, Databases, Libraries, Tablespaces, Rollback Segments, Sequences, and so on.

    A new physical storage object is added.

  2. Enter a name for the physical storage and press Enter.

    The physical storage object is added to the Model Explorer and the model diagram. Physical storage objects display in alphabetical order.
