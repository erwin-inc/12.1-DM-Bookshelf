Copy a Drawing Object

You can copy a drawing object in the diagram in which you create it or you can copy a drawing object to one or more diagrams in your model. For example, if you create a company standard legend, you may want to copy the legend to multiple diagrams.

To copy a drawing object

  1. Select the drawing object that you want to copy and click Copy Object from the Edit menu.

    The drawing object is stored in the clipboard.

  2. Click Paste from the Edit menu.

    The drawing object is copied over the original. You must select it and drag it to another location.
