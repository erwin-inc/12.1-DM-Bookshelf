Validation Rules Dialog for Oracle

Use the Validation Rules dialog to create, modify, or delete validation rules. The following describes the options available on the Validation Rules dialog when your target server is Oracle:

Validation Name/Validation Rule

Lists the name of existing validation rules and the actual syntax that appears in the schema when the rule is generated.

New

Opens the New Validation Rule dialog where you can specify the name for the new validation rule.

Rename

Opens the Rename Validation Rule dialog where you can assign a different name to the validation rule.

Delete

Lets you select a validation rule from the Validation Name list box to delete.

The Validation Rules dialog also includes the following tabs:

General Lets you create a validation rule.

ORACLE Specifies the validation expression for Oracle.

Comment Specifies comments for a validation rule.

UDP Specifies a user-defined property value for a validation rule.
