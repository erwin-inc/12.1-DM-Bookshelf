Delete a Script for a Table or View

During modeling activities, it may necessary to delete a script from a table or view. To delete a script, click the table or view script template in the Model Explorer and click Delete. The script is removed from the table or view on the diagram and in the Model Explorer.

Rename a Script for a Table or View

During modeling activities, it may necessary to change a script name. To rename a script, click the table or view script template in the Model Explorer and type over the old name with the new name.
