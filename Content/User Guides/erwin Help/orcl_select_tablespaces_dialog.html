Oracle Select Tablespaces Dialog

The Oracle Select Tablespaces Dialog opens when you click the button that is next to the Tablespace list box when you have selected the Hash partitioning option in the Partitions tab of either the Oracle Table Editor, Oracle Materialized Views Editor, or the Oracle Materialized View Log Editor. This dialog allows you to select the tablespace that you want as the partition. The following describes the options for the Oracle Select Tablespaces dialog:

Tablespaces

Displays the tablespaces in the current model. Select the tablespace that you need, and click the button to move the selected tablespace over to the Selected Tablespaces list.

Selected Tablespaces

Displays the tablespaces that you selected and moved from the Tablespaces list. If you decide that a tablespace does not belong in the Selected Tablespaces list, click on it and use the button to move it back to the Tablespaces list.

Tablespaces

Opens the Oracle Tablespace Editor where you can modify the properties for the selected tablespace.
