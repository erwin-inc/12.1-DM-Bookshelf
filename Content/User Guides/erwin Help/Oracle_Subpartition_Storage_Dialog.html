Oracle Subpartition Storage Dialog

Use the Oracle Subpartition Storage Dialog to assign tablespaces for subpartition storage. The following describes the options for the Oracle Subpartition Storage Dialog:

Tablespace

Displays the available tablespaces in the current model. Click the Tablespaces button to open the Oracle Tablespace Editor to add or modify tablespaces.

Overflow

(Index organized tables only) Assigns an overflow tablespace. When you select this option, select your overflow tablespace from the drop-down list.

LOB Items

LOB items appear in the grid. Enter a value for Segment Name. Click on a cell in the Columns column, and the Oracle Select Columns Dialog opens where you can select the columns that you need. Select a tablespace by clicking on the down arrow to display a list of available tablespaces. Click on a cell in the Parameters column and the Oracle LOB Parameters Dialog opens where you can assign the parameters for LOB storage.
