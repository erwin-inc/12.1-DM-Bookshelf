Enter a Comment for the Current Schema Generation Option Set

Use the options on the Comment tab in the Schema Generation dialog to enter a comment for the current option set.

To enter a comment for the current schema generation option set

  1. Click Forward Engineer, Schema... on the Actions menu.

    The Schema Generation dialog opens on the Options tab.

  2. Select the name of the current option set and choose from the following options on the Comments tab:
    Comment

    Enter a comment for the option set.

    Cut

    Cuts the text.

    Copy

    Copies the text.

    Paste

    Pastes the text.

    Open

    Opens the Text Editor.

    Click Generate to start the schema generation process for forward engineering or alter script generation or OK to save the changes and close the dialog.
