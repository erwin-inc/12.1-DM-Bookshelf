Reorder the Columns Referenced by a View

If you need to reorder the columns referenced by a view in the physical model, you can do this in the View Editor.

To reorder the columns referenced by a view

  1. Right-click the view in the Model Explorer for which you want to reorder columns and click Properties.

    The View Editor opens.

  2. Click the Select tab.
  3. Select the column in the Columns list that you want to move and click either the up or down arrow button to move the column up or down one place in the list.
  4. Repeat the previous step until the Columnslist shows the order you want.
  5. Click Close.

    View columns are reordered and the View Editor closes.
