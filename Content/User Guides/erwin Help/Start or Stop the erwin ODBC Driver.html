Start or Stop the erwin ODBC Driver

When the erwin ODBC driver is active, you can run SQL queries against the current model using the erwin® Data Modeler Query Tool or third-party reporting tools such as Microsoft Excel.

Use the ODBC Driver section on the Reporting tab of the Options dialog to start and stop the erwin ODBC driver.

Follow these steps:

  1. Start erwin Data Modeler.
  2. On the ribbon, go to the Tools tab and click Options.
    The Options dialog opens.
  3. Click Reporting.
    Based on the status of the ODBC driver, one of the following options display in the ODBC Driver section:
    Start ODBC Driver for this instance
    Indicates that the ODBC driver for the current modeling session is stopped. Click Start ODBC Driver for this instance to start the ODBC driver.
    Stop ODBC Driver for this instance
    Indicates that the ODBC driver for the current modeling session is active. Click Stop ODBC Driver for this instance to stop the ODBC driver.
  4. Click OK.
