Execute a Query

You can execute a new or saved query against the model or against a database catalog.

Use the following process to execute a query:

  • Open the Query Tool from the Tools tab.
  • Create a new query or open an existing query (*.qry) file.
  • Connect to erwin ODBC or to a database.
  • (Optional) Select the option to restrict row count and enter a number to indicate the number of rows by which to restrict results.
  • Click the Execute icon on the toolbar.

After your query executes, the results display in the Results box. Use other features in the Query Tool to save the query or save your result set.

For sample queries and the details of schemas and tables, refer to the Useful Queries topic.
