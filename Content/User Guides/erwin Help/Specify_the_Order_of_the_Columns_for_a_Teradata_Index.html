Specify the Order of the Columns for a Teradata Index

You can use the Ordering tab to modify the order in which columns are displayed in an index. The Hash by Column grid shows all of the valid columns that you can select to participate in the hash. The Order by Columns grid shows all of the valid columns that can participate in the ordering.

To specify the order of the columns for a Teradata index

  1. Right-click an index and select Index Properties.

    The Teradata Table Index Editor opens.

  2. Click the Ordering tab.

    The Ordering tab opens.

  3. Select the check box next to the column name in the Hash by Column or Order by Column grid.
    Hash by Columns

    Displays the valid columns that you can select to participate in the hash. This property only displays for a table index.

    Order by Columns

    Displays the valid columns that you can select to participate in the ordering.

    Use the up and down arrow buttons to order the columns in each grid.

  4. Click Close.

    The editor closes.
