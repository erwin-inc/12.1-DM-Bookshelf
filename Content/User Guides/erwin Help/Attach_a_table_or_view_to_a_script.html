Attach a Table or View to a Script

You can use the Script Template Browser to attach a table or a view to a script.

To attach a table or view to a script

  1. Click Script Templates on the Model menu.

    The Script Template Editor opens.

  2. Select the Table Level script that you want to attach and click the Script Template Browser button .

    The Script Template Browser opens.

  3. Select a table or view in the Unattached Object list that you want to attach and click the right arrow button.

    To attach all tables and views in the Unattached Object list, click the double right arrow button.

    The tables and views that you selected to attach to the script display in the Attached Object list.

  4. Click Close.

    The Script Template Browser closes.

  5. Click Close.

    The Script Template Editor closes.
