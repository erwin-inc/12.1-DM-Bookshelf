Assign a Domain to a View Column

You can use the General tab in the View Column Editor to assign a domain to a view column. The Domain Parent list displays a hierarchical list of all the domains in the current model. By default, view columns are automatically attached to a domain, which is the set of properties of the source table or view column. Default naming is either <TableName.TableColumnName> or <ViewName.ViewColumnName>.

To assign a domain to a view column

  1. Right-click the view that contains the view column to which you want to assign a domain and click Column Properties.

    The View Column Editor opens.

  2. Click the view column in the Navigation Grid to which you want to assign a domain and select a domain from the Domain Parent drop-down.
  3. Click Close.

    The view column domain is assigned and the View Column Editor closes.
