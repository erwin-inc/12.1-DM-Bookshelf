Create a New Spell Check Dictionary

Create a new spell check dictionary to begin the work of creating a customized list of words and actions for your spell check.

To create a new spell check dictionary

  1. Click the Spell Check icon in the Comment tab of a property editor.

    The Check Spelling dialog opens.

  2. In the Check Spelling dialog, click the Dictionaries button.

    The Spelling Dictionaries dialog opens.

  3. Click the New Dictionary button and work with the following options:
    • Type a name for the new dictionary in the File Name box.
    • Select a language.
  4. Click OK to save your file, close the New Dictionary dialog and return to the Spelling Dictionaries dialog.

    Use features in the Spelling Dictionaries dialog to add words and actions to your dictionary.
