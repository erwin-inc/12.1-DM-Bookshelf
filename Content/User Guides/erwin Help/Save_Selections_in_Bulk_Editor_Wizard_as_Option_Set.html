Save Your Selections in the Bulk Editor Wizard as an Option Set

You can save object types, property types, and object selections as an option set so you can save time and not have to make the selections more than once.

To save Bulk Editor Wizard selections as an option set

  1. Click the button on the Bulk Editor toolbar.

    The Bulk Editor Wizard opens.

  2. Make your selections on the Object Types, Property Types, and Object Selection pages as needed then click the button on the option set toolbar.

    The Save Current Option Set dialog opens.

  3. Select one of the following options from the Save To drop-down control:
    <Current model name>

    Stores the option set with the current model. Enter a name for the option set in the Option Set Name field.

    This option set can be used only when you are working in this model. It saves Object Types, Property Types, and Object Selection page selections.

    XML File

    Stores the option set in an external XML file. Specify a file name and location in the Location field using the ellipsis button.

    This option set can be used in any model, however, only the Object Types and Property Types page selections are saved, not the Object Selection page selections.

  4. Click OK.

    The Save Current Option Set dialog closes.

  5. Click Edit.

    The option set is saved, the Bulk Editor Wizard closes, and the selections you made in the wizard are applied to the Bulk Editor.
