Define Constraints that are Associated with the Table

You can use the Validation tab to specify constraints on a table in a Teradata physical model.

Existing validation rules display in the Validation Rule column. You can use the Teradata Validation Rule Editor to create a new validation rule and apply it to a table.

To define validation rule usage for a Teradata table

  1. Click Tables on the Model menu.

    The Teradata Table Editor opens.

  2. Click the Validation tab.
  3. Click New on the Validation tab toolbar.

    The new Validation Rule with an erwin-generated name appears at the end of the list.

  4. Work with the following options:
    Validation

    Specifies the name for the usage of the validation rule by this table. Click the validation rule name to edit it.

    Validation Rule

    Specifies the validation rule to associate with the validation rule name. Create or select from the drop-down list.

  5. Click Close.

    The Teradata Table Editor closes.
