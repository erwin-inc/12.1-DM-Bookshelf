Define Db2 LUW Work Class Sets

Use the Db2 for LUW Work Class Set Editor to create sets from selected work classes. You then associate Work Class Sets with a Work Action Set.

This topic applies to Db2 for LUW versions 9.5 and 9.7 only.

To define a Db2 LUW work class set

  1. Right-click a Work Class Set in the Model Explorer and select Properties.

    The Db2 for LUW Work Class Set Editor opens.

  2. Select a work class set name and work with the following options:

    Click New on the toolbar to create a work class set name. Use the Enter filter text text box to filter a large list of names to locate the one you want to work with.

    Name

    Specifies the name of the work class set.

    Generate

    Specifies whether or not to generate DDL for this object during Forward Engineering.

  3. Click the Work Classes tab to select work classes you want to be a part of the selected work class set.
    Name

    Specifies the name of the work class.

    Type

    Specifies the work class type.

  4. (Optional) Click the Commenttab and enter any comments that you want to associate with the object.
  5. (Optional) Click the Where Used tab to view where the object is used within the model.
  6. (Optional) Click the UDPtab to work with user-defined properties for the object.
  7. (Optional) Click the Notestab to view and edit user notes.
  8. (Optional) Click the Extended Notes tab to view or edit user notes.
  9. Click Close.

    The Db2 for LUW Work Class Sets Editor closes.
