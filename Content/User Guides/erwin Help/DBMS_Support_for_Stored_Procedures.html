DBMS Support for Stored Procedures

Support for stored procedures is provided for the Db2 for i, Informix, MySQL, ODBC, Progress, SAS, SQL Server, SAP ASE, and SAP IQ target servers. Additional database-specific support for stored procedures is provided for these target servers:

  • Db2 for LUW and z/OS
  • Oracle
  • SQL Server
  • Teradata
