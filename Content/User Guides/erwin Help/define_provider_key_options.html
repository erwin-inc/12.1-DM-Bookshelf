Define Provider Key Options

Use the Provider tab to define the how you want to execute the cryptographic provider keys for SQL Server databases.

For more information targeted to your Oracle database environment, refer to your SQL reference documentation specific to your SQL Server version.

Follow these steps:

  1. Click Model, Target - SQL Server, Security, Symmetric Keys.

    The SQL Server Symmetric Key Editor opens.

  2. In the Navigation Grid, select the symmetric key for which you want to define the provider key options.
  3. Go to the Provider tab and work with the following options:
    Cryptographic Provider

    Specifies an Extensible Key Management (EKM) provider and name. Select the name of the provider from the drop-down list, or create a provider.

    Provider Key Name

    Specifies the name that you have included in the Extensible Key Management provider.

    Disposition

    Specifies whether you want to create a new key or map an existing symmetric key to an existing Extensible Key Management key.

  4. Click Close.

    The symmetric key is defined and the editor is closed.
