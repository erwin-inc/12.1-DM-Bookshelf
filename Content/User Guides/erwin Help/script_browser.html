Script Template Browser

Use the Script Template Browser to see the tables and views that are attached to a script and view the code that is used by a script. You can also attach tables and views in a model to a script. The following describes the fields used by the Script Template Browser:

Template Name/Template Code

Displays the names of all the available table-level script templates and their template code.

Unattached Object

Displays the available tables and views to which you can attach a table-level script template.

Attached Object

Displays the list of tables and views that are currently attached to the selected table-level script template.

Template (Read Only)

Displays the code for the selected table-level script template.

Template Expansion (Read Only)

Displays the code of the table-level script template as it will appear in the physical schema.
