Edit Macro Categories

You can work with features in the Macro Categories dialog to create new macro categories, add existing macros to existing categories, or add existing macros to new categories.

Disabled macros displayed in the Macro Categories dialog indicate hard-coded membership in its selected category. You cannot change category assignments for these macros.

To edit macro categories

  1. Select Forward Engineer, Forward Engineering Templates from the Actions menu (in either a physical or logical/physical model).

    The Template Editor opens.

  2. Click the Edit Macro Categories icon in the Template Editor toolbar.

    The Macro Categories dialog opens.

  3. Select either the Category View or Macro View option:
    Category View

    Changes the display to show the alphabetical list of categories and the macros that belong to each category.

    Macro View

    Displays the alphabetical list of macros and the categories to which they belong.

    This display changes based on your selection.

  4. Select or clear the check boxes in the Macros list to include or exclude a macro from the selected category display and click OK.

    Your selections are changed and the Macro Categories dialog closes.
