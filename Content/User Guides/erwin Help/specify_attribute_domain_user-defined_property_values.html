Specify Attribute Domain User-Defined Property Values

You can specify logical user-defined property values in the UDP tab of the Domain Editor for both non-inheritable and inheritable UDPs in a logical model. You work with UDPs you created in the User Defined Properties : Logical editor.

To specify attribute domain user-defined property values

  1. Click Domains on the Model menu.

    The Domain Editor opens.

  2. Click the name of the domain in the Navigation Grid for which you want to specify user-defined property values.

    To filter a large list of domains, enter the name (or the first few characters) of the domain for which you want to specify user-defined property values in the Enter filter text box.

  3. Click the UDP tab.
  4. Click the Value cell of the property for which you want to specify a value and enter the value according to the property's data type (List, Integer, Real Number, Text, Command, or Date) for either Domain User Defined Properties or UDPs Inherited by Attributes.

    If no UDPs are defined, click to open the User Defined Properties Editor. Click the button to reset a selected property to restore inheritance.

  5. Click Close.

    UDP values for the domain are saved and the Domain Editor closes.
