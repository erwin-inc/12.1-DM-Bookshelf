Oracle Materialized View Log Editor

Use the Oracle Materialized View Log Editor to modify the properties for a materialized view log.

Open the Oracle Materialized View Log Editor by clicking on the Materialized View log tab in the Oracle Table Editor, selecting the Materialized View log check box, and clicking on the Edit Properties button.

Materialized view logs are supported in the following features:

  • Forward Engineering
  • Reverse Engineering

The Oracle Materialized View Log Editor dialog includes the following tabs:

General Select general options for the materialized view log such as refresh types and filter columns.

Other Options Use for selecting cache type and parallel type.

Storage Select the physical storage object.

Partitions Select the partitioning options for the materialized view log.

Comment Add or edit comments for a materialized view log.

UDP Enter user-defined property values for a materialized view log.
