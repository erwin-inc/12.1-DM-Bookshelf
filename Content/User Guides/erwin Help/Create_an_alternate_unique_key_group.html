Create an Alternate (Unique) Key Group

You can work with options in the Key Group Editor in a logical model to create an alternate key group and associate attributes with it.

To create an alternate (unique) key group

  1. Right-click the entity for which you want to create an alternate key group and click Key Group Properties.

    The Key Group Editor opens.

  2. Click the New button on the toolbar and select New Alternate Key (Unique).

    A new alternate key group is added to the Navigation Grid for the selected entity. Use in-place editing to change the default name in the Name field, and select the Logical Only check box if you want the alternate key group to only appear in the logical model.

  3. Click the Memberstab and select one or more attributes from the Key Group Members list to specify as members of the alternate key group. Use the toolbar to sort the list of attributes, move attributes up or down the list, open the Attribute Editor to work with attribute properties, or reset attribute order.
  4. Click Close.

    The alternate key group is created and the Key Group Editor closes.
