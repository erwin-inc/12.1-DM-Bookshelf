Oracle Rollback Segment Editor

Use the Oracle Rollback Segment Editor to add and specify the parameters for a rollback segment.

Open the Oracle Rollback Segment Editor by selecting Rollback Segments from the Database menu.

Rollback Segments are supported in the following features:

  • Forward Engineering
  • Reverse Engineering

The Oracle Rollback Segment Editor dialog includes the following tabs:

General Add and specify the parameters for a rollback segment.

Dependencies Displays any dependencies for a rollback segment.

Comment

Lets you enter any comments that you want to associate with the object.

UDP

Lets you enter user-defined property values for the object.

Notes

Lets you view or edit user notes.

The following describes all of the other options available in this dialog:

Rollback Segment

A list box that displays the rollback segments in the current model.

New

Opens the Create Rollback Segment dialog where you can specify the name for the new rollback segment.

Delete

Select a rollback segment from the Rollback Segment list box that you want to delete, and click the Delete button.

Generate

Generates DDL for this object during Forward Engineering. If you do not select the check box, no DDL will be generated for this object during Forward Engineering.

DB Sync

Starts the Complete Compare task so you can synchronize the physical objects defined in your model with the information stored on the server.
