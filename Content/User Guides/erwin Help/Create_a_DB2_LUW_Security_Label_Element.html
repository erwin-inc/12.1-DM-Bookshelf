Create a Db2 LUW Security Label Element

A security label element has one or more criteria (elements) grouped in a relationship. You can open the Db2 for LUW Security Label Element Editor to work with additional properties.

To define a Db2 security label element

  1. In the Db2 for LUW Security Label Editor, click the Security Label Element Editor icon on the toolbar.

    The Security Label Element Editor opens.

  2. Work with the following options:

    Click New on the toolbar to create a Security Label Element. Use the Enter filter text text box to filter a large list of names to locate the one you want to work with.

    Name

    Specifies the name of the Security Label Element.

    Security Label Component

    Specifies the Security Label Component to which the Security Label Element belongs. Select from the drop-down list.

  3. Click the General tab.

    The Security Elements associated with the selected Security Label Component appear.

  4. Select one or more Security Elements that you want associate with the Security Label Element.
  5. (Optional) Click the Notestab to view and edit user notes.
  6. (Optional) Click the Extended Notes tab to view or edit user notes.
  7. Click Close.

    The selected Security Elements are associated with the Security Label Element and the Db2 for LUW Security Label Element Editor closes.
