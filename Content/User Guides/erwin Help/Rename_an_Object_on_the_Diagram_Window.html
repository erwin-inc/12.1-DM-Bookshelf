Rename an Object on the Diagram Window

You can rename an object on the Diagram Window.

To rename an object on the Diagram window

  1. Click the Select icon in the erwin toolbox.
  2. Click to select the object that you want to change, wait briefly, then click again. An edit box is drawn around the object's name.
  3. Click the edit box. Edit or erase some or all of the existing name and enter the new name.
  4. Press Shift+Enter or click outside the edit box to save your changes or press the Tab key to go to the next related object (for example, to edit the next attribute in the entity).

    The changes you enter are automatically applied throughout the model. For example, if you change the name of a primary key or an alternate key, the name of the corresponding migrated foreign key also changes.

To rename an object, you can also select the object, press the F2 key and enter the new name.
