Oracle Partitions Tab - Global-Range

The following describes the options available when you select the Global-Range partitioning type in the Partitions tab:

Available Columns

Displays the columns in the current model. To add a column to the Selected Columns list, click on the column that you want from the Available Columns list, and click the button to add.

Selected Columns

Displays the columns you selected and moved from the Available Columns list. To remove a column from this list and place it back into the Available Columns list, click on the column that you want from the Selected Columns list, and click the button to remove. Use the button and the button to move a column up or down in the list.

Delete
Partitions

Displays the partitions. The following describes the information in this grid:

Partition Name

Enter a partition name in this column.

Edit Properties

When you click on a button in this column, it opens the Oracle Partition Description dialog where you can modify the properties for the selected partition. Additional columns display in this grid when there are columns displayed in the Selected Columns list. The column names of the Selected Columns list are the additional column headings within this grid. Click within a cell under a column name and a drop-down appears where you can assign a value. You must have columns in the Selected Columns list in order to click on a button in the Edit Properties column to be able to open the Oracle Partition Description dialog.

The following describes general information regarding the Global-Range type:

  • Global partitioning is not available for bitmap indexes.
  • List and Composite partition options are not valid for a Global partition.
