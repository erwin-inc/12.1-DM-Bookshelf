Change Attribute Key Group Membership in the Key Group Editor

You can work with options in the Key Group Editor in a logical model to change attribute key group membership. You work with the navigation icons to add, remove, or change the order of attributes.

To change attribute key group membership in the Key Group Editor

  1. Right-click the entity that contains the key group that you want to change and click Key Group Properties.

    The Key Group Editor opens.

  2. Select the key group in the Navigation Grid that you want to change and click the Members tab.
  3. Work with the following options:
    Key Group Members

    Displays the attributes that are currently members of the selected key group. Select the check box of each attribute that you want to include in the key group, or clear the check boxes of attributes you want to remove from the key group. The Key Group Members option contains a toolbar that includes the following options:

    Sort

    Sorts the key group members by alphabetic, reverse alphabetic, attribute order, or key group member order. You select the method you want using the drop-down menu that opens after you click the Sort button. Visual cues are provided for how the key group member list is currently sorted:

    Specifies that the key group member list is sorted in alphabetic order.

    Specifies that the key group member list is sorted in reverse alphabetic order.

    Specifies that the key group member list is sorted in attribute order.

    Specifies that the key group member list is sorted in key group member order.

    Moves the selected key group member to the top of the list.

    Moves the selected key group member up one position in the list.

    Moves the selected key group member down one position in the list.

    Moves the selected key group member to the bottom of the list.

    Opens the Attribute Editor so you can create a new attribute or modify the properties of an existing one.

    Resets attribute order.

  4. Click Close.

    Attribute key group membership is changed and the Key Group Editor closes.
