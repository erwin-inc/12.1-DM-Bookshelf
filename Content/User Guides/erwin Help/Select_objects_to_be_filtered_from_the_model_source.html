Select Objects to Be Filtered from the Model Source

You use the options in the Type Selection pane to apply filters to the types of objects that you want to add from the source model.

To select objects to be filtered from the source model

  1. Click Design Layers, Add Model Source on the Actions menu.

    The Add Model Source Wizard opens on the Source Model pane.

  2. Click the Type Selection link in the Explorer pane.

    The Type Selection pane opens.

  3. Work with the following options:
    Compare Level

    Specifies the set of objects that will be added to the target model. Only the filter options relevant to the source model type are displayed. For example, if you designate a physical model as the model source, the logical filter is not available.

    Option Set

    Lists the objects and properties you want to include when you add the model source. You can open, modify, or delete an existing option set, or create a new option set. You can also select one of the preset option sets. The Advanced Default Option Set includes all objects and properties in the model source. The Standard Default Option Set includes fewer selected objects and properties.

  4. Click Next.

    The Source Model Object Selection pane opens.

You can click the Add button at any time to add the model source.
