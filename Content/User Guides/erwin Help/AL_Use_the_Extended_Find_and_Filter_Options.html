Use the Extended Find and Filter Options

You can find and filter the information in the Summary tab in the Extended Find and Filter dialog. Changes made to a model at the object and property levels appear in the tree.

You cannot Reverse or Undo a transaction.

To use the Action Log Extended Find and Filter option

  1. Click Extended Find and Filter on the Action Log toolbar. This option is available on the Summary pane.

    The Extended Find and Filter dialog opens.

  2. Work with the following options:
    Find

    Search all the properties, the three built-in categories; Graphical objects, Graphical properties, and Read only properties, and one occurrence for each class node entry that appear in the tree of the Extended Find and Filter dialog. You can expand each class to see the objects and properties for that class. There are three columns (Create, Modify, Delete) corresponding to the summary actions that took place. These columns contain check boxes for the class, objects, and properties that were involved in the change to the model.

    Show/Hide filter

    Toggle to display or hide the filtering options.

    Filter Options

    Displays the Filter Criteria dialog, which contains two check box lists for Properties and Categories and a Class list containing items from the action summary. Set Filter searches the text and removes it from the tree.

    Properties Details

    Displays the details of the property change. The old and new values are displayed.

  3. Click Close.

    The Extended Find and Filter dialog closes and you are returned to the workspace.
