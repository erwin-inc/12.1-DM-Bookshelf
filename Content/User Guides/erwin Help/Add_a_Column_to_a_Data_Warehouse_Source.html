Add a Column to a Data Warehouse Source

You use the Data Movement Table tab in the Data Warehouse Source Object editor to add a column to a data warehouse source table.

To add a column to a data warehouse source

  1. Select Data Sources in the Model menu.

    The Data Warehouse Source Object editor opens.

  2. Select the Data Movement Table tab and click the Data Movement Table Editor .

    The Data Movement Table Editor opens.

  3. Work with the following options:
    Data Source Object

    Specifies the data source object that the table belongs to, for which you want to add the column.

    Data Source Table

    Specifies the data source table for which you want to add the column.

  4. Work with the following options on the Data Movement Column:
    Name

    Specifies the name of the data source column that you want to add.

    Physical Datatype

    Specifies the data type of the column that you want to add.

    Definition

    Specifies any comments about the column.

    The column is added to the data warehouse source.
