Specify the Order of Rows and Columns to Display in the Bulk Editor

The Display Order page is used to change the order in which objects and properties are displayed in the Bulk Editor grid. This page overrides the ordering performed in the Bulk Editor grid, however once the Bulk Editor Wizard is closed, all ordering is performed in the Bulk Editor grid.

To specify display order in the Bulk Editor

  1. Click the button on the Bulk Editor toolbar.

    The Bulk Editor Wizard opens.

  2. Click Display Order and work with the following options:
    Row Order

    Specifies the row order of the objects selected in the Object Types page. Select each object that you want to move and use the up and down arrows on the toolbar to move the object up or down in the list.

    Column Order

    Specifies the column order of the properties selected in the Property Types page. Select each property that you want to move and use the up and down arrows on the toolbar to move the property up or down in the list

  3. Click Edit.

    Display order is specified for the Bulk Editor and the Bulk Editor Wizard closes.
