Db2 for LUW Index Extension Exploitation Rule Editor

Use the Db2 for LUW Index Extension Exploitation Rule Editor to describe the search targets and search arguments and how they can be used to perform the index search through a search method. Specify index exploitation rule on a DETERMINISTIC with NO EXTERNAL ACTION function.

Follow these steps:

  1. To access this editor, open the Db2 LUW Function Editor. Go to the Predicate Exploitation Rules tab and click IndexExploitationRule Editor.
  2. Click New and enter a unique name for the Index Exploitation rule.
  3. Work with the following options on the General tab:
    Search Key

    Specifies the search target. You can specify only one key per target.

    Search Argument

    Specifies the parameter that you want to use with the search method.

    Method Name

    Specifies the search method that you want to use from the methods that are specified in the index extension.

  4. (Optional) Click the Notes tab to view and edit user notes.
  5. (Optional) Click the Extended Notes tab to view or edit user notes.
