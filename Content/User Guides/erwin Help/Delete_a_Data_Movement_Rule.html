Delete a Data Movement Rule

You use the Data Movement Rule Editor to document the data movement maintenance rules associated with the tables in your model. The Data Movement feature is available in a physical model when you select the Data Movement check box on the General tab in the Model Editor.

To delete a data movement rule

  1. Select Data Movement Rules from the Model menu.

    The Data Movement Rule Editor dialog opens.

  2. Select the rule you want to delete and click the Delete icon on the toolbar.

    The data movement rule is deleted.
