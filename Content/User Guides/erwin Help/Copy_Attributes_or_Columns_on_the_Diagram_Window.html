Copy Attributes or Columns on the Diagram Window

On the diagram window, you can copy and paste attributes or columns from one entity, table, or view to another using the CTRL+drag and drop key sequence.

To copy attributes or columns on the diagram window

  1. Select one or more attributes or columns that you want to copy on the diagram window, then use CTRL+drag and drop to copy the attributes or columns to a different entity, table, or view.

    The copied attributes or columns are copied on the new location.

  2. Change the names of all copied attributes or columns to enforce unique naming in the model. If you are working in a Logical/Physical model, change the name on the logical side, to automatically change the name on the physical side. If you rename the object on the physical side, the change is not automatically reflected on the logical side.
