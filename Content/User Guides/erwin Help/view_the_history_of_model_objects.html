View the History of Model Objects

When you configure to trace the history of model objects, you can view the recorded history for those objects.

You can view the history for a specific object in the History tab of the object editor. Click the History tab in any of the following editors to view the recorded events for that object:

  • Entities
  • Attributes
  • <Database> Table Editor (for example, Oracle Table Editor)
  • Columns
  • Views
  • Cached Views
