Oracle Editors - History Tab

Use the History tab to view the history of an object. The History tab is identical for the Oracle property editors that use it. The following describes the options available for this tab:

Event

Displays the history of the object, including the date and time information.

Description

Displays a short description of the event that is highlighted in the Event window and is read-only text.

Comment

You can enter or change an existing comment for the object history. Your changes are committed when you click OK, or select another object from the list box that contains the objects for that editor.
