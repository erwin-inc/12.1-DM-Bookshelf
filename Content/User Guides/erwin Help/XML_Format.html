XML Format

You can use XML format to export model data to applications that support XML. XML format replaces the text-based ERX format used in earlier versions of this product. XML format provides structured markup of model metadata according to elements that are defined in the XML Schema.

The following support for XML format is provided:

Save as XML

Indicates XML as the file format.

Open as XML

Selects a model file previously saved in XML format.

All of the metadata in your model is defined as Elements in the XML tagged format. You can learn more about XML format support in this product by saving a small model in XML format and reviewing the output in a web browser that displays XML, such as Google Chrome.

You can edit the content, but not the tags in an XML file. For example, you can save your model in XML format, open the XML file in a text editor and then search and replace text elements, such as entity and attribute names. However, you should be careful not to modify any of the XML tags or you will not be able to use this product to open the file again.
