Specify Image Properties

You can associate an icon or image file with certain model objects. Use the Image Properties editor to select the image.

To specify image properties

  1. Click the Icon tab in the Entity or Table editor and click an image in the Small Icon or Large Icon section.

    The <database> Diagram Image Editor opens.

  2. Select an image and work with the following options:

    Click New on the toolbar to create a new image object.

    Name

    Specifies the name of the image.

    Image Type

    (Read-only) Specifies the image type.

    Image Storage Type

    Specifies how the image is stored: Built in, Internal, External.

  3. Work with the following fields on the General tab:
    Image Type

    (Read-only) Specifies the image type.

    Image Storage Type

    Specifies how the image is stored: Built in, Internal, External.

    Image Path

    Specifies the directory path for the image file.

    Image Size

    (Read-only) Indicates the image size.

    Preview

    Displays a preview of the image.

  4. (Optional) Click the Where used tab to review where the image has been used.
  5. (Optional) Click the Definition tab and enter a definition you want to associate with the rule.
  6. (Optional) Click the UDP tab to work with user-defined properties for the audit policy.
  7. Click Close.

    The Image Properties Editor closes.
