XML File Validation

When you attempt to open an XML file, an XML file validation dialog opens asking you if you want to validate the XML file against the XML schema prior to opening it.

An XML schema is a document or set of documents that defines the structure and legal elements of the XML file. The XML validation process ensures that the XML file that you want to open contains correct syntax and conforms to the defined XML schema. The XML file validation process is useful because you can uncover errors prior to opening the XML file. This is particularly useful if any of the following conditions apply:

  • The XML file was generated by a third party.
  • An error occurred while opening the XML file. If the error is due to a syntax or format error in the XML file, the file validation process will identify the location of this error in the file.
  • The XML file is large, and you want to minimize the chance that the file may not open.

The XML file validation process only checks the syntax and the elements of the XML file. It does not check the semantics of the model represented in the XML file; that is done while the XML file is being opened.

When you select Open from the File menu, and you select XML Files from the Files of type box, the File Open dialog opens where you must select how you want to handle checking the structure and content of the XML file according to the schema. You have the following selections available:

  • You can select Yes so that the XML file is validated against the XML schema before the XML file is opened. If the validation fails, then the XML file is not opened; if the validation succeeds, the XML file is opened.
  • You can select No so that the XML file is not validated against the XML schema, and an attempt is immediately made to open the XML file.
  • Select Cancel to cancel the XML file open operation.

If you want to set a default selection for the file validation option, select Options from the Tools menu and then select the XML tab.

The time it takes to complete an XML file validation is in addition to the time it takes for erwin Data Modeler to open the XML file.
