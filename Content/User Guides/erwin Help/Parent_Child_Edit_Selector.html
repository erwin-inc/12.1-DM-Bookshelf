Parent Child Edit Selector

When you press the Shift key and double-click a relationship line, the Parent/Child Edit Selector dialog opens. Use this dialog to select the parent or child end of the relationship for editing. After you have made your selections and click OK, the object editor associated with the parent or child opens (such as the Attributes Editor or the Columns Editor).

The Parent/Child Edit Selector includes the following options:

Relationship

Displays the name of the relationship you selected.

Edit Parent <name>

Displays the name of the object on the parent side of the relationship.

Edit Child <name>

Displays the name of the object on the child side of the relationship.

OK

Opens the editor for the selected parent or child object.
