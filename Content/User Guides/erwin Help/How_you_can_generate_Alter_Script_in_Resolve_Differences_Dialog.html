How you can generate an Alter Script in the Resolve Differences Dialog

In the Resolve Differences dialog, you can update the database or script file that is associated with the model by generating an ALTER script. When a change allows the generation of ALTER scripts, the Left or Right Alter Script icons become active in the toolbar.

The process includes the following steps:

  • Begin your Complete Compare session and navigate to the Resolve Differences dialog.
  • Work with the options in the Resolve Differences dialog to reconcile differences between the models.
  • When active, click the Left Alter Script/Schema Generation icon or Right Alter Script/Schema Generation icon.
  • Work with options in the Alter Script wizard to generate the script.
