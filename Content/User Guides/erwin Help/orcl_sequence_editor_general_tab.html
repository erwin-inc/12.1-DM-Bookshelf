Oracle Sequence Editor - General Tab

Use the General tab in the Oracle Sequence Editor to add and specify the parameters for a sequence. The following describes the options available for the General tab of the Oracle Sequence Editor dialog:

Starting Value

Enter a value in this text box.

Increment By

Enter a value in this text box.

Use Cycle

Select one of the options:

No Cycle
Cycle
Use Order

Select one of the options:

No Order
Order
Set Cache Size

Select one of the options:

No Cache
Value

If you select this option you must provide a value in the text box.

Use Maximum Value

Select one of the options:

No Max Value
Max Value

If you select this option you must provide a value in the text box.

Use Minimum Value

Select one of the options:

No Min Value
Min Value

If you select this option you must provide a value in the text box.
