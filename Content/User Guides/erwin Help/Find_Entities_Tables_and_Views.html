Find Entities, Tables, and Views

You can filter objects based on their types and find a specific entity, table, or view in your model using the Go To feature. This feature is helpful if you have a large model or diagram and the entity, table, or view that you want to find is not visible in the current diagram window (or view level).

To find an entity, table, or view

  1. On the Edit menu,click Go To.

    The Go To editor opens.

  2. In the Filter Type drop-down, select the object type that you want to find. You can also enter the name of the object in the filter text box to narrow down the list of objects further
  3. Select the object that you want to find.

    Click Go To.

    The dialog closes and the selected object is found.
