Change the Order of Script Execution

You can change the order of script execution using the Object Creation Order tab in the Script Template Editor.

To change the order of script execution

  1. Click Script Templates on the Model menu.

    The Script Template Editor opens.

  2. Click the Object Creation Order tab and change script order. Use the toolbar to specify sort order or move a selected script up or down in execution order.
  3. Click Close.

    Script execution order is changed and the Script Template Editor closes.
