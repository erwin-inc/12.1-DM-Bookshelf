Set Spelling Options

You can customize options in the Spell Check editor to narrow or broaden the spell check that is performed. You can also choose a dictionary language, and set the relative speed at which the spellings are checked.

To set spelling options

  1. Click the spell check icon in the Comment tab of a property editor.

    The Check Spelling dialog opens.

  2. Click the Options button.

    The Spelling Options dialog opens.

  3. Select or clear the check box for the following options:
    Ignore capitalized words

    Specifies that capitalized words are not included in the spell check. Example: Candad

    Ignore all-caps words

    Specifies that words that consist of all capital letters are not included in the spell check. Example: ASAP.

    Ignore words with numbers

    Specifies that words with numbers are not included in the spell check. Example: Win95

    Ignore words with mixed case

    Specifies that words with mixed case are not included in the spell check. Example: SuperBase.

    Ignore domain names

    Specifies that domain names (domainname.com) are not included in the spell check. Example: xyz.com.

    Report doubled words

    Specifies that instances of two consecutive identical words be checked. Example: the the.

    Case sensitive

    Specifies that the spell check be case sensitive. (selected by default)

    Phonetic suggestions

    Specifies that phonetic suggestions be displayed with standard suggestions from the dictionary.

    Enabling this option increases the time required to locate suggestions.

    Suggest split words

    Suggests two separate words as a replacement for a misspelling containing two joined words. For example, is the is suggested as a replacement for isthe.

    Auto correct

    Changes the misspelled word automatically. If this option is not selected, the Spell Checker prompts you before changing.

  4. Select the default dictionary language from the Main Dictionary Language drop-down list.

    The value you select here sets the language of the main dictionary that is used to check spelling. The list shows only languages for which dictionaries are installed on your computer. To check spelling in a different language, select the language in the list

  5. Click one of the following options in the Suggestions section:
    Fast but less accurate

    Presents the most likely suggestions as quickly as possible.

    Moderately fast and accurate

    Presents a more extensive list of suggestions as quickly as possible (default).

    Slow but accurate

    Queries all dictionaries to present a complete list of suggestions.

  6. Click OK to save your selections, close the Spelling Options dialog, and return to the Check Spelling dialog.
