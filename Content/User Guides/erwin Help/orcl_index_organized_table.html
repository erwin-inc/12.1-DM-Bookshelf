Oracle Index Organized Table

Use the Oracle Index Organized Table Dialog to assign various properties for an index organized table.

Open the Oracle Index Organized Table Dialog when you click on the IOT Properties button in the General tab of the Oracle Table Editor and Oracle Materialized Views Editor. The following describes the options for the Oracle Index Organized Table Dialog:

Mapping

A drop-down list box where you can select a mapping table.

PCT Threshold

Enter a value in the text box.

Compress Type

If you select this option, select one of the following options:

Not Compressed
Compressed

If you select this option, enter a value in the text box.

Overflow
Including Column

A drop-down list box that displays the available columns in the current model.

Edit Overflow

When you click this button, it opens the Oracle Overflow Segment Attributes Editor where you can select a tablespace and modify the overflow properties for that tablespace. After you select a tablespace and click OK to close the Oracle Overflow Segment Attributes Editor and return to the Oracle Index Organized Table Dialog, this tablespace populates the text box that is above the Edit Overflow button.
