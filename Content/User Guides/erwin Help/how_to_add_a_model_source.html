How to Add a Model Source

You use the Add Model Source wizard to link two independent models and specify that one model is a source for the other, or specify that a model has multiple sources. The model you open before you use the wizard is the target model into which a model source that you select contributes a set of objects.

You use the Add Model Source wizard to customize options.

The Add Model Source wizard includes sections that let you do the following:

  • In the Source Model section, select a model source for the current model.
  • In the Type Selection section, apply filters to control the type of objects that are added. You can filter by logical, physical, or database level objects, and use a customized option set of filter options.
  • In the Object Selection section, select the model objects to copy from the source to the target model.
  • In the Naming Standards section, define model object naming rules, and designate a naming standards file to use when enforcing standards in the target model. Applies only when you are adding a logical model as the source.

The Target Model section is a read-only section to indicate the current model to which you will add the model source.
