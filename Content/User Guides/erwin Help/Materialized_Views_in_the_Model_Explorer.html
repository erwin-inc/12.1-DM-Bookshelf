Materialized Views in the Model Explorer

The Model pane in the Model Explorer lists each materialized view in a model. Each materialized view is listed in alphabetical order. When you right-click the Materialized Views folder or a specific materialized view in the Model Explorer, a shortcut menu appears.

When you right click the Materialized Views folder, the following options appear on a shortcut menu:

New

Creates a new materialized view in the Model Explorer and in your model.

When you click a materialized view in the Model Explorer, the following options appear on a shortcut menu:

Go To Diagram

Switches the model focus to the selected materialized view.

Delete

Deletes the selected materialized view.

Properties

Opens the Materialized View Editor.
