Specify the Table Schema

Specifying a table schema helps identify tables.

To specify the table schema

  1. Click Tables on the Model menu.

    The Table Editor opens.

  2. Select the table in the Navigation Grid for which you want to specify the table schema and select the schema from the Schema drop-down menu.

    Use the Enter filter text box to filter a very large list of tables to quickly locate the one that you need.

  3. Click Close.

    The table schema is defined and the Table Editor closes.

You can set a default owner for new tables and globally reset the designated table schema using features in the Defaults tab of the Model Editor.
