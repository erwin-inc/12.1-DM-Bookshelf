Preview the Script (Forward Engineering)

The script is displayed based on the options that are selected. For example, if you select the CREATE TABLE option in Oracle, a CREATE TABLE statement is displayed for each selected table.

Follow these steps:

  1. Click Create Subject Area in the physical model and select the subject area for which you want to generate the schema.
  2. Click Action, Forward Engineer, Schema.
  3. Select an option set from the Option Set list.
  4. Select the schema generation options.
  5. Click Previewto open the Schema Generation Preview dialog.

    The <database> Schema Generation Preview dialog appears. Review the script.
