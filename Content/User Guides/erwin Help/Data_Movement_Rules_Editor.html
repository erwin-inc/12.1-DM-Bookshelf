Data Movement Rules

The Data Movement Rules editor is used to document the data movement maintenance rules associated with the tables in your model. This editor supports the following types of standard management rules:

  • Refresh
  • Append
  • Backup
  • Recovery
  • Archiving
  • Purge

You use the Data Movement Rules editor to do the following:

  • Enter or edit the name of a data movement rule.
  • Enter a definition for the rule selected in the grid.
  • Use the Type Filter to select the type of rule that you want to display in the grid. To display all rule types, select All from the list.
  • Attach the rule selected in the grid to tables in your model.

The Data Movement Rules editor is available when you select the Data Movement check box on the General tab in the Model Editor. You open the Data Movement Rules editor when you select Data Movement Rules from the Model menu in the physical model.
