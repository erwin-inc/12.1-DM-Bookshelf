Defining SQL Server External Data Sources

The following properties are applicable to a SQL Server External Data Source object.

Tab

Property

Description

Additional Information

General

Location

Specifies the connectivity protocol and path to the external data source in the '<prefix>://<path[:port]>' format

  

Connection Options

 Specifies additional options while using ODBC to connect to an external data source  
Credential Specifies a database-scoped authentication information required connect to the external data source  
Pushdown Specifies whether computation can be pushed down to the external data source  
External Data Source Type

Specifies the type of external data source being configured

Hadoop: Use when the external data source is Cloudera, Hortonworks, or an Azure Storage account.

Blob_Storage: Use while executing bulk operations from Azure Storage account.
Resource Manager Location If specified, indicates that the Query Optimizer will improve performance Use this property only while connecting to Hortonworks or Cloudera.
