Oracle Tables, Materialized Views, and Materialized View Logs - Partitions Editor

Use the partition editor to specify physical partition values for a table, materialized view, or materialized view log as follows:

Table

In the Model Explorer, under Tables, right-click Partitions and click New.

Or, in the Diagram Area, right-click a table and then click Partition Properties.

Materialized View

In the Model Explorer, under Materialized Views, right-click Partitions and click New.

Or, in the Diagram Area, right-click a materialized view and then click Partition Properties.

Materialized View Log
  • Open table editor and click the Materialized View Log tab.
  • Select the Materialized View Log check box and then click Edit Properties. The materialized view log editor opens.
  • Click Partitions to open the partition editor.
