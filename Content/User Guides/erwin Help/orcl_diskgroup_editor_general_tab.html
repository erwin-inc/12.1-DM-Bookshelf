Oracle Disk Group Editor - General Tab

Use the General tab to select a redundancy type for the disk group. The following describes the options in the General tab of the Oracle Disk Group Editor:

Redundancy Type

Select this check box and select one of the following options:

  • HIGH REDUNDANCY
  • NORMAL REDUNDANCY
  • EXTERNAL REDUNDANCY

The grid below Redundancy Type displays Fail Groups. If a selected disk group is part of a Fail Group, it appears in this grid along with columns showing # of Disks and Disk Properties. When you click the button within the Disk Properties column, the Oracle Disk Properties Dialog opens so that you can modify disk properties.
