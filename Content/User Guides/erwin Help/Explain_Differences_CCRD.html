Explain Differences in the Resolve Differences Dialog

To easily understand the compare results displayed in the Resolve Differences dialog, you can review the information in the Explanation dialog.

Select a row, right-click, and then click Details, the Explanation dialog opens and displays several options. The following options are not self-explanatory:

Type

Displays the label for the object or property row that is currently selected. For example, Entity/Table in the Object View column.

Description

For object rows, the text explains why the two objects were aligned. For property rows, the text explains why the two property values are equal.

Show Selected Items - Do not track the item

You can display the row explanation in two different modes. By default, as you move the cursor over a row, the explanatory text changes. Alternatively, you can select the "Show Selected Item - Do not track the item" check box to display a selected row, and not track the cursor movement.
