Set Display Options for Relationships

You can use the Relationship tab on the Diagram Editor to set display options for relationships in either a logical or a physical model.

To set display options for relationships

  1. Click the Relationship tab on the Diagram Editor and work with the following options:
    Relationship Logical Display Options

    Lets you specify display options for relationships in the logical model. Select any or all of the following display options:

    Display Logical Relationship Name

    Specifies to display relationship name in the logical model. Clear to hide the relationship name.

    Display Logical Cardinality

    Specifies to display cardinality symbols (P, Z, or n) for the relationships in the logical model. Clear to hide relationship cardinality.

    Display Logical Referential Integrity

    Specifies to display referential integrity symbols (for example, U:R, I:R, D:C) for the relationships in the logical model. Clear to hide referential integrity.

    Display Subtype Discriminator

    Specifies to display the subtype discriminator assigned to the subtype relationship.

    Relationship Physical Display Options

    Lets you specify display options for relationships in the physical model. Select any or all of the following display options:

    Display Physical Relationship Name

    Specifies to display the physical relationship name in the model.

    Display Physical Cardinality

    Specifies to display cardinality symbols (P, Z, or n) for the relationships in the physical model. Clear to hide relationship cardinality.

    Display Physical Referential Integrity

    Specifies to display referential integrity symbols (for example, U:R, I:R, D:C) for the relationships in the physical model. Clear to hide referential integrity.

    Display Ungenerated Relationships

    Specifies to display relationships that will not be generated when you generate the schema. Clear to hide relationships that will not be generated.

  2. Click Close to save your changes and close the editor.
