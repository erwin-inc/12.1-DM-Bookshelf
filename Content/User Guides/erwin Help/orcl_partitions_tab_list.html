Oracle Partitions Tab - List

The following describes the options available when you select the List partitioning option in the Partitions tab:

Column

Displays the columns in the current model from which to select the column that you want.

Partitions

Displays the partitions in the table. The following describes the information in this grid:

Partition Name

Enter a value for a new Partition Name.

Values

To add a value to the partition, click on a cell in the Values column to open the Oracle Partition Values Dialog.

Edit Properties

To edit tablespace properties for a partition, click the button within the Edit Properties column to open the Oracle Partition Description dialog.
