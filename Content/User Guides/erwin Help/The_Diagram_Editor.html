The Diagram Editor

The Diagram Editor includes all the options that you set for a diagram. You must click the Members tab and indicate which of the available objects to include in the diagram. Other options have a default setting, which you can change if required.

The Diagram Editor has the following sections and tabs:

  • Navigation Grid
    • Select the model or subject area for which you want to edit ER diagrams.
    • Move to the previous or the next diagram for the selected model or subject area.
    • Sort the diagrams in alphabetical or reverse alphabetical order.
    • Create a new diagram name.
    • Delete a diagram.
    • Harden, override, or restore inheritance for drawing styles of a diagram.
    • Filter through a list of diagrams to quickly select the one that you want to edit.
    • Set default display to Logical, Physical, or Dimensional (if you have selected Data Warehousing for Physical notation in the Model Editor).
    • Auto-populate a diagram.
  • General tab
    • Create or edit an author name.
    • Select an optional Theme for the diagram.
    • Indicate an override to the Theme background fill.
  • Members tab
    • Select the objects you want to include in the diagram.
    • Automatically populate the list of included objects from a subject area.
  • Relationships tab
    • Select the relationships you want to include in the diagram.
  • Layout tab
    • Set options for manual layout, grid alignment, and visual cues.
  • Display tab
    • Set options for shadow, display length, and relationship display options.
  • Entity
    • Set the logical display level and entity display options.
  • Table/Node/Collection/Record/Document/JSON Object
    • Set the physical display level and table, node, collection, record, document, or JSON object display options. The tab name differs based on the model's target database.
  • View
    • Set view display options.
  • Relationship
    • Set logical and physical relationship display options.
  • Page Setup tab
    • Specify page setup options for the selected diagram. These options are reflected on the General tab of the Page Setup Editor when you print the diagram.
  • Page Setup Margins tab
    • Specify page margins for the selected diagram. These options are reflected on the Margins tab of the Page Setup Editor when you print the diagram.
  • Comment
    • Add a comment for the diagram.
  • UDP
    • Create a UDP for the diagram.
