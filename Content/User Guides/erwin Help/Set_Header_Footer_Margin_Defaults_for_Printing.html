Set Header, Footer and Margin Defaults for Printing

You can use the Page Setup Margins tab on the Diagram Editor to set defaults that display on the Margins tab of the Page Setup Editor in the Print dialog.

To set header, footer, and margin defaults for printing

  1. Open the Diagram Editor and click the Page Setup Margins tab. Work with the following options:
    Header/Footer

    Specifies elements that appear in the header and footer of your printout. Click in the text box, then click an item on the Header/Footer toolbar to add an element.

    Available preset items on the Header/Footer toolbar include:

    • Current Page
    • Page Count
    • File Name
    • Subject Area Name
    • Diagram Name
    • Time
    • Date

    When you make a selection, a preview of the header and footer text appears in the area below the entry box.

    The values for Current Page and Page Count appear when you view Print Preview.

    Margins

    Specifies the margin (in inches) for the top, left, right, and bottom margins, and for the header and footer margins.

  2. (Optional) Click the Page Setup tab to set defaults for page size, orientation, zoom level, borders, and colors.
  3. Click Close to save your changes and close the editor.
