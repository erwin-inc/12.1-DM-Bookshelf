Add an Attribute Note

In a logical model, or the logical side of a logical/physical model, you can use the Notes tab in the Attribute Editor to add additional notes for each attribute in your model. The attribute note does not migrate to the physical model as a comment.

To add an attribute note

  1. Right-click the entity that contains the attribute to which you want to add a note and click Attribute Properties.

    The Attribute Editor opens.

  2. Select the attribute name in the Navigation Grid for which you want to add a note and click the Notes tab.
  3. Click the New button.

    The Attribute Notes editor opens.

  4. Enter note text. You can use the toolbar to cut, copy, or paste text, open the Notes editor, or check spelling.
  5. Click Close.

    The Attribute Notes editor closes.

  6. Click Close.

    The Attribute Editor closes.
