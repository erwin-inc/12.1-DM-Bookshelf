Data Warehouse Source Selector Dialog Options

The options in the Data Warehouse Source Selector dialog are used to select available data warehouse sources. The Data Warehouse Source Selector dialog is only available when you select the Data Movement check box on the General tab in the Model Editor. You use the Data Warehouse Source Selector dialog to view the available data warehouse sources hierarchically by source, table, and column. You can add a source column to the Selected Data Sources list, or you can remove a data source. You also can edit data warehouse source definitions when you open the Data Warehouse Sources dialog from this dialog.
