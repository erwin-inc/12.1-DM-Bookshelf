Use the Extended Find and Find Next Options

You can search text in the Action Log based on Properties, Categories, and Class. Properties contain all the properties that appear in the Extended Find and Filter dialog. There are three Categories: Graphical objects, Graphical properties, and Read-only properties. Class displays one occurrence for each class node in the summary tree. The Extended Find feature searches the list for an occurrence of text, whereas the Extended Filter feature changes the information displayed in the list.

To use the Action Log Find option

  1. Click Extended Find and Filter on the Action Log toolbar. This option is available on the Summary pane.

    The Extended Find and Filter dialog opens.

  2. Click Find on the Extended Find and Filter dialog.

    The Find Criteria dialog opens.

  3. Select from the following options:
    Properties

    Select the properties to search for. Anytime a property is selected, the corresponding occurrences of the property in the tree list are also selected.

    Categories

    Select from three built-in categories; Graphical objects, Graphical properties, and Read only properties.

    Class

    Select each class node entry that displays in the tree. You can expand each class to see the objects and properties for that class. There are three columns (Create, Modify, Delete) corresponding to the summary actions that took place. These columns contain check boxes for the class, objects, and properties that were involved in the change to the model. The check boxes appearing on this screen can be in one of three states:

    • Selected items-displays a check mark in the check box.
    • Unselected items-displays an empty check box.
    • Indeterminate state-displays a faded gray check mark in the check box, indicating items are in an indeterminate state. For example, if some graphical properties are selected and the rest are cleared then the state of the category Graphical properties is indeterminate.
  4. Click Find.

    The text is selected.

  5. Click Close.

    The Extended Find and Filter dialog closes and you are returned to the workspace.
