Define a Volumetrics Report

You can use the Report tab in the Volumetrics dialog to report on the database by physical objects, database objects, or tables according to your table settings. You can report on the initial size or projected future database size based on your estimated monthly row (record) growth. You can save your report externally as a CSV file.

To define a volumetrics report

  1. Click Volumetrics on the Tools menu.

    The Volumetrics dialog opens.

  2. Click the Report tab and select from the following options:
    Physical Objects

    Select to display the volumetric report grouped by physical storage object.

    Database Objects

    Select to display the volumetric report grouped by database object.

    Tables

    Select to display the volumetric report grouped by table.

    Initial

    Select to display the volumetric report using initial table row size.

    Projections

    Select to display a volumetric report using projected table values. The projection uses the Grow By estimates in the Settings tab.

    Select or type the projected number of months in the Months field (available only when you click Projections).

    Save as CSV

    Select to save the calculated data as an external CSV file. You are prompted to save the file, which you open using a spreadsheet or database application that supports CSV files.

  3. Click OK.

    The report executes. You can view the object size calculations in the Physical Objects list.
