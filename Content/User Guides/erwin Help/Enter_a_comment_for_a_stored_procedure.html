Enter a Comment for a Stored Procedure

You can use the Comment tab in the Stored Procedure Editor to add, change, or copy comments about the selected stored procedure. If your target server supports comments, you can generate comments in the physical schema DDL script.

To enter a comment for a stored procedure

  1. Click Stored Procedures on the Model menu.

    The Stored Procedure Editor opens.

  2. Select the stored procedure in the Navigation Grid for which you want to enter a comment and click the Comment tab.
  3. Enter stored procedure comments. Use the toolbar to cut, copy, and paste text, work with the Text Editor, or check spelling.
  4. Click Close.

    Stored procedure comments are saved and the Stored Procedure Editor closes.
