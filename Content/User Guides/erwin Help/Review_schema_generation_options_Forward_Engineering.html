Review Schema Generation Options (Forward Engineering)

You can change any of the selected options and can generate the correct SQL statements. The options appear in a hierarchical tree structure, and you can expand or collapse the tree to review each group of options.

Follow these steps:

  1. Click Create Subject Area in the physical model and select the subject area for which you want to generate the schema.
  2. Click Actions, Forward Engineer, Schema.
  3. Select an option set from the Option Set list.
  4. Click the Summary tab.
  5. Choose Show Selected Only.

    The options that you have selected are displayed. Reivew the�selected options and change them if required.
