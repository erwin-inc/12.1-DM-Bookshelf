The Text Editor

The Text Editor is available in dialogs where you can work with text, such as the Comment tab in the property editors. The Text Editor enables you to use additional features to work with the text that otherwise are not available in the main editor.

When you click the Text Editor icon , a new window opens where you can edit your existing text or create new text. The toolbar provides the following features:

Save

Opens the Save As dialog so you can save the text.

Print

Prints the text.

Find

Opens the Find dialog to search the text.

Find Again

Repeats the last search you entered in the Find dialog.

Replace

Opens the Replace dialog, where you can enter text to find and replace.

Cut

Cuts the selected text.

Copy

Copies the selected text to the clipboard.

Paste

Pastes the text from the clipboard.

Checks the spelling of the text entered in the Text Editor.
