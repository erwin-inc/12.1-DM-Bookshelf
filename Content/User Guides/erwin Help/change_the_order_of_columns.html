Change the Order of Columns

During modeling activities you can use the Table Column Editor to change the order in which columns appear in the diagram and the Model Explorer.

To change the order of columns

  1. Right-click the table that contains the columns that you want to reorder and click Column Properties.

    The Table Column Editor opens.

  2. Click on the Navigation Grid toolbar and select one of the following options:

    If the icon is disabled, click Sort Items and select either Column Order or Physical Order.

    Reset order to match attribute order

    Resets column order to match the order of the corresponding attributes in the logical model.

    Reset all

    Opens the Reset Order dialog so you can specify the scope and type of the column order reset actions. Click OK when you are done with your selections.

  3. Click Close.

    The Table Column Editor closes, and the reordered columns display in the table on the diagram and in the Model Explorer.
