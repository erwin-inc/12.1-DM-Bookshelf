%ForEachColumn()

Expands the macro code for each of the columns in the given table.

Syntax

%ForEachColumn() {

<macro code>

}

Syntax Example

%ForEachColumn() {

%AttFieldName

}

Expansion

renting_customer

master_number

movie_copy_number

rental_record_date

rental_date

due_date

rental_status

payment_amount

payment_date

payment_status

overdue_charge

Full Syntax

%ForEachColumn(<table>,<separator>,<sort order>) {

<macro code>

}

Full Syntax Example

%ForEachColumn(Parent, ", ") {%ColName}

Expansion

master_number, movie_copy_number, general_condition

Usage

Classified as an Entity macro that you can use to create a Trigger Override in the Table Trigger Editor (Triggers dialog). An entity must be in scope; <table> can be Parent or Child (if a relationship is in scope), or empty. <separator> is optional.
