Define an Oracle Default Value

The Oracle Default Value Editor is used for viewing and defining properties for default values.

To define Oracle default values

  1. Click Default Values on the Model menu.

    The Oracle Default Value Editor opens.

  2. Select the default value in the Navigation Grid that you want to define.

    Click New on the toolbar to create a new default value. Use the Enter filter text box to filter a very large list of default values to quickly locate the one that you want to define.

  3. Click the Oracle tab and enter a default server value in the Value field.
  4. Click Close.

    The Oracle default value is defined and the Oracle Default Value Editor closes.
