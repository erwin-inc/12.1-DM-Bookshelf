Horizontally Partition a Table with a Transformation

You can horizontally partition a table with a transformation.

Important! Once you apply the Horizontal Partition transformation, it cannot be reversed. While in the current modeling session however, you can undo the action.

To horizontally partition a table with a transformation

  1. Select a table.

    The Horizontal Partition icon becomes active on the Transformations toolbar.

  2. Click the Horizontal Partition icon.

    The Horizontal Partitioning Wizard opens.

  3. Follow the instructions in the wizard. Click Finish.

    A table is created for each partition you specify in the wizard.
