View or Change a Model Source

You can use the General tab in the Model Sources Properties dialog to view or change the source file name and review the last synchronization date for a model source.

To view or change a model source

  1. Right-click a model source in the Model Explorer, and then choose Properties.

    The Model Sources Properties dialog opens.

  2. Click the General tab. The following information is displayed for the selected Source and Name:
    Source Model

    Indicates if the source model is a local *.erwin file, or a model located in a mart.

    Model

    Displays the full path and name of the source model.

    Click this button to browse your file system to locate a model source file.

    Synchronization
    Last Synched

    Displays the date and time you last synchronized the model source with the current model.

    Sync. . .

    Click this button to open the Sync with Model Source wizard and synchronize the source and current model.

  3. Click OK.

    The dialog closes.
