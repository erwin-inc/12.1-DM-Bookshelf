Define the Code to a Stored Procedure, Function, or Trigger

You can use the Code tab to review or assign code to a stored procedure, function, or trigger. The following describes the options for the Code tab:

Code

Displays the template code, which can include SQL statements and macros.

Macro Toolbox

Opens the Macro Toolbox where you can select a macro and double-click it to add the macro to the stored procedure, function, or trigger.
