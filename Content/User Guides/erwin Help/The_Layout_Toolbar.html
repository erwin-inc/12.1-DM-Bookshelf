The Layout Toolbar

The Layout Toobar includes options to lay out objects in your model. You can select a group of objects to lay out, or apply the layout preference to the entire model. Two options are included to redraw and reset objects.

Layout types include:

  • Circular Layout
  • Hierarchical Layout
  • Orthogonal Layout
  • Symmetric Layout
  • Tree Layout

Other options include:

Reset Relationship Paths

Reshapes the selected relationship lines on the diagram window. Manual reshaping of any relationship line will be lost.

Reset Objects to Automatic Sizing

Resizes the selected entities, tables, and views in the main workspace. Manual resizing applied to any entity, table, or view will be lost.

In versions of erwin® Data Modeler before r8, these options appeared on the Format Preferences dialog.
