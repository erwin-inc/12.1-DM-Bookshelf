Attach a Materialized View to a Script

To attach a materialized view to a script

  1. In a physical model based on Oracle, right-click a materialized view in the diagram, and select Pre & Post Scripts.
  2. Select Browser and select a script in the Script Template Name list.
  3. Select the materialized view that you want to attach to the script in the Unattached Table list and click .
  4. Click Close and click OK.

Select the Attach to New Table check box before you close the dialog to automatically attach the script to each new materialized view that you create.
