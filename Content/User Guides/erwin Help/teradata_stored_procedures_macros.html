Teradata Stored Procedures (Macros)

When your target DBMS is Teradata, you use the Teradata Macro Editor to add, change, view, or delete macros for the model, tables, or views. The functionality is identical to the Stored Procedure Editor used by all the other target DBMSs. You can open the Teradata Macro Editor by selecting Target - Teradata, Macros on the Model menu when Teradata is defined as the target server. See the section on Teradata Server Property Editors for information specific to the Teradata DBMS.
