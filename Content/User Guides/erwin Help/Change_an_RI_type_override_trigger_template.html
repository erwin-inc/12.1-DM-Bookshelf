Change an RI Type Override Trigger Template

You can use the Trigger Template Editor in a physical model to change an RI Type Override template which changes the behavior of the built-in template for an RI Trigger Type.

To change an RI type override trigger template

  1. Click RI Trigger Templates on the Model menu.

    The Trigger Template Editor opens.

  2. Select the trigger template that you want to change in the Trigger Template Name grid.
  3. Select an option to change the template:
    Change the template name

    Select the template in the Trigger Template Name grid and enter a new name.

    Add text or code to a template

    Select the template in the Trigger Template Name grid, and click the Code tab. Enter the text or code in the code area. You can use the toolbar to cut, copy, or paste text, open the Macro Toolbox, or work with the Text Editor.

    Add macro code to a template

    Select the template in the Trigger Template Name grid to which you want to add macro code and click the Code tab. Position the cursor in the code area where you want to insert the macro code. Click the Macro Toolbox button to open the Macro Toolbox. Select the macro you need and click the Insert Macro button. Type any parameters required in the code area.

  4. Click Close.

    The RI Type Override template is changed and the Trigger Template Editor closes.
