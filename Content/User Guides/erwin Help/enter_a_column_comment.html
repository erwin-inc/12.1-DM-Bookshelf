Enter a Column Comment

Use the Comment tab in the Table Column Editor to enter a comment for the selected column.

To enter a comment for a column

  1. Right-click the column in the Model Explorer for which you want to enter a comment and click Properties.

    The Table Column Editor opens.

  2. Click the Comment tab and work with the following options:
    Comment

    Lets you enter a comment for the selected column. You can click an icon on the toolbar to cut, copy, or paste text, restore inheritance, work with the Text Editor, or to run a spell check.

    If your target server supports comments, you can generate comments in the schema DDL script.

    Update Attribute Definition To Match

    Specifies to have column comments in a Logical/Physical model to also appear as the definition for the corresponding attribute in the logical model. If the check box is cleared, then the column comments for corresponding columns and attributes in a Logical/Physical model will contain different text.

  3. Click Close.

    Comments are saved and the Table Column Editor closes.
