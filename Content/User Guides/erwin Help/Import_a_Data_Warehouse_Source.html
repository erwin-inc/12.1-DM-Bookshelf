Import a Data Warehouse Source

You use the Data Movement Table tab in the Data Warehouse Source Object Editor to import a data warehouse source in a physical model. You can define tables of a data warehouse source or import source information from an existing database, script file, *.erwin file, a model located in a mart, or flat file.

Ensure you have selected the Data Movement option in the General tab of the Model Editor.

To import a data warehouse source

  1. Select Data Sources in the Model menu.

    The Data Warehouse Source Object editor opens.

  2. Select the Data Movement Table tab and click Import From .CSV to import a CSV file, or click Import Other to import from a database, script file, or *.erwin file. If you select to import from a database, script file, or *.erwin file, the Data Source Import wizard opens which enables you to select the items to import, set import options, and select case conversion of physical names options.

    When you complete the import process the imported tables and their columns display in the Detail tab of the Data Warehouse Source Object Editor.

  3. Click OK.

    The imported data warehouse source is saved.
