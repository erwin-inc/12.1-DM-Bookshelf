Working with Transformations

The r7.x transforms functionality has been significantly changed for r8. r8 now utilizes transformations, which is a "non-live" version of a transform that is applied to certain constructs of the model, but no transform objects are created and no live updates of source and target objects are maintained.

One of the most important differences between the live transforms support in previous releases and the non-live transforms support in r8 is the fact that property data will no longer be kept in sync between the source and target objects of any given transform.
