View Relationships Dialog

Use the View Relationships dialog to specify the properties of a view relationship (view derivation). You can perform the following modeling activities in this dialog:

  • Specify an alias for each source table referenced through the selected view relationship.
  • Change the order of the tables in the FROM clause of the view by ordering the view relationships.
  • Add comments for the view relationship.

You can perform these activities using the following options and tabs:

Parent Table

Displays the name of the parent table or view in the selected view relationship.

Child View

Displays the name of the child view in the selected view relationship.

Properties

Lets you enter an alias for the source table or view referenced through the selected view relationship.

Comment

Lets you enter a comment for the selected view relationship.
