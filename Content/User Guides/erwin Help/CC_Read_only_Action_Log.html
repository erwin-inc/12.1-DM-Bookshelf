Read-only Action Log

The Resolve Differences dialog includes a read-only version of the Action Log. The two-pane window tracks the actions made in the right and left panes of the dialog.

As you reconcile differences on your model, each transaction moves the previous transaction down the tree in the Action Log, with the first transaction always appearing at the bottom of the tree.

The Action log dialogue provides the following options:

  • Metamodel Filter
  • Find
  • Extended Find/Filter
  • Property Details
