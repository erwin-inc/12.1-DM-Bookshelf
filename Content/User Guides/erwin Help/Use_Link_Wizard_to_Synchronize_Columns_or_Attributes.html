Use the Link Wizard to Synchronize Columns or Attributes

The Link Wizard lets you create a link object to link the selected attribute/column with another attribute/column. When columns/attributes are linked, changes are synchronized between the objects based on the selected synchronization direction, and the selected synchronization properties.

Use the wizard to perform the following tasks:

  • On the Link page, enter a name for the link object, and define the synchronization direction. You can also create a definition for the link object.
  • On the Target page, select the target column for the link object. You can also create a new column on this page.
  • On the Properties page, select the properties you want to synchronize between the two link objects.

After you create the link object and establish the synchronization parameters, the linked objects are synchronized in real time. You can change the properties to synchronize in the Link tab of the Column/Attribute Editor.
