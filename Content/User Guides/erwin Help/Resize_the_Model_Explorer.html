Resize the Model Explorer

You can change the size and the position of the Model Explorer. When you close and open erwin the next time, the Model Explorer appears in the most recently saved size and position.

To change the horizontal size of the Model Explorer, position the pointer over the right Model Explorer boundary. When the pointer changes, drag the Model Explorer to the size you want.

To change the vertical size of the Model Explorer, you must undock it. Double-click the Model Explorer docking toolbar to undock it, and then position the pointer over the boundary you want to drag; when the pointer changes, drag the boundary to the size you want.
