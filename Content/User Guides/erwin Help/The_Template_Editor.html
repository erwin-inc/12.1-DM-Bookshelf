The Template Editor

You use features in the Template Editor to view, modify, and create new templates for use with schema generation. The Template Editor features give you a wide range of options for controlling schema generation for your models. You can review and print database-specific features of the default template files, create a custom template file based on a default template file, or create a completely new template file. All of the currently supported erwin® Data Modeler macros are available in the Template Editor, along with online documentation and customization options.

The Template Editor includes a toolbar, a Sample Context box, and four panes.

The Template Editor Toolbar provides easy access to the main editor features.

A Sample Context box displays beneath the toolbar. It allows you to select the starting object to use for the expansion of the template.

The Templates Pane displays the templates found in the template file that are available for editing and are separated into the following categories: Create, Alter, Drop, Insert, Clause, and Miscellaneous.

The Template Source Pane displays the source text for the selected template. You can navigate and edit the template text, insert macros, and print the text.

The Expanded Text Pane displays the read-only results of expanding the current template against the currently selected Sample Context object. You can change the Sample Context object by selecting a new object from the drop-down list.

The Macros Pane lists all available erwin® Data Modeler macros that you can use with templates.
