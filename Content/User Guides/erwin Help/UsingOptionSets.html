Using Option Sets

Each physical or logical/physical model has a FE Option Set object associated with it that holds the option settings you select. There is a default option set that is created by erwin® Data Modeler. Though several FE Option Set objects can exist in the model, only one can be active at a time.

When the Template Editor expands a template, it takes into account the current settings for the model. However, options are not saved to the model immediately upon selection in the user interface. To have the settings written to the model and available to the Template Editor, in the Forward Engineer Schema Generation dialog you must either save them explicitly using the Save or Save As buttons, or save them implicitly using the Generate or Preview buttons.
