Rename an Object in the Model Explorer

You can rename an object in the Model Explorer.

To rename an object in the Model Explorer

  1. Click on the object to select it, and click again. Alternatively, you can press F2 on the object.

    An edit box displays around the object's name.

  2. Enter the new name in the edit box and click Enter.

    The new name appears throughout the Model Explorer and the model. Any unique naming rules that you have selected for your model are enforced.

You can open the property editor for the object to rename it. Select the object and click Property Editor to open the editor for the object. Alternatively, you can right-click an object and select Properties to open the editor.
