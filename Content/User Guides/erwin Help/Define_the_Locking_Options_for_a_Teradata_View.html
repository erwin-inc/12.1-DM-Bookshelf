Define the Locking Options for a Teradata View

You can use the Locking tab of the Teradata View Editor to define the locking options for existing views for Teradata physical models.

To define the locking options for a Teradata view

  1. Click Views on the Model menu.

    The Teradata View Editor opens.

  2. Click the Locking tab and work with the following fields:

    Click New on the toolbar to create a new lock.

    Name

    Specifies the lock name.

    Locking Object

    Specifies the type of object to lock. Select from the drop-down list. Valid Values are: Database, Table, View, or Row.

    Lock Level

    Specifies the lock level. Select from the drop-down list. Valid Values are: Access, Read, Share, Write, or Exclusive.

    Lock No Wait

    Specifies to wait to acquire a lock.

    Database

    Specifies the database to lock. Select from the drop-down list.

    Entity Lock

    Specifies the entity to lock. Select from the drop-down list.

  3. Click Close.

    The Teradata View Editor closes.
