Macros Pane

The Macros Pane lists the macros supported in TLX.

For more information about macros, see the Template Language and Macro Reference guide, which is installed by default to the \Doc directory during product installation.

You can perform the following tasks in the Macros Pane:

  • Change how the macros display. Macros are listed alphabetically by default but you can change it to category display. Click the Toggle Macro Categories icon on the Template Editor toolbar to toggle between alphabetical and category display.
  • Insert a macro from the Macros Pane into the location you highlight in the Template Source pane. Highlight the area in the template source where you want to insert the macro and then double-click the macro to insert it.

    A macro may display more than once in the alphabetical list if it belongs to more than one category.

  • Create new macro categories in the Macro Categories dialog. Click the Edit Macro Categories icon to open this dialog.
  • Hide macros that are discouraged or deprecated or select categories to hide using the Template Editor Preferences dialog. Click the Configure icon to open this dialog. Discouraged and deprecated macros display in the Macros Pane using the following conventions:
    • Discouraged macro. Indicates that the macro has been replaced by an alternate macro. You are encouraged to switch to the new syntax when editing new code. These macros are fully supported and there is no intention to remove them from the product. However, they either may not run as efficiently as other approaches or may not have the same features available.
    • Deprecated macro. Support for this macro will eventually be removed from the product. Generally, deprecated macros are replaced by the post-load processors when a model is upgraded to a new release of the software.
