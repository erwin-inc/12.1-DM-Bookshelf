Import or Export Items in the Resolve Differences Dialog

After you compare two models, you can export or import items from one model to the other using the Resolve Differences dialog.

To export or import items in the Resolve Differences dialog

  1. In the Resolve Differences dialog, select the items you want to import or export:
    • To select one item and all its properties, click on the item in the Differences list.
    • To select multiple items, hold down the SHIFT or CTRL key and click on the items.
  2. Choose one of the following actions:
    • Click to move the selected item or property from the model in the left pane to the model in the right pane.
    • Click to move the selected item or property from the model in the right pane to the model in the left pane.

The status of the selected item is changed, and model is updated based on the action you select in the Resolve Differences dialog.

The toolbar buttons are available or unavailable (dimmed) depending on whether or not the action is available for the selected item.
