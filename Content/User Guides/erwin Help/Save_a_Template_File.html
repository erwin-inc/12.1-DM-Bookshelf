Save a Template File

If you modify a template file and do not want to overwrite the default template file provided with erwin® Data Modeler, you can save it with a new name. You can then work with the new template file in the same way you work with the default template files.

To save a template file

  1. Click the Save Template File icon on the Template Editor toolbar.

    The Save As dialog opens.

  2. Enter a name (*.fet) for the new template file, and click Save.

    Your new template file is saved.
