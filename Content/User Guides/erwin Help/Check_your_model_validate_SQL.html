Check Your Model/Validate SQL

You can create a Model Validation report before you forward engineer your model. The validation process reports errors associated with correct forward engineering. You can also run the Model Validation report against your open model when you click the Forward Engineer - Check Model icon on the Database Toolbar.

To check your model/validate SQL

  1. Open the model that you want to forward engineer. On the Actions menu, click Forward Engineer, Check Model.

    For Logical/Physical models, switch to the Physical view.

    The report opens in a new window. The report includes a header with the name of the model, the date of the model, and a list of relevant corrupt objects or errors.

  2. Click Save As.

    The report is saved in RTF format to an external file.

  3. Click Close to close your report.
