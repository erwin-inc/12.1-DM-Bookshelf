Option Sets

An option set is a customized list of objects and properties saved with the model, or as an XML file. A default option set is provided, which you can customize to your particular needs.

When you work with any of the Design Layers tools, you can use option sets to create custom filters for deriving, synchronizing, linking, or adding a model source or comparing models, databases, or script files using Complete Compare.

Option sets let you do the following:

  • Apply a default option set, which includes all objects in the applicable object tree. The default option set is read-only and cannot be deleted.
  • Modify the default option set and save it as an XML file, or with the model.
  • Retrieve an option set you saved to an XML file. After you open an option set, it appears in the Option Set drop-down list.
  • Update an existing option set and save your changes.
  • Delete an option set you no longer use.
