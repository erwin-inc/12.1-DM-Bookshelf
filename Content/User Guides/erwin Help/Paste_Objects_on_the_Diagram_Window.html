Paste Objects on the Diagram Window

You can paste entities, tables, views, or drawing objects. A different process exists to copy attributes or columns.

To paste entities, tables, views or drawing objects

  1. Select one of the following options:
    • Copy the selected objects to the clipboard.
    • Cut the selected objects to the clipboard.
  2. Perform one of the following options:
    • Select New from the File menu to create a new model.
    • Select Open from the File menu to open an existing model.
    • Open a file in the word processing, spreadsheet, or presentation application into which you want to include the model objects.
  3. Perform one of the following options:
    • Select Paste from the Edit menu to paste the model objects into the new or existing model.
    • Select Paste Special in the word processing, spreadsheet, or presentation application and save the file as a bitmap (.bmp) or metafile (.wmf).

If you do not see the objects you paste on the diagram window, scroll to find the pasted objects or use the Go To option from the Edit menu to locate them.
