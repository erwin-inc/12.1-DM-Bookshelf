Override Font and Color for a Shape

Although you choose to inherit a theme for an entire diagram, you can select a different font and color for a specific table. In other words, you can override the inherited settings.

To override font and color for a table

  1. From the ER Diagrams or the Subject Areas section of the Model Explorer, select the diagram in which you want to change the font and color.
  2. Right-click and select Switch to.

    The diagram appears in the workspace adjacent to the Model Explorer.

  3. In the diagram, locate and select the table for which you want to change the font and color.
  4. Right-click and select Override Fonts and Colors.

    The Style tab of the Model Shape Editor dialog opens.

  5. Work with the following options:
    Theme

    Specifies the theme that you want to apply to the selected shape. Click to edit the theme, or click to create a new theme.

    If you select values for the options that are listed under Local overrides to Theme, it means that these values will be used and the values mentioned in the theme will be overridden.

    Font Options

    Specifies the default font styles to display the name of the selected shape. Click to change the font style details. You can also choose to inherit, override, or harden this setting.

    Fill Options

    Specifies the default fill color for the selected shape. Click to change the fill style details. You can also choose to inherit, override, or harden this setting.

    Line Options

    Specifies the default line styles for the selected shape. Click to change the outline style details. You can also choose to inherit, override, or harden this setting.

    Outline Options

    Specifies the default outline styles for the selected shape. Click to change the outline style details. You can also choose to inherit, override, or harden this setting.

  6. Click Close.

    The changes are applied to the selected table and the Model Shape Editor dialog closes.
