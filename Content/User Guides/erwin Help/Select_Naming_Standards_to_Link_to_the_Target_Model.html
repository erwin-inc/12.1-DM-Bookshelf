Select Naming Standards to Link to the Target Model

Use the Naming Standards pane of the Link Model Source wizard to apply naming standards to the target model as you link to the model source.

To select naming standards to the target model

  1. Click Design Layers, Link Model Source on the Actions menu.

    The Link Model Source wizard opens on the Source Model pane.

  2. Click Naming Standards.

    The Naming Standards pane opens.

  3. Work with the following options:
    Name Mapping
    Object Type

    Displays the objects for which you can set name conversion preferences.

    Property/Value Table

    Displays a two column table that lists the property and value for the selected object type. To change the value, click the row in the table to edit the text (for example, to change the macro) or display a drop-down list with the available options.

    Macro Text

    Displays the default macro that is used to derive the physical model object name from the logical model object name. You can enter a new macro for the selected object type.

    Use Glossary

    Click this row in the table to select true or false. Select true to use a glossary of words and abbreviations to derive the physical model object name. Select false to not use a glossary of words and abbreviations.

    Abbreviation Type

    Select Abbreviation or Alternate Abbreviation from the list to specify whether to use an abbreviation or alternate abbreviation from the glossary to derive the physical model object name. Available only when you select true in the Use Glossary option.

    Prefix

    Click the row in the table, then enter the prefix text to add before table, view, column, domain, relationship, index, and validation names in the physical model.

    Suffix

    Click the row in the table, then enter the suffix text to add after table, view, column, domain, relationship, index, and validation names in the physical model.

    Special Characters
    Leave

    Preserves special characters in object names in the derived model.

    Remove

    Removes special characters and replaces them with the character indicated in the Replace With box.

    Replace With

    Indicates the character to use when replacing special characters in object names in the derived model.

    Remove Vowel

    Select this check box to remove vowels from object names in the derived model.

    Use Naming Standards

    Indicate a naming standards file to use when enforcing standards in the derived model. Use Browse File System or Browse erwin MM to select a standards file.

    Click Next.

    The Options pane of the Link Model Source wizard opens.
