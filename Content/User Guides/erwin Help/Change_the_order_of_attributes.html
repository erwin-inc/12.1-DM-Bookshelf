Change the Order of Attributes Using the Attribute Editor

You can change the order of attributes using the Attribute Editor.

To change the order of attributes using the Attribute Editor

  1. Right-click the entity that contains the attributes that you want to reorder, and click Attribute Properties.

    The Attribute Editor opens.

  2. Select the attribute that you want to move, then use the following reordering buttons on the Navigation Grid toolbar as needed to position the attribute in the Attribute list.

    Moves the selected attribute to the top of the list.

    Moves the selected attribute up one position in the list.

    Moves the selected attribute down one position in the list.

    Moves the selected attribute to the bottom of the list.

  3. Repeat as necessary for each attribute that you want to move.
  4. Click Close.

    Your changes are saved and the Attribute Editor closes.

When you change the order of attributes in the Attribute Editor, the order in which the attributes appear also changes in the Model Explorer and in the diagram window. However, the order of related columns does not change in a logical/physical model.
