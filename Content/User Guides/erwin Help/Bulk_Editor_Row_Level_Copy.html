Bulk Editor Row-Level Copy

Use the row-level copy feature of the Bulk Editor to copy selected rows in the Bulk Editor to the clipboard, which can then be pasted into another spreadsheet program, such as Microsoft Excel. This makes it easier to share the data for editing property values, especially if another person on your team needs to perform the edits and does not have access to erwin® Data Modeler. When editing work is complete simply copy the updated rows to the clipboard for pasting back into the Bulk Editor.

The Bulk Editor uses a special textual format so each row and column is unambiguously identified and can be pasted into Microsoft Excel.

The data includes a column heading for each column. The column heading contains the internal erwin® Data Modeler property ID and a user-ready name for the property represented by that column.

The data also has an extra column that precedes the other columns. It contains the erwin® Data Modeler object type, two colons, and then the object ID. This information is used during a Paste operation to uniquely identify an object in the model.
