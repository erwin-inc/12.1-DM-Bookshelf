The Display Toolbar

The Display Toolbar contains options to work with display levels in your model and to toggle between logical and physical display (in a logical/physical model).

You can apply the display options in two ways:

  • Select an object or a group of objects in the workplace, then click the icon to apply that display setting to those objects.
  • Do not select any objects, then click the icon to apply that display setting to the entire diagram. If you have already set display options for individual objects, you can override those settings for the entire diagram when you choose this option.

Display options include:

  • Entity/Table Display Level
  • Primary Key Display Level
  • Attribute/Column Display Level
  • Keys Display Level
  • Definition/Comment Display Level
  • Graph Display Level (available only in case of graph database-based models)
  • Subject Area (drop-down menu)
  • Logical/Physical (toggle box)

In versions of erwin® Data Modeler before r8, these options appeared on the Standard Toolbar.
