Shortcut for Adding Objects

The default action for adding an object is to click the object toolbar, and click anywhere on the diagram window to insert that object. To add another object of the same type, you are required to repeat this procedure. However, you may want to add multiple objects of the same type without returning to the object toolbar.

When you press the Ctrl key after you select the object icon, you can simply click again in the diagram window to add another object of the same type. This shortcut feature applies to the creation of entities, materialized query tables, materialized views, tables, relationships, views, and all drawing objects.

For example, to add multiple consecutive entities:

  1. Click the entity icon in the erwin Toolbox.
  2. Hold down the Ctrl key.
  3. Click anywhere on the diagram window.
  4. Click again anywhere on the diagram window to add another entity.

When you release the Ctrl key, the multi-object create mode is canceled, and you can proceed with on-diagram editing to name your entities, add attributes, and so on.

If you continue to hold down the Ctrl key, you can click another object on the toolbar to begin adding objects of that type. For example, after adding multiple entities, click a relationship icon to add multiple relationships.
