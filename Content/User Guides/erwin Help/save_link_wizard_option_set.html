Save Your Property Selections in the Link Wizard as an Option Set

You can save your property selections as an option set so you can save time and not have to make the selections more than once. This procedure assumes you are already in the Link Wizard Properties page.

To save Link Wizard property selections as an option set

  1. Click New on the Option Set toolbar.

    The Save Current Option Set dialog opens.

  2. Select one of the following options from the Save To drop-down control:
    <Current model name>

    Stores the option set with the current model. Enter a name for the option set in the Option Set Name field.

    XML File

    Stores the option set in an external XML file. Specify a file name and location in the Location field using the ellipsis button.

  3. Click OK.

    The Save Current Option Set dialog closes.

  4. Click Apply.

    The option set is saved and the Link Wizard closes.
