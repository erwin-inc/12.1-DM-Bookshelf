Create a User-Defined Property

You can use the User Defined Properties dialog in either the physical or logical model to create, update, and maintain your user-defined properties. Based on the object class you choose, you can then assign the UDP to a specific model object from within the editor for that object. For example, if you create a UDP for the table object class, when you open the Table Editor, your UDP is available on the UDP tab.

To create a user-defined property

  1. Click User Defined Properties on the Model menu.
    • The User Defined Properties: Logical dialog opens if you are in a logical model.
    • The User Defined Properties: Physical dialog opens if you are in a physical model.
  2. Select the object class to which you want to apply a user-defined property from the Class drop-down list.

    When you click an existing UDP, the row is highlighted and you are able to either delete the UDP or move the UDP up or down in the list.

  3. Enter the user-defined property information in the following grid columns:
    Name

    Specifies the UDP name.

    Type

    Specifies the UDP data type.

    Default

    Specifies a UDP default value, when applicable.

    Description

    Specifies a description for the UDP.

  4. Click Close.

    User-defined properties are created and the User Defined Properties dialog closes.
