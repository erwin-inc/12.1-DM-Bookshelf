Change Properties to Synchronize Between Linked Objects

You can use the Link tab on the Column or Attribute Editor to change the list of properties synchronized between the target and source objects.

  1. Open the Column or Attribute Editor and select the column or attribute for which you want to modify the synchronization properties.
  2. Click the Link tab, then click the Choose Properties to Synchronize icon on the toolbar.

    The Link Synchronization Properties dialog opens.

  3. Work with the following options:
    Select properties to synchronize

    Lets you specify the properties to synchronize. Select the check box next to the property name to set it as a synchronized property. The list of properties is populated based on the model type you are working with (Logical, Physical, or Logical/Physical).

    Select user-defined properties to synchronize

    Lets you select the user-defined properties to synchronize.

    The dialog includes controls that enable you to quickly select all or clear all properties, or undo the last select/clear action.

  4. Click OK.

    The changes are saved, the Link Synchronization Properties dialog closes, and you are returned to the property editor.

  5. Click Close.

    The editor closes.
