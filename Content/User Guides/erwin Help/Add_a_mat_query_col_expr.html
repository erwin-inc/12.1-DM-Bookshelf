Add a Materialized Query Column Expression

You can use the Materialized Query Columns editor to add a materialized query column expression in a Db2 model.

To add a materialized query column expression

  1. Right-click a Materialized QueryTable and select Column Properties.

    The Materialized Query Columns Editor opens.

  2. Click New.

    A new Materialized Query Table Column is added to the list.

  3. Work with the following fields:
    Name

    Defines the name of the materialized query column expression.

    Domain Parent

    Specifies the domain parent for the materialized query column.

    Physical Data Type

    Specifies the physical data type for the selected materialized query column.

    Expression

    Indicates that you want to use expression text defined on the Expression tab.

  4. Click Close to save the new materialized query column and close the Materialized Query Column Editor.
