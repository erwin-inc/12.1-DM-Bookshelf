Specify Password Options of a Teradata Profile

You can use the Teradata Profile Editor to specify the password options of a profile that a group of users can share in a Teradata physical model.

To specify password options of a Teradata profile

  1. Click Target-Teradata, Profiles on the Model menu.

    The Teradata Profile Editor opens.

  2. Click the Password tab and specify the values for the following options:
    Password Special Char

    Specifies if the password will use special characters. Select an option from the drop-down list.

    Allow Username in Password

    Specifies if the user name is permitted in the password. Select True or False from the drop-down list.

    Require Mixed Case Password

    Specifies if mixed cases are required in the password. Select True or False from the drop-down list.

    Require Alpha Characters in Password

    Specifies if alpha characters are required in the password. Select True or False from the drop-down list.

    Special Characters in Password

    Specifies if special characters are permitted in the password. Select No, Yes, or Required from the drop-down list.

    Password Digits Required

    Specifies if digits are required in the password. Select No, Yes, or Required from the drop-down list.

    Password Expire

    Specifies the number of days before the password expires.

    Password Min Char

    Specifies the minimum number of characters in a password.

    Password Max Char

    Specifies the maximum number of characters in a password.

    Password Max Logon Attempts

    Specifies the number of invalid login attempts before locking the user account.

    Password Reuse

    Specifies the number of days that must elapse before a password can be reused.

    Allow Restricted Words for Password

    Specifies that restricted words are allowed in the password. Select True or False from the drop-down list.

    This property does not apply to Teradata 2.6.

    Password Lock User Expire

    Specifies the number of minutes before locking a user account.

  3. Click Close.

    The Teradata Profile Editor closes.
