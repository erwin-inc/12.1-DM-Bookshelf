Db2 for i Support Summary

The following list summarizes objects supported under Db2 for i. To create any of these objects and define their properties, refer to the Defining Db2 for i Objects topic.

  • Stored Procedures
  • Script Templates
  • Table
    • Index
    • Table Column
  • View
    • View Column
