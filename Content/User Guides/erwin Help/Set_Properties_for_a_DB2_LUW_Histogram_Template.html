Set Properties for a Db2 LUW Histogram Template

Use the Db2 for LUW Histogram Template Editor to set properties for a histogram template. You create histogram templates to specify the uppermost bound for the range of data that the corresponding histogram captures.

This topic applies to Db2 for LUW versions 9.5 and 9.7 only.

To set properties for a Db2 histogram template

  1. Right-click a Histogram Template in the Model Explorer and select Properties.

    The Histogram Template Editor opens.

  2. Select a histogram template and work with the following options:

    Click New on the toolbar to create a histogram template. Use the Enter filter text text box to filter a large list of names to locate the one you want to work with.

    Name

    Specifies the name of the histogram template.

    Top Value

    Specifies the top value of the second to last bin (the last bin has an unbounded top value).

    Generate

    Specifies whether or not to generate DDL for this object during Forward Engineering.

  3. (Optional) Click the Comment tab and enter any comments that you want to associate with the object.
  4. (Optional) Click the UDP tab to work with user-defined properties for the object.
  5. (Optional) Click the Notes tab to view and edit user notes.
  6. Click Close.

    The Db2 for LUW Histogram Template Editor closes.
