Open a Template

Opening a template and using the features in the Template Editor, you can modify or update it, review template source and expanded text, insert macros, print template source, and so on.

To open a template

  1. Select Forward Engineer, Forward Engineering Templates from the Actions menu (in either a physical or logical/physical model).

    The Template Editor opens.

    When the editor is invoked, it attempts to select the Create Entity template by default.

  2. Search for the template you want to open in the Templates pane and double-click it.

    The template opens, template source displays in the Template Source pane, and the Expanded Text pane displays the processing status. You can now use the features in the Template Editor and apply them to the template.
