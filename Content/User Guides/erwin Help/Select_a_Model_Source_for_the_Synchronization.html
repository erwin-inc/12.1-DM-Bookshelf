Select a Model Source for the Synchronization

You select the model to which you want to synchronize in the Model Sources pane. No action is required in this pane if you have only one model source for the current model.

To select a model source for the synchronization

  1. Click Design Layers, Sync with Model Source on the Actions menu.

    The Sync with Model Source wizard opens on the Model Sources pane.

  2. Select the model source you want to synchronize with, then click Next.

    The Type Selection pane opens.

You can click the Sync button at any time to start the synchronization process.

Important: Select the "Set selected model source as read-only" check box to open the model with read-only restrictions. When the Resolve Differences dialog displays, you cannot move objects or properties to a model that has been marked "read-only."
