%DomainDef

Returns the definition of a domain.

This macro is a legacy macro and is used for logical-only or combination logical/physical models. For any new physical-only models you need to use %DomainComment. For more information on legacy macros and their related new macros, see the topic "Legacy Macros".

Syntax

%DomainDef(<domain name>) or %DomainDef

Syntax Example

%ForEachDomain() {

%DomainName

%DomainDef

}

Expansion

definition1

definition2

Usage

Classified as a Constraint macro that you can use globally with an argument %DomainDef(<domain name>) or %DomainDef within the scope of a domain (for example, %ForEachDomain).
