Select the Source Model for a Derived Model

When you start the Derive Model wizard, your open model is selected by default as the source model. You can select a different model from a file, from open models in memory, or from a erwin® Data Modeler Workgroup Edition library.

To select the source model for a derived model

  1. Click Design Layers, Derive New Model on the Actions menu.

    The Derive Model wizard opens on the Target Model pane.

  2. Click the Source Model link in the Explorer pane.

    The Source Model pane opens.

  3. Work with the following options:
    Load From

    Select a different *.erwin file or a model located in a mart for your source model. You can skip this step and keep the default selection, or pick another model open in memory.

    Open Models in Memory

    Displays the name and location of models you have open in the current modeling session. The model you have open when you start the Derive Model wizard is selected by default.

  4. Click Next.

    The Target Model pane opens.

You can click the Derive button at any time to start the derive process.
