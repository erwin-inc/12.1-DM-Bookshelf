Oracle Editors - From Tab

The From tab is identical for the Oracle property editors that use it. You use it to specify the source table, view, or materialized view. The following describes the options for the From tab:

Available Tables/Views/Materialized Views

Contains the tables, views, and materialized views in your model.

From

Shows the table, view, or materialized view that you selected from the Available Tables/Views/Materialized Views list box.

Alias

Enter an alternative name for the selected source in the From list box. An alias is typically a new name that is more appropriate in the context of the table, view, or materialized view selected. The Alias option becomes available only after you have selected a source in the From list box.
