Create a Teradata Replication Rule

You use the Teradata Replication Rule Editor to set properties for a replication rule. You can associate a replication rule with a replication rule set.

To create a Teradata replication rule

  1. Click Target-Teradata, Replication Rule Sets on the Model menu.

    The Teradata Replication Rule Set Editor opens.

  2. Click the Rule tab then click the Replication Rule Editor Icon.

    The Teradata Replication Rule Editor opens.

  3. Select a replication rule and work with the following options:
    Name

    Specifies the name of the replication rule.

    Object Type

    Specifies the object type for the replication rule. Select an object type from the drop-down list. Valid values are Index, Macro, Table, Temporary Table, Trigger, or View.

  4. Click the General tab and specify the following properties:
    • Like Clause
    • Like Escape Character
    • Not Like Clause
    • Not Like Escape Character
  5. (Optional) Click the Comment tab and enter any comments that you want to associate with the object.
  6. (Optional) Click the UDP tab to work with user-defined properties for the object.
  7. (Optional) Click the Notes tab to view and edit user notes.
  8. Click Close.

    The Teradata Replication Rule Editor closes.
