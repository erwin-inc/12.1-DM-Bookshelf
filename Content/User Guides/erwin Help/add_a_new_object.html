Add a New Object

When you begin a new modeling project, you will need to add new logical and physical objects to your model. To add a new object, in the Model Explorer right-click the object type you want to add, click New, enter the name for the new object, and hit Enter. The object is added to the model.

Use the object's property editor to define additional properties for the object.
