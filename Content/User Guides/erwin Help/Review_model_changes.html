Review Model Changes

The Resolve Differences dialog displays a list of changes to the current model since you opened the model from the mart. You can review the changes that you made, however you cannot save your changes back to the mart.

Attribute\Column Datatype changes made in the Mart Model are not displayed in the Resolve Differences dialog.

To review changes in your model

  1. Click Review Changes Against Mart on the Actions menu.

    The Right Model pane of the Complete Compare wizard opens.

  2. Click Compare.

    The Resolve Differences dialog opens.

  3. Use features in the Resolve Differences dialog to compare the model in the left pane (the current model that you have checked out of the mart) with the read-only model in the right pane (the Mart version) and review and resolve the differences.
  4. Click Finish.

    You return to the Complete Compare Wizard. You can change any options in the wizard and begin the compare process again, or click Close to close the wizard.
