Select Objects from the Source Model for Synchronization

You use options in the Source Object Selection pane to select objects from the source model to participate in the synchronization process.

To select objects from the source model for synchronization

  1. Click Design Layers, Sync with Model Source on the Actions menu.

    The Sync with Model Source wizard opens on the Model Sources pane.

  2. Click the Source Object Selection link on the Explorer pane.

    The Source Object Selection pane opens.

  3. Complete from the following options:

    Choose Objects Using Sets

    Select this check box to use preset options for selecting objects sets, such as Subject Areas. Clear the check box if you plan to use a customized set of selected options from the object list. For the category Source Model Objects, you can select from the following filter subsets:

    • Objects Linked to this Target Model - Include objects in the source model that are linked to the target (current) model.
    • Objects Not Synced with this Target Models - Include objects in the source model have never been synced with the target (current) model.
    • Objects Synced with this Target Model but not imported - Include objects in the source model that are not linked to any objects in the target (current) model.

    Selected Objects

    Displays the tree of model objects for the source model. Select or clear the check box next to the object name. Selected objects are included in the synchronization process.

    If you create a customized list of objects from the object list, then select the Choose Objects Using Sets option, the tree of model objects is reset to the default.

  4. Click Next.

    The Options pane opens.

You can click the Sync button at any time to start the synchronization process.
