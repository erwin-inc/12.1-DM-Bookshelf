Use the Go To Editor

The Go To editor helps you to quickly go to a selected model object on the diagram. You can also use this editor to open an object's property editor and override default color and formatting.

To use the Go To editor

  1. On a diagram, press Ctrl + G.

    The Go To editor opens.

  2. Work with the following options:
    Filter Type

    Lets you filter objects by their type. By default, it is set to ALL.

    Show Drawing Objects in List

    Displays the names of drawing objects (shapes) created in the diagram. By default, the drawing objects are not displayed.

    Sort

    Opens a menu so you can sort the current list of objects in alphabetical or reverse alphabetical order. When you select an option, the toolbar button changes to provide you a visual indication of your choice:

    Alphabetic

    Reverse Alphabetic

    Alphabetic, by Owner, Name

    Reverse Alphabetic, by Owner, Name

    Names

    Opens a menu so you can select to show object names as either logical names or physical names. When you select one of the options, the toolbar button changes to provide you a visual indication of your choice:

    Logical Names

    Physical Names

    Physical Names, show owner

    Physical Names, show owner for Non Current User

    The Logical Names option does not display if you are working in either a logical-only or physical-only model.

    Override Theme Style Settings

    Opens the Model Shape Editor for the selected object so you can override the style of the current theme of that object.

    Edit the selected object

    Opens the object editor for the selected object.

    Enter filter text

    Lets you filter a large list of objects. When you enter the first few letters of the object name, the corresponding objects appear in the list.

    Name

    Specifies the name of the object.

    Type

    Specifies the object type.

    Parent

    Specifies the name of the parent if any, of the selected object.

    Child

    Specifies the name of the child if any, of the selected object.

  3. Click Go To.

    The Go To editor closes and the object selected in the editor is highlighted on the diagram. The focus of the diagram is adjusted in the Overview pane as well.
