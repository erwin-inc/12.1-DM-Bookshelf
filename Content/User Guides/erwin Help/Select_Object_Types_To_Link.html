Select the Object Types You Want to Link Between the Source and Target Model

You use the options in the Type Selection pane of the Link Model Source wizard to specify the object types you want to link between the source and the target model. You can also choose a filter to select only those objects in the source model not linked to any model source, or create an option set to specify specific objects. By default, all objects in the source model are included in the link process.

To select the object types you want to link between the source and target model

  1. Click Design Layers, Link Model Source on the Actions menu.

    The Link Model Source wizard opens on the Source Model pane.

  2. Click Type Selection.

    The Type Selection pane opens.

  3. Work with the following options:
    Compare Level

    Choose the compare level as the basis for the link model source process. The type of level you select defines the set of objects that will be linked.

    • Logical Level - All objects contained on the logical level of a model.
    • Physical Level - All objects contained on the physical level of a model.
    • Database Level - All objects on the database level.
    Option Set

    A default option set (including all objects) is selected for the compare level you choose. You can filter out specific objects by clearing the check box next to the object name and saving the selections in a new option set.

    Open, Save, Save As, or Delete an Option Set

    You can create a new option set, which you can save for future use, or open, modify, or delete an existing option set.

    Click Next.

    The Object Selection pane of the Link Model Source wizard opens.
