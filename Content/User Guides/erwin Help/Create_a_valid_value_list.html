Create a Valid Value List

The validation rules that you create can establish a value, a list of values, or a range of acceptable values to store in a table, column, entity, or attribute. Use the Validation Rule Editor to create a valid value list for your validation rules.

To create a valid value list

  1. ClickValidation Rules on the Model menu.

    The Validation Rule Editor opens.

  2. Select a validation rule in the Navigation Grid for which you want to create a valid value list and ensure that the Valid Values option is selected from the Type drop-down.
  3. Click the General tab.
  4. Click the empty cell in the Valid Value column, enter a new name, then do one of the following:
    • Enter the definition for the value in the Definition field.
    • Copy valid values from another validation rule.
    • Click the Import button to select the .csv or .txt file that contains the valid values you want to import.
  5. Click Close.

    The valid value list is created and the Validation Rule Editor closes.
