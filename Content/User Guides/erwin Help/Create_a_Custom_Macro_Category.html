Create a Custom Macro Category

You can create a new macro category to suit the needs of your modeling environment. When you add a new macro category it is available for use in all template files.

To add a custom macro category

  1. Select Forward Engineer, Forward Engineering Templates from the Actions menu (in either a physical or logical/physical model).

    The Template Editor opens.

  2. Click the Edit Macro Categories icon on the Template Editor toolbar.

    The Macro Categories dialog opens.

  3. Ensure Category View is selected and click the New button.

    The New Macro Category dialog opens.

  4. Enter a name for the new category and click OK.

    The New Macro Category dialog closes and the new macro category displays in alphabetical order in the Categories list.

  5. Select the check box for each existing macro that you want to add to the new macro category and click OK.

    The new macro category is created and contains the macros you selected, and the Macro Categories dialog closes.

  6. Click Close.

    The Template Editor closes.
