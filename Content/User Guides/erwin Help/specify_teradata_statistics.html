Specify Statistics Options for a Teradata Table, Hash Index, or Join Index

In Teradata 13, you can specify statistics options for a Teradata table or join index on the Statistics tab. You use the statistics feature to collect demographic data for one or more columns of a table or join index, and compute a statistical profile of the collected data

To specify statistics options for a Teradata table, hash index, or join index

  1. Open the Teradata Table or Teradata Join Index editor and click the Statistics tab.
  2. Click New to create a new statistics object or work with an existing statistics object.
  3. Specify the following:
    Name

    Specifies the name for the statistic object.

    Using Sample

    Specifies to scan a system-selected percentage of table rows rather than scanning all rows for a table.

    Use Partition Columns

    Specifies that statistics are to be collected on the system-derived PARTITION column set for an SLPPI table.

    Key Group

    Specifies the index for which statistics are to be collected.

  4. (Optional) Click the Statistics Editor icon to open the Statistics Editor where you can specify statistics members.
  5. Click Close.

    The statistics options are defined and the editor closes.
