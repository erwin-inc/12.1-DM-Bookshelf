Oracle Partitions Tab - Local-Composite Hash and Local-Composite List

Local-Composite Hash Option

The following describes the options available for the Local-Composite Hash partitioning type in the Partitions tab:

Partition Definition

Do the following to create partition definitions:

Store In

Click the button to open the Oracle Select Tablespaces Dialog to add a tablespace to the Tablespace list box.

Partition Name

Create individual partitions. Enter a value in the Partition Name column. When you click on a specific partition name, you can use the button and the button to move it up or down in the list, or click the Delete button to delete it.

Edit Properties

When you click on the button within the Edit Properties column, it opens the Oracle Partition Description dialog where you can modify the properties for the selected partition.

Subpartitions

When you click the button within the Subpartitions column, it opens the Oracle Index Subpartitions Editor where you can modify the properties for the selected subpartition.

Local-Composite List Option

The following describes the options available for the Local-Composite List partitioning type in the Partitions tab:

Partition Definition

Do the following to create partition definitions:

Partition Name

Create individual partitions. Enter a value in the Partition Name column.

Edit Properties

When you click on the button within the Edit Properties column, it opens the Oracle Partition Description dialog where you can modify the properties for the selected partition.

Subpartitions

When you click the button within the Subpartitions column, it opens the Oracle Index Subpartitions Editor where you can modify the properties for the selected subpartition.

When you select the Local-Composite List partition option, the Store In option is unavailable from this screen, however, it is available within the Oracle Index Subpartitions Editor.
