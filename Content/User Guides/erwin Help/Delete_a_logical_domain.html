Delete a Logical Domain

You can work with features in the Domain Editor to delete a domain from the logical model.

You cannot delete any of the standard domains (<default>, Blob, Datetime, Number, and String).

To delete a logical domain

  1. Click Domains on the Model menu.

    The Domain Editor opens.

  2. Click the name of the domain in the Navigation Grid that you want to delete and click the Delete icon.

    To filter a large list of domains, enter the name (or the first few characters) of the domain that you want to delete in the Enter filter text box.

    The domain is removed from the Navigation Grid.

    Even if the domain you want to delete is assigned to an attribute, you are not prompted to confirm the delete action. When you delete a domain, all attribute properties set by that domain are reset to the parent domain setting. A message is displayed in the Advisories pane at the bottom of the Domain Editor letting you know which parent domain is now assigned to the attribute as a result of the deletion. If you decide that you do not want to delete the domain you can click the Undo button.

  3. Click Close.

    The domain is deleted and the Domain Editor closes.
