Forward Engineering Options for Couchbase

The following are the forward engineering options for Couchbase in erwin DM.

Option Selection

Parameter

Description

Additional Information
Option Set Specifies the option set template for forward engineering

Open: Use this option to open a saved XML option set file.

Save: Use this option to save a configured option set.

Save As: Use this option to save an option set either in the model or in the XML format at some external location.

Delete: Use this option to delete an option set.
Database Template Specifies the database template for controlling schema generation

Browse: Use this option to browse and select a database template.

Edit: Use this option to edit a template in the Template Editor.

Reset: Use this option to reset the Database Template option.
Script Option Specifies the script options for the schema generation

Pre-Script: Indicates whether pre-scripts attached to the schema are executed

Post-Script: Indicates whether the post-scripts attached to the schema are executed
Bucket Syntax Option Specifies the bucket syntax options for the schema generation

Create: Specifies whether the Create syntax for buckets is executed

Delete: Specifies whether the Delete syntax for buckets is executed

Comments: Specifies whether the Comments for buckets are included in the generated schema
Document Syntax Option Specifies the document syntax options for the schema generation

Insert: Indicates whether the Insert syntax option for documents is executed

Blank Value: Indicates whether the Blank Value syntax option for documents is executed
Index Syntax Option Specifies the index syntax options for the schema generations

Create GSI: Indicates whether the Create GSI syntax option for indexes is executed

Drop GSI: Indicates whether the Drop GSI syntax option for indexes is executed

Create FullText: Indicates whether the Create FullText syntax for indexes is executed

Drop FullText: Indicates whether the Drop FullText syntax for indexes is executed
View Syntax Option Specifies the view syntax options for the schema generations

Create: Specifies whether the Create syntax for views is executed

Delete: Specifies whether the Delete syntax for views is executed
User Syntax Option Specifies the user syntax options for the schema generations Create: Specifies whether the Create syntax for users is executed

Delete: Specifies whether the Delete syntax for users is executed

Document Filter

Parameter

Description

Additional Information
Documents Specifies the selected documents for the schema generation

 
Display either Logical Names or Physical Names

Logical Names: Indicates that only logical names of the documents are included in the generated schema

Physical Names: Indicates that only physical names of the documents are included in the generated schema

Physical Names, show owner: Indicates that physical names and owners of the documents are included in the generated schema

Physical Names, show owner using User: Indicates that the physical names and owners of the documents are included in the generated schema. Owners of the documents are displayed using User.
Select all of the items in the list Use this option to select all the documents in the list.

 
Unselect all of the items in the list Use this option to unselect all the documents.  
Select all unselected items, and unselect all selected items Use this option to select all the unselected documents and unselect all the previously selected documents.  

Preview

Parameter

Description

Additional Information
Viewer Displays the schema in the viewer editor

Collapse All: Use this option to collapse all the nodes.

Search: Use this option to search a text entered in the search box.

Find Previous: Use this option to navigate to previous search string in the search results

Find Next: Use this option to navigate to next search string in the search result.
Text Displays the schema in the text editor

Save: Use this option to save the generated schema.

Search: Use this option to search through the generated schema.

Print: Use this option to print the generated schema.

Replace: Use this option to find and replace text in the generated schema.

Copy: Use this option to copy the selected text in the schema.

Text Options: Use this option to edit window settings, fonts, and syntax color.

Git: Use this option to commit the FE script to a Git repository.
