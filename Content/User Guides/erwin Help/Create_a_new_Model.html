Create a New Model

If an existing model does not meet your requirements, you can create a new model.

Follow these steps:

  1. Do one of the following:
    • On the File tab, click New.
    • On the Home tab, click New.
    • On the Quick Access Toolbar, click New.

    The New Model dialog box opens.

  2. Work on the following options:
    1. Select Logical, Physical, or Logical/Physical.
    2. Select the target database and its version number.
    3. Select <Default> for Template. Click OK.

      A new blank model opens.

To create a model that is same as an existing model, complete the following fields:

  1. Select Match template.
  2. Click Browse Files or Browse Mart and select a template model from your local computer or Mart.
  3. Click OK.

    If you are not connected to the Mart and you click the Browse Mart button, you are prompted to connect to the Mart. Connect to the Mart to select a template model from Mart.

    The objects in the selected template model appear in the new model. The type, target database, and the theme of the template model are applied to the new model.

To create a model that is same as an existing model, but of a different type, work on the following fields:

  1. Select Logical, Physical, or Logical/Physical.
  2. Select Match template target server.
  3. Click Browse Files or Browse Mart and select a template model from your local computer or Mart.
  4. Click OK.

    The objects in the selected template model appear in the new model, as appropriate for the selected type. The theme of the template model is applied to the new model. The target database of the template model is set for the new model, if necessary.

To create a model that is based on a template, complete the following fields:

  1. Select Logical, Physical, or Logical/Physical.
  2. Select the target database and its version number.
  3. Click Browse Files or Browse Mart and select a template model from your local computer or Mart.
  4. Click OK.

    The objects in the selected template model appear in the new model, as appropriate for the selected type and target database. The default theme is applied to the new model.

Preserve Template Binding:

If you choose to use a template while creating a model, select Preserve the template binding to:

  1. include template information, objects, and properties in the new model.
  2. be able to sync the model with the template.
