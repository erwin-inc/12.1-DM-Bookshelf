Create a Query

You can create a query (SELECT statement) by entering text directly in the Query box, or by pasting text from another source.

To create a query

  1. On the ribbon, click the Tools tab.
  2. Click Query Tool.

    The Query Tool opens.

  3. Enter your query in the Query box, or click the paste button on the toolbar to paste a query from another source.

    Your query is ready for execution. Use other features in the Query Tool to execute and save the query.

Before you can execute a query you must be connected to erwin ODBC or to a database. For sample queries and the details of schemas and tables, refer to the Useful Queries topic.
