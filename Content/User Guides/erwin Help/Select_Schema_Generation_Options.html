Select Schema Generation Options

Use the options on the Option Selection tab to select schema generation options.

To select schema generation options

  1. On the Actions tab, click Schema.
  2. Click the Option Selection tab.
  3. Select the name of the current option set and choose from the following options:
    <Database> Schema Generation

    Displays available schema generation groups from which you can select schema generation options.

    Schema

    Displays available schema generation options for the selected schema generation group. The target server you select determines which schema generation options you can select.

    Database Template

    Displays the default database template for the selected target server, or the name of the customized template currently in use. Click Browse to change the template. Click Reset Template to restore the default template. For more information, see this topic: Using custom or default database templates during schema generation.

  4. Click the Owner Override tab and set a global name override, or a name override for the listed object groups.
  5. Click the Table Filter tab and select the tables that you want to forward engineer.
  6. Click the Preview tab and preview the schema DDL script generated based on the options you selected.
  7. Click Generate to start the schema generation process for forward engineering or alter script generation or click OK to save the changes and close the dialog.
