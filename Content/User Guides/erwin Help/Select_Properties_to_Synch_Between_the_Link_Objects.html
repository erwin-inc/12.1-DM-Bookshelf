Select Properties to Synchronize Between the Link Objects

Use the Properties page in the Link Wizard to select the properties you want to synchronize between the linked objects.

To select link object properties to synchronize

  1. Click the Properties page in the Link Wizard.

    The Properties page displays the available properties you can synchronize between the objects.

  2. Work with the following options:
    Option Set

    Lets you select an existing option set to populate the property selections on this page. An option set contains the definition of the properties selected in a previous session. Use the toolbar icons to load, save, create, or delete an option set.

    Select All

    Selects the check boxes for all the items in the list.

    Select None

    Removes all check box selections so that no items are selected in the list.

    Toggle Selection

    Specifies to select check boxes for any unselected items and remove check box selections for selected items.

  3. Click Apply.

    The selected properties are synchronized between the linked objects.

    If you want to change the synchronization properties list, you can click the Choose Properties to Synchronize icon on the Link tab toolbar to open the Link Synchronization Properties dialog.
