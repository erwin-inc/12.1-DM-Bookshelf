Add a Table to a Data Warehouse Source

You use the Data Movement Table tab in the Data Warehouse Source Object editor to add a table to a data warehouse source. You can also use this tab to add a new table to the Tables list, change the name of the selected table, or delete a table as a data warehouse source.

To add a table to a data warehouse source

  1. Select Data Sources in the Model menu.

    The Data Warehouse Source Object editor opens.

  2. Select the Data Movement Table tab and click New on the toolbar to create a new table.

    The new table is added to the data warehouse source. You can click to add columns to the new table.
