Enter Materialized View Column User-Defined Property Values

To enter materialized view column user-defined property values

  1. In a physical model based on Oracle, add a user-defined property for materialized view columns.
  2. Right-click a materialized view table, and select Database View Columns.
  3. Select a materialized view column from the Columnlist.
  4. Select the UDP tab and click in the Value box for a user-defined property.
  5. Enter the user-defined property value and click OK.
