Manage Profiles

As a Mart Administrator, you can assign a user to a profile to control the actions that the user can perform on an object. You can assign a user to multiple profiles and give the user a greater control over objects. Assign at least one profile to each user. However, you do not need to assign all users to a profile immediately after creating the Mart. When you install Mart, Admin, Architect, Modeler, and Viewer profiles are created.
