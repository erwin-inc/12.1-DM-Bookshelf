Delete an Add-In Software Application Menu Item

If you want to remove an add-in software application because you no longer need to use it, use the Add-In Manager to delete the item from the menu.

To delete an add-in software application menu item

  1. Click Add-Ins, Manage on the Tools menu.

    The Add-In Manager dialog opens.

  2. Click the add-in application item that you want to delete in the menu items tree and click the Delete Menu Item button .

    A dialog opens prompting you to confirm that you want to delete the menu item.

  3. Click Yes.

    The dialog closes and the add-in application is deleted from the menu items tree.

  4. Click Save.

    The Add-In Manager dialog closes.

Make sure you expand all the items in the menu items tree so you can view all the add-in applications and not delete the wrong add-in application menu item.
