Change an Add-In Software Application Menu Item

You can make changes to an existing add-in software application item such as changing the executable information, changing the name of the item, or adjusting the order in which items appear on the Add-Ins menu. You can perform all of these activities using the Add-In Manager.

To change an add-in software application menu item

  1. Click Add-Ins, Manage on the Tools menu.

    The Add-In Manager dialog opens.

  2. Select one of the following in the menu items tree:
    • Select the command icon of the add-in application that you want to change.
      • For Com Objects, you can change the add-in name, COM programmatic identifier, or function name in the Name, COM, ProgID, and set Function Name to Run fields.
      • For executables, you can change the add-in name, target file, and directory path in the Name, Target, and Start In fields.
    • Select the submenu icon of the add-in application submenu name that you want to change and change the add-in submenu name in the Name field.
    • Adjust the location of the menu item in the Add-Ins menu hierarchy by clicking either the Move Up button or the Move Down button .
  3. Click Save.

    Your changes are saved and the Add-In Manager dialog closes.
