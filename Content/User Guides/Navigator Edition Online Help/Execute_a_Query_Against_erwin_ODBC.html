Execute a Query Against erwin ODBC

To run a query against the current erwin Data Modeler model, you must first activate the ODBC connection. The connection may already be active, if you selected the option to start the ODBC driver on the Reporting tab in the Options dialog.

To connect to erwin ODBC

  1. On the ribbon, click the Tools tab.

  2. Click Query Tool.
    The Query Tool opens.
  3. Open a query (*.qry or *.sql) file or create a query.

    Query text displays in the Query box and is ready for execution.

  4. Click the Connect to erwin ODBC icon on the toolbar.

    The ODBC connection is established. The Disconnect icon becomes active. Use other features in the Query Tool to execute and save the query and result set.
