Set Print Margins

You can specify margins for all pages of your printed model.

To set print margins

  1. Click Page Setup in the Print dialog.

    The Page Setup dialog opens.

  2. Click the Margins tab.
  3. Enter a value between 0 and 1.5 inches for the left, right, top, and bottom of the page.
  4. Click OK.

    The Page Setup dialog closes.

Margin measurements are considered when displaying the embedded Diagram Window in the Print dialog.
