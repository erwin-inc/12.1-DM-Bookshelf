Object Properties

Using the Model Explorer, you can open the property editor for an object two ways:

  • Right-click the object and click Properties.
  • Select the object and click the Property Editor icon in the Model Explorer toolbar.

For bulk editing of object properties, you can use the Bulk Editor.

Regardless of the method you choose to add or edit an object or its properties, the Model Explorer and the data model are always in sync.

Object properties also migrate through relationship lines. In other words, objects can inherit properties from their parent objects.
