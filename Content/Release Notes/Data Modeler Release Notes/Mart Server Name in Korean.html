Mart Server Name in Korean

Suppose that you provide the Mart Server machine name in the Korean language (Korean characters) and configure the Mart Server. If you try to connect to the Mart from erwin DM and you provide the Korean language name for Server Name, the following message appears:

The client application version is different from the Mart Server version. Install the same version of the application and the Mart Server and then try.

If you try to log in to erwin Mart Administrator by using the Korean language name, it does not display.
