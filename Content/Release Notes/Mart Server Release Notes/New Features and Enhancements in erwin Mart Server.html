New Features and Enhancements in erwin Mart Server

Following new features and enhancements are available in erwin Mart Server .

erwin Mart Server on Cloud

Starting release 12.1, erwin Mart Server is available in two flavors, on-premise and on-cloud.

The on-cloud offering works same as the older, on-premise offering, except for the following features:

  • User Types: User types, Azure AD User and Azure AD Group User have been added.
    For more information, refer to the Mart Users topic.

  • erwin Mart application access: To connect to erwin Mart on cloud from erwin Data Modeler, the connection parameters differ from the erwin Mart on premises.
    For information, refer to the Connect to Mart topic.

For more information or queries about erwin Mart Server on cloud, contact erwin Support.

Enhancements

erwin Mart Server has undergone several enhancements as follows:

  • In case of failures in upgrade, you can now rerun report upgrade for all models.

  • You can now control the level of permission and assign a user or a group to a profile, library, or model.

  • erwin Mart Server now supports LDAP with SSL.

DM Connect for DI Support

  • The DM Connect for DI feature has been upgraded to support:

    • Business Entity Types: Model export jobs store business entity type (system environment type) under environment details in erwin Data Intelligence (erwin DI).

    • erwin DI v12.0

Third Party Tools

Support for third-party tools has been upgraded as follows:

  • Tomcat has been upgraded from v9.0.52 to v9.0.62.
  • erwin Mart Server now uses AdoptOpenJDK (build 11.0.6+10)
