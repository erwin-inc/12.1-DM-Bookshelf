Mart Enhancements

Several additions and enhancements have been implemented to Mart. These enhancements are:

  • Special Characters Support

  • Search Models

  • Validate Connection

  • Session Timeout

  • DM Connect for DI

  • DB Connection Details

  • Run Now

  • Mart Permissions

Special Characters Support

erwin Mart now supports more characters for username and email address for all server users, group users, and windows users. For more information, refer to the Character Limitations topic.

Search Models

As data models become complex and large, it becomes difficult to search a specific catalog from all models available in mart. It is necessary to search catalogs quickly. To facilitate this, erwin mart now includes a Loaded Model option that helps search models within models loaded from mart. Also, this increases performance of mart server. For more information, refer to the Manage Catalogs topic.

Validate Connection

erwin Mart now provides an option to validate your connection for active directory details. This option is available erwin Mart Configuration dialog box. For more information, refer to the Manage Catalogs topic.

Session Timeout

erwin Mart server has updated its session timeout from 5 minutes to 30 mins.

DM Connect for DI

Dm connect for DI now supports Couchbase 7.0, erwin DI Suite 10.2, and erwin DI Suite 11.0.

DB Connection Details

erwin DM now stores database connection parameters such as DBMS Name/DSN, Host Name/IP Address, and Port in environment details, which is useful while reverse engineering.

Run Now

The Run Now option enables you run the job immediately while scheduling job using DM connect for DI. To access this option go to DM Connect for DI page.Schedule a job and then, switch to Job Information tab.
