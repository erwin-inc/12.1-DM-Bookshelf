Assign or Change a Domain Icon

You can assign any image (.bmp file) as the icon for the non-inheritable property of the domain. You can assign the same image or can select a different image that the attributes or columns of the domain can inherit.

You can change the domain icon only in the logical model.

Follow these steps:

  1. Click Model, Domains.
    The Domain Editor opens.
  2. Select the domain in the Navigation Grid that you want to work with and click the General tab.
  3. Specify the image to use in the Image drop-down or click New to open the Image Editor and create one.
  4. Click Close.
    The icon is assigned to the selected domain.
