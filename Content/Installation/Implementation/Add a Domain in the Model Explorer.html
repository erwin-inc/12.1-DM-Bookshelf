Add a Domain in the Model Explorer

The Model view lists the domains for the current model, which include the default domains and the domains created by you.

Follow these steps:

  1. Right-click a domain type (blob, string, number, and so on) and click New.
    An edit box opens with a default domain name.

    The selected domain type is assigned as the parent.

  2. Enter a name for the domain in the text box and press Enter.The new domain appears according to the sort order that you specify (either alphabetically or hierarchically). To switch the sort order of the Domains, right-click the Domains folder, and select the sort option that you prefer.

For more information about domains, see Working with Data Models.
