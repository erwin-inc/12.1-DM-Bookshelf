Generate a Model Lock Report

The Model Lock report provides information about the lock status of all the models in a library.

Connect to Mart to view the available reports and to generate a model lock report.

Follow these steps:

  1. Click Tools, Mart Reports, Other, Model Lock.

    The Database Login dialog opens.

  2. Enter login information and click Connect.
  3. Enter additional parameters for the report.

    The report is generated and opened in the viewer.

  4. Use the toolbar options to export or print the report.
  5. Click Exit on the File menu.

    The Report Viewer closes.
