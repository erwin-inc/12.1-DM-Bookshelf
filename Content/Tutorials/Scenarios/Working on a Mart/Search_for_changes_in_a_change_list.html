Search for Changes in a Change List

When you save a model with changes to the mart, and conflicts with the current model in the mart are detected, the Difference Reconciliation dialog opens.

To search for changes in a change list

  1. Click Find on the primary toolbar in the Difference Reconciliation dialog.

    The Find dialog opens.

  2. Enter what you want to find, select your search options:
    Search

    Transaction, Object, or Property. The default is object level.

    Match Case

    Search case-sensitive text.

    Match whole cell

    Search for the entire text string.

    Search Up

    Search for the text from the bottom up. The default is to search from the top down.

    Expand hidden branches

    Expands all branches in the tree that were hidden.

    Deep Search

    Indicates that you want to search all rows, including ones that are hidden. When you select this check box you override the Expand hidden branches search option, which is then dimmed.

    Type

    Check to search for changes of a specific type (for example, Create), select Type.

    Left model

    Check to search for changes that apply to the left model, select Left model.

    Right model

    Check to search for changes that apply to the right model, select Right model.

    Click Find.

    The first occurrence in the change list that matches the criteria you specified is selected. The button name changes to Find Next after the first change is found.
