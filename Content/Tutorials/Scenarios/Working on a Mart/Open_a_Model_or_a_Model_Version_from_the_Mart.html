Open a Model or a Model Version from the Mart

When you work in a multiuser environment, Mart acts as a central repository. You open a model from Mart to work collaboratively.

The Open Version drop-down lists all the versions that are associated with the selected model. The Current Version is the current state of the model and the remaining versions are previous Delta versions or Marked versions of that model.

You can select the lock option when you open a model. Model locking helps to secure models and manage how changes are saved to the mart.

Follow these steps:

  1. On the ribbon, click Mart > Open.

    The Open dialog opens.

  2. Select your model from the library.
  3. (Optional) Select the Show Templates Only check box.

    Only the models that are saved as templates are displayed.

  4. (Optional) From the Lock Type drop-down list, select a lock.
  5. (Optional) From the Open Version drop-down list, select the model version and click Open.

    The model opens in the workspace.
