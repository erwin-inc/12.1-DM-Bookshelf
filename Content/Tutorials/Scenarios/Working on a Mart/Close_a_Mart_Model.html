Close a Model

When you finish working on a model, either from Mart or locally, you can close the current data model and continue working on another data model. You can close a data model by closing either the model window or erwin� Data Modeler.

Follow these steps:

  • Perform the following steps to close an offline mart model or a non-mart (local) model:
    1. On the ribbon, click Mart > Close.

      The Save Model dialog opens.

    2. Select the Save check box to save the changes in the model and click OK.

      The Save Model dialog closes and the model is closed.

  • Perform the following steps to close an online mart model:
    1. On the ribbon, click Mart > Close.

      The Save Model dialog opens.

    2. (Optional) Click Browse for a Mart Path.

      The Save As dialog opens.

    3. (Optional) From the Catalog, select a new location and click Save.

      The model is saved in the new location and the Save As dialog closes. In the Save Mart Models dialog, Model Path displays the new location.

    4. In the Save Mart Models dialog, select the Save check box for the models that you want to save and click OK.

      The Save Model dialog closes and the Mart Offline dialog opens. In the Save Mart Models dialog, Model Path displays the new location.

    5. Select the model, and from the Save to drop-down list, select your option.
      Offline

      Saves the model offline to the local directory. You can work on the offline model without connecting to Mart. Every update to the offline model is recorded untill you rebind the model to Mart.

      Detach

      Copies the model and saves the copy to the local directory. The detached model is not connected to the mart and it is considered as a non- mart model.

      As a best practice, use the Detach option and save the mart model as a local model.

    6. (Optional) If you have selected Offline or Detach, click Open Folder, and navigate to the desired folder location to save this data model.

      The Model Path field shows the new location of the data model.

    7. Click OK.

      The Mart Offline dialog closes and the data model is closed.
