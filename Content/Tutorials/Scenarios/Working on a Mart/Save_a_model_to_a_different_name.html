Save a Model to a Different Name

You must save a model using a different name if you want to save changes to the model and avoid overwriting information from the original version of the model.

A model name must be less than 130 characters.

To save a model using a different name

  1. Click Mart, Open on the File menu.

    The Open dialog opens.

  2. Select a Library in the list of libraries in the left pane, a Model Name in the right pane, and click Open.
  3. Make the appropriate changes to the model, then click Mart, Save As on the File menu.

    The Save As dialog opens.

  4. Select the library in which you want to save the new model from the Library list. Enter a different name for the model in the Model text box, clear the Save Model as a Template check box, and click OK.

    The model is saved with the new name.

Important: If you save a model using an existing name, all versions of the original model with that name are deleted.
