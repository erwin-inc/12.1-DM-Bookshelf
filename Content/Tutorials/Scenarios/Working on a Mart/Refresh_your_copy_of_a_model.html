Refresh Your Copy of a Model

The Refresh option lets you update the current model with any changes made to the server version by you or by other users. You can use the features in the Difference Reconciliation dialog to review and resolve any differences or conflicts to the model.

To refresh your model, click Mart, Refresh on the File menu after you have made changes to an open model.

The model is refreshed.
