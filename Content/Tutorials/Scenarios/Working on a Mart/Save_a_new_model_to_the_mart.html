Save a Model to Mart

You save a model to Mart so that multiple users can work on the same model. You can save the model in the mart root node or libraries.

If you want to create another library, you can open Catalog Manager from the Save dialog and create a library.

Follow these steps:

  1. On the ribbon, click Mart > Save.

    The Save dialog opens.

  2. Select the library where you want to store your data model.
  3. Enter a name for the model in the Model Name text box.

    The model name must not exceed 255 characters and must be unique in the library. If you enter an existing model name, a warning dialog appears and lets you choose whether to replace the existing model.

  4. (Optional) Select the Mark as template check box to save the data model as a template.
  5. (Optional) Select the Maintain multiple versions check box to maintain multiple versions of the data model.

    If you clear the Maintain multiple versions check box, then erwin� Data Modeler permanently deletes all the existing versions and it maintains only one version. erwin� Data Modeler stops creating a delta version for any incremental save.

  6. (Optional) Type a short description about the model in the Description text box.
  7. Click Save.

    The model is saved.

If you update an existing Mart model, then a dialog appears where you can provide a description of the changes. You can suppress this dialog if you do not want the dialog to appear every time you update a model. The updated model is saved as a version. Every time when you update a model, a new version is created with all the changes in that model.
