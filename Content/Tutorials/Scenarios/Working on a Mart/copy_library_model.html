Copy a Library or Model

You can copy a library to a different library or Mart. Similarly, you can copy a model to the same library, to a different library, or to a Mart. When you copy a library or model, all the objects associated with it are also copied.

Follow these steps:

  1. Select the library or the model that you want to copy.
  2. Hold Ctrl, and drag and drop the library or the model to the target location.

    The library or the model, and the objects associated with it are copied to the target location.

If you drag and drop without holding the Ctrl key, you cut the model, but not copy.
