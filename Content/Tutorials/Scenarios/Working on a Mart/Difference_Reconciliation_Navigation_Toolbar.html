Difference Reconciliation Navigation Toolbar

The following lists the buttons that you can use in the Difference Reconciliation Navigation toolbar and describes the purpose for each button.

Go to Next Difference

Highlights the next object or property conflict.

Go to Previous Difference

Highlights the previous object or property conflict.

Go to Next Conflict

Highlights the next object or property difference.

Go to Previous Conflict

Highlights the previous object or property difference.

The definitions of conflict and difference are as follows:
  • Conflict: When two users open the same model, change the same object, and save the model, then there is a conflict for that object.
  • Difference: When two users open the same model, modify different attributes of the same entity, and save the model, the attributes have a difference.
Find an Item

Searches for an object or property.

Find Next Item

Searches again, using the search parameters already established.

Collapse all

Collapses the deepest level of the comparison tree.

Expand all

Expands the lowest branch of the comparison tree.
