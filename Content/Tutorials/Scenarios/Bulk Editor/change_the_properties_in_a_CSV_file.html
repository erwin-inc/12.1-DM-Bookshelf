Change the Properties in a .CSV File

Change the properties of the selected objects in a CSV file if your data modelers do not have access to erwin� Data Modeler. You can export the properties of the selected objects to a CSV format and can provide the CSV file to your data modelers to edit.

Do not edit the first two columns in the CSV file. If you edit the first two columns, you cannot import your changes to erwin� Data Modeler.

Follow these steps:

  1. In the Bulk Editor pane, Click Save.

    The Save As dialog opens.

  2. Enter a file name, provide a location, and click Save.

    The file is saved with .csv extension.

  3. Open the .csv file, make your changes, and save the file.

    The changes are saved and the CSV file is ready for the import to erwin� Data Modeler.
