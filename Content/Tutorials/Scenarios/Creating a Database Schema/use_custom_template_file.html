Use a Custom Template File

You can use a customized forward engineering template to generate the schema.

To use a customized template, click Browse and select the appropriate template file. If you decide to use the default forward engineering template instead, you can use the Reset Template option.

Important! If you use a custom template file, the file path is saved with the model. If you move the custom template file to another location, the link to the custom template is broken. The option to use the default template is restored.
