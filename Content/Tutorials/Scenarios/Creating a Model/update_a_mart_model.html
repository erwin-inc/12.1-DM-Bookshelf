Update a Mart Model

Update a model to complete it. If you work on Mart, you can update not just your models, but also the models that other users have created. The objects that you can update depend on the profile you are assigned to. The Mart live permissions feature verifies your permissions as you update a model.

The following permissions are required at a minimum:

  • You have the Modify permissions on the model you want to update.
  • You are assigned to the library in which the model exists.
  • You have the Create and Modify permissions on the model objects you want to create or update.

Follow these steps:

  1. Open erwin DM and connect to Mart.
  2. Open a model and create or update model objects and relationships as required.
  3. Create or update drawing objects. Use the auto-layout option and rearrange the model objects.
  4. Save the model.
