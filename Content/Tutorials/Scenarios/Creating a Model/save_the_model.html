Save the Model

Save the model to save your work. You can save a model to your local computer or to Mart. To save your model to Mart, you must be logged on to Mart.

Follow these steps:

  1. To save a model to your local computer, click File and then click Save or Save As.
  2. Select a folder, enter a name for the model, and then click Save.
  3. To save a model to Mart, click File, Mart, and then click Save or Save As.
  4. Select a library, enter a name for the model, and then click Save.

    The model is saved in the selected location and you are assigned as the creator of the model. A default profile is attached.

    Mart Administrator selects the default profile that is attached to the creator of the model.

  5. Close the model or close erwin� Data Modeler.

    The Mart Offline dialog opens. The Mart Offline dialog lists all the mart models that are currently open.

  6. Select the model, and from the Save to drop-down list, select one of the following options:
    Offline

    Saves the model offline to a local folder. Click Open Folder and select a folder where you want to save the offline model. You can work on the offline model without connecting to Mart. The updates are saved until you rebind the model to Mart. The permissions that you have on Mart apply to an Offline model as well.

    Detach

    Copies the model and saves the copy to a local folder. Click Open Folder and select a folder where you want to save the detached model. The detached model is not connected to Mart and is considered as a non-Mart model.

  7. Click OK.

    The Mart Offline dialog closes.
