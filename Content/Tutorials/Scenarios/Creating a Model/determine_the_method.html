Determine the Method

You can create a model in several ways. From the following methods, determine the best method that suits your needs:

  • Create a new model: Use this method when you are starting a new project.
  • Create a model that is same as a template: Use this method when you want to copy all the objects from a template model and change only a portion of the new model.
  • Create a model that is based on a template and matches the selected database and model type: Use this method to copy all the objects from a template model to suit the database that you select.
