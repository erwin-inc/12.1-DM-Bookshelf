Add a New Relationship in a Physical Model

Add a new relationship to your physical model using the Model Explorer.

To add a new relationship in a physical model

  1. Right-click the Relationships icon in the Model Explorer and click New.

    The New Relationship dialog opens.

  2. Enter information in the following fields and click OK.
    Tables

    Specifies the parent and child table to use for the relationship. Make a selection from each drop-down.

    Type

    Lets you select one of the following relationship types to create:

    Identifying

    Specifies to create an Identifying relationship, which is a relationship between two tables in which an instance of a child table is identified through its association with a parent table.

    Non-Identifying

    Specifies to create a Non-Identifying relationship, which is a relationship between two tables in which an instance of the child table is not identified through its association with a parent table.

    The relationship is added.
