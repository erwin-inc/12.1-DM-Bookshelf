Search the Resolve Differences Dialog

When you have a large list of items to compare, you scroll down the Resolve Differences dialog to see a specific object or property. Instead of scrolling down, use the Find dialog to find text strings.

Follow these steps:

  1. In the Resolve Differences Dialog, click the Find icon.
  2. Complete the fields to search for the required keyword. The following fields are not self-explanatory:
    Match whole cell

    Indicates that you want to search against the text string in each cell of the Resolve Differences display and only return identical matches. By default this check box is cleared, so that string fragments return a match.

    Expand Hidden Branches

    Include all branches of a tree in the search, including hidden branches. Clearing this check box limits your search to only those branches that are expanded in the display. This check box is selected by default.

    Deep Search

    Indicates that you want to search all rows in the Resolve Differences dialog. When you select this check box you override the Expand Hidden Branches search option, which is then grayed.

    Type

    Indicates that you want to search for changes of a specific type.

  3. Click Find Next.

    The first occurrence that matches the criteria you specified is selected.
