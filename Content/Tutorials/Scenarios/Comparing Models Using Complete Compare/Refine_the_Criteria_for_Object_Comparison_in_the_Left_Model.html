Refine the Criteria for Object Comparison in the Left Model

You can further refine the criteria for comparison between the left and right models. When you refine the criteria, you narrow the range of objects for comparison. For example, you can select to compare only the subject areas that a specific user owns. The objects displayed in the Selected Object pane change depending on the kind of compare you are performing.

Follow these steps:

  1. In the Left Model Object Selection pane, filter objects by clearing the check box next to the object name.

    Choose from the following options:

    Choose Objects Using Sets

    Uses preset options for selecting objects sets, such as Subject Areas, Triggers, or objects by owner. The available sets appear in a tree in the wizard. Clear this check box if you plan to use a customized set of selected options from the object list.

    Selected Objects

    Displays the tree of model objects for the left Model. Select or clear the check box next to the object name. For example, suppose that you select only the Employee table in this step and the next step. The Resolve Differences dialog displays comparison information only for the Employee table.

    Hide Unselected Objects

    Removes unselected objects from the Selected Objects display. This feature makes it easier to display the objects you selected, when you are working with a long object list.

  2. Click Next.

    The left object selection is saved and the Right Model Object Selection pane opens.
