(Optional) Generate Alter Script

While you work in the Resolve Differences dialog, you can update the database or script file that is associated with the model by generating an alter script. When a change allows the generation of alter scripts, the Left or Right Alter Script icons become active in the toolbar.

Follow these steps:

  1. Begin your Complete Compare session and navigate to the Resolve Differences dialog.
  2. Work with the options in the Resolve Differences dialog to reconcile differences between the models.
  3. When active, click the Left Alter Script/Schema Generation icon or Right Alter Script/Schema Generation icon.
  4. Work with options in the Alter Script wizard to generate the script.

Alter Script is generated.

(Optional) Manage Complete Compare Snapshots

While you work in the Resolve Differences dialog, you can mark a row as a "known difference" and can save your selections as a snapshot. When you mark a row as a known difference, you remove it from the list of differences, effectively filtering it out of the compare scenario.

Snapshots are the marked differences between the two models that are being compared. Therefore, a snapshot is valid and is available when the same two models are being compared.

Follow these steps:

  1. In the Resolve Differences dialog, select the rows to include in the snapshot, and click the Marks/Unmarks a Row as a Known Difference icon.

    When you mark a row as a known difference, the row and all dependent rows disappear from the display.

  2. Click the Save Snapshot icon to open the Save Snapshot dialog.
  3. Complete the following fields:
    Left Model/Right Model

    Specifies whether you want to include the model in the snapshot. The left model is selected by default.

    Snapshot Name

    Specifies a name for the snapshot. To overwrite an existing snapshot, select it from the drop-down list.

    Description

    Displays an optional description for the snapshot.

  4. Click OK to close the Save Snapshot dialog.
  5. (Optional) To load a Complete Compare snapshot, perform the following steps:
    1. Click the Load Snapshot icon to open the Load Snapshot dialog.
    2. Select a snapshot from the list and click OK.

    The saved snapshot is loaded and the known differences that are established in the snapshot are applied to your current compare session. The known differences identified in the snapshot are removed from the Resolve Differences dialog display.

You have successfully compared models with databases.
